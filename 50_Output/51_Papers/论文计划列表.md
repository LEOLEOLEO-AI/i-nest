## Paper 3（独立论文，新增 2026-03-18）

**题目候选：** *Device-Algorithm Decoupling: A Unified Framework for Matching Physical Substrates to Neural Computation*

**核心贡献：**

### 1. α_dev统一第一性原理推导
```
α_dev = ln(χ_unit)
χ_unit = 单个功能单元在临界工作点的非线性放大比（统一颗粒度）
```

器件α_dev对照表（全覆盖）：

| 器件 | α_dev | 物理来源 | 文献 |
|------|-------|---------|------|
| CMOS逻辑门 | 1.00 | NAND开关比 | Taur 1998 |
| 生物HH神经元 | 3.00 | 动作电位临界增益 | Hodgkin 1952 |
| 忆阻器(阵列级) | 3.70 | R_OFF/R_ON对数 | Yao 2020 Nature |
| 光子MZI | 2.30 | 非线性饱和吸收 | Feldmann 2021 Nature |
| 自旋MTJ | 2.50 | 1+TMR比 | Grollier 2020 NatElec |
| 单qubit | 0.69 | ln(2)，两态叠加 | Nielsen 2000 |
| 量子纠缠集群 | 4.00 | 海森堡极限纠缠增益 | Preskill 2018 |
| SNN-LIF | 3.50 | 全有全无+STDP | Maass 1997 |
| SNN-AdEx | 3.80 | 自适应指数IF爆发 | Brette 2005 |

### 2. α_algorithm（算法层非线性，各架构）

```
α_algorithm = ln(χ_algorithm)
χ_algorithm = 激活函数在工作点的有效信息增益比
```

| 算法/架构 | α_alg | 推导依据 |
|-----------|-------|---------|
| DNN/ReLU | 2.00 | 半激活，有效信息保留率≈0.5 |
| CNN/ReLU | 2.20 | 空间不变性额外增益+0.2 |
| LSTM | 2.30 | 4门控等效非线性 |
| Transformer/Softmax | 3.33 | 注意力集中度，Clark 2019 |
| SNN-LIF | 3.50 | 全有全无脉冲 |
| SNN-AdEx | 3.80 | 自适应爆发模式 |
| 现代Hopfield | 3.00 | 指数注意力，Ramsauer 2020 |

### 3. 「脱节系数」D = α_algorithm / α_device

```
D = 1.0 → 器件-算法完美匹配（最优）
D > 1.0 → 算法超越器件，器件成瓶颈（CMOS困境）
D < 1.0 → 器件超越算法，算法浪费器件（忆阻器+ReLU）
```

关键结论：
- **CMOS+Transformer**：D=3.33，CST极低L0 → "大力出奇迹"本质=用参数规模弥补D
- **生物神经元**：D=1.0，器件即算法，进化最优解
- **忆阻器+SNN-AdEx**：D≈1.03，当前工程最优匹配，CST→L4
- **量子纠缠+量子算法**：D=1.0，理论最优，但Γst≈0（退相干）→ 实际L0

### 4. 量子的深层分析
- 量子优势不来自α（单qubit α=0.693，比CMOS低！）
- 量子优势来自Sc（希尔伯特空间指数爆炸）
- 但Γst≈0（退相干=时空解耦）将CST拉回L0
- 容错量子计算（Γst→0.5）才能释放真正潜力

| 光子+Transformer | 1.45 | 0.56 | L0 | 难 | 2030+ |
| 忆阻器+SNN-AdEx | 1.03 | 2.96 | L4 | 中 | 2027-2031 |

## Paper 5（新增 2026-03-19）

**题目候选：** *Active Inference on Neuromorphic Substrates: A CST-Guided Framework for Physical World Models*

**核心贡献：**

当前所有物理世界模型（NVIDIA Cosmos、具身智能等）的根本局限：目标函数外生（人为定义损失函数），无法实现真正的感知-行动自洽闭环。本文提出以**自由能原理（FEP）替代损失函数**，以**CST框架量化其智能等级提升效果**。

### 核心方程对比

```
当前范式（外生目标）：
  L = ||预测 - 标签||²   ← 人定义"正确"
  行动 = argmax Reward   ← 人定义"好坏"

新范式（内生目标）：
  F = 惊讶度 + KL散度    ← 物理熵，无需人定义
  F = -ln P(o|m) + D_KL[Q(s) || P(s|o)]
  行动 = argmin E_Q[F]   ← 主动减少惊讶
```

### Γst的物理自洽定义

自由能原理为Γst提供了物理自洽的数学形式：

$$\Gamma_{st}^{\text{FEP}} = \text{NMI}(P_{\text{gen}}, Q_{\text{rec}}) \propto -\frac{dF}{dt}$$

自由能下降速度直接等于时空耦合强度——**Γst不再是人为测量值，而是从感知-行动循环中自然涌现**。

### 与现有框架的核心差异

| 维度 | NVIDIA Cosmos | CST-PhysWorld（本文） |
|------|--------------|----------------------|
| 目标函数 | 人为定义重建误差 | 自由能F（物理自洽） |
| 行动逻辑 | 最大化奖励R | 最小化预期惊讶E_Q[F] |
| 世界模型 | 被动预测视频 | 主动生成+贝叶斯更新 |
| 学习信号 | 外部标注/演示 | 感知-行动循环自洽 |
| Γst性质 | 静态（训练后固定） | **动态（随交互自然增长）** |
| CST预测 | ~0.08（L0） | 0.25–0.40（L1–L2） |
| 生物对应 | 无 | 皮层预测编码（Rao & Ballard 1999）|

### 架构：四模块感知-认知-执行-反馈闭环

```
感知：P(o, s, a) 多尺度生成模型，Sc由物理因果图决定
认知：Q(s|o) 识别模型，自由能F最小化，不需要外部标签
执行：π = argmin E_Q[F]，行动=主动减少惊讶（非最大化奖励）
反馈：P ← P - η·∂F/∂P，Γst随交互自动收紧，CST随时间增长
```
