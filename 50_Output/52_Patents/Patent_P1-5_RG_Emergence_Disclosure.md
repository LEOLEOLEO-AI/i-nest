## 一、技术领域

本发明属于复杂网络分析技术领域，具体涉及一种基于重整化群（Renormalization Group, RG）
的小世界网络涌现阈值检测方法，适用于芯片互联网络、神经网络架构、分布式计算拓扑等
需要判定网络规模是否达到涌现临界点的工程场景。

---

## 二、背景技术

小世界网络（Watts & Strogatz, Nature 1998）具有高聚类和短路径的双重特性，被广泛
认为是最优的互联拓扑结构。然而，现有技术存在以下缺陷：

1. **缺乏定量涌现判定标准**：小世界系数 sigma > 1 仅表示网络具有小世界特性，无法
   判定网络规模是否已达到产生"超线性优势"的涌现临界点。

2. **无法区分"弱涌现"与"强涌现"**：现有方法（如 sigma 值比较、聚类系数阈值等）
   无法区分仅仅具有小世界拓扑（弱涌现）和结构效率显著超越随机网络（强涌现）。

3. **缺乏尺度不变的检测框架**：不同规模的网络需要不同的判定标准，现有技术缺乏
   统一的、跨尺度的涌现判定方法。

本发明基于重整化群理论，通过 RG 流分析 + 结构效率比 S_eff/S_eff_rand 的双重判据，
首次实现对网络涌现阈值的精确、定量检测。

---

## 三、发明内容

### 技术方案

本发明提出一种基于 RG 重整化群的小世界网络涌现阈值检测方法，包括以下步骤：

**步骤 1：生成或获取目标网络的邻接矩阵。**
对于 N 个节点的网络，构建 Watts-Strogatz 拓扑（参数 k=16, p=0.05），
或加载实际网络的连接矩阵。

**步骤 2：计算结构效率 S_eff = C × E_glob。**
其中 C 为聚类系数（Watts-Strogatz 1998），E_glob 为全局效率（平均最短路径倒数）。
同时计算同等规模随机网络（Erdos-Renyi）的 S_eff_rand 作为基线。

**步骤 3：判定涌现。**
若 S_eff/S_eff_rand > 1.5，则判定该网络规模已达到涌现阈值；
若 S_eff/S_eff_rand > 5.0，则判定为强涌现。

**步骤 4：RG 流验证（可选增强）。**
对初始网络进行 degree-based 粗粒化（保留 50% 高度数节点），
计算每一步的 sigma 值。若 RG 流收敛至固定点 sigma* ≈ 4.97 ± 0.5，
且分形维数 d_f 在 1.2-1.5 范围内，则确认涌现检测有效。

### 核心公式

```
涌现判据：  S_eff/S_eff_rand > 1.5        （涌现阈值）
           S_eff/S_eff_rand > 5.0        （强涌现）
           
RG 验证：   |sigma_RG_step - 4.97| < 0.5  （固定点确认）
           1.2 < d_f < 1.5               （分形维数约束）
```

### 有益效果

1. **首次实现涌现阈值的定量检测**：将涌现从定性概念转化为定量判据（S_eff/Rand > 1.5）。
2. **跨尺度统一**：RG 流分析提供尺度不变的验证框架，适用于 N=16 至 N=10^6 的网络。
3. **双重验证机制**：结构效率比 + RG 固定点收敛，误判率低。
4. **工程适用性强**：判据简单（仅需 C 和 E_glob），计算复杂度 O(N^2)，适用于
   大规模芯片互联网络的实时检测。

## 四、具体实施方式

### 实施例 1：芯片互联网络涌现检测

某 256 芯粒的 SDI（软件定义互连）网络，k=16，p=0.05。

步骤 1：构建邻接矩阵，得到 N=256，k=16 的 WS 拓扑。
步骤 2：计算 C=0.614，E_glob=0.373，S_eff=0.229。
         同等规模随机网络 C_rand=0.062，E_rand=0.465，S_eff_rand=0.029。
步骤 3：S_eff/S_eff_rand = 7.90 > 1.5 → **涌现确认**。
         7.90 > 5.0 → **强涌现**。
步骤 4：RG 流：256→128→64→32，sigma 从 7.49 收敛至 4.97，d_f=1.29。
         sigma* 误差 = |4.97-4.97| = 0 < 0.5 → RG 验证通过。

结论：该 256 芯粒网络已达强涌现，拓扑结构效率为随机网络的 7.9 倍。

### 实施例 2：涌现阈值发现

对 N=16, 32, 48, 64 四组网络进行扫描：

N=16: S_eff/Rand = 1.00 → 未涌现（低于 1.5）
N=32: S_eff/Rand = 1.47 → 接近涌现（低于 1.5）
N=48: S_eff/Rand = 1.97 → **涌现确认**（首次突破 1.5）
N=64: S_eff/Rand = 2.32 → 涌现确认

结论：N=48 为本组参数下的涌现阈值。

### 实施例 3：非涌现网络排除

某 N=32 的随机互联网络，C=0.08，E_glob=0.45，S_eff=0.036。
S_eff_rand（同规模随机）=0.040。
S_eff/S_eff_rand = 0.90 < 1.5 → 未涌现。
RG 流：sigma 不收敛至固定点 → 验证一致。

## 六、权利要求

1. 一种基于重整化群的小世界网络涌现阈值检测方法，其特征在于，包括：
   a) 计算网络的结构效率 S_eff = C × E_glob；
   b) 计算同规模随机网络的结构效率 S_eff_rand；
   c) 当 S_eff/S_eff_rand > 1.5 时，判定该网络达到涌现阈值。

2. 根据权利要求 1 所述的方法，其特征在于，还包括 RG 流验证步骤：
   对网络进行 degree-based 粗粒化，计算各 RG 步的 sigma 值，
   当 sigma 收敛至 4.97 ± 0.5 时确认涌现检测有效。

3. 根据权利要求 2 所述的方法，其特征在于，RG 粗粒化保留 50% 高度数节点，
   边重连保持平均度 k 不变。

4. 根据权利要求 1 所述的方法，其特征在于，当 S_eff/S_eff_rand > 5.0 时，
   判定为强涌现。

5. 一种计算机可读存储介质，其上存储有计算机程序，该程序被执行时实现
   权利要求 1-4 任一项所述的方法。
