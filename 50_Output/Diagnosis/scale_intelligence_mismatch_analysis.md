## 【第 3 部分】理论的规模要求分析

### 3.1 什么是"自组织临界态"(SOC)?

**数学定义**：
```
SOC = 系统在无外部调谐的情况下，自发进入临界点附近
的状态，表现为：
1. 幂律分布的雪崩事件：P(s) ∝ s^(-α), α ≈ 1.5
2. 多尺度自相似性：缩放不变性
3. 1/f 噪声：功率谱 P(f) ∝ f^(-γ)
```

### 3.2 SOC 需要的最小规模

**物理学的普遍共识**：
```
SOC 现象的观测需要：
- 足够多的相互作用单元（通常 > 1000）
- 足够长的时间序列来显示幂律（至少 10^4 步）
- 足够的动力学复杂性（混沌或临界）

典型例子：
- 自组织临界沙堆：> 10^4 粒沙子
- 地震分布：> 10^4 次地震
- 神经雪崩：> 10^3 个神经元
```

**结论**：
```
C.elegans 的 302 个神经元
  ↓
处于"太小"看不到真正的 SOC 现象的范畴
  ↓
即使形式上满足"幂律指数 α = 1.5"
也只是"巧合"而非"真正的临界态"
```
