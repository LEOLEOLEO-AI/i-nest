## 【第 2 部分】各类仿真的具体评估

### 2.1 v25 仿真的问题

**当前地位**：
```
论文 V25 中的"多物种 SDI 验证"
├─ 声称：证明了 SOC、TCC 范式
├─ 实际数据：random.beta()、random.lognormal() 生成
└─ 结论：❌ 无法支撑任何结论
```

**具体问题**：
```python
# sdi_network_v30.py 的代码
def _synthetic(self):
    for region, size in region_sizes.items():
        n_synapses = int(size * np.random.uniform(40, 80))  # ← 虚假数据
        clustering = float(np.random.beta(2, 5))             # ← 虚假数据
        small_world = float(np.random.lognormal(1.5, 0.3))   # ← 虚假数据
```

**能证明什么**：
```
✗ 不能证明 SOC 存在
✗ 不能证明 TCC 范式
✗ 不能证明 C.elegans 的神经拓扑特性
✗ 不能证明规模与智能的映射

✓ 只能证明："在这些参数范围内，生成的数值恰好落在目标范围内"
  → 这叫"参数匹配"，不叫"理论验证"
```

**实际用途**：
```
最多只能用于：
1. 代码原型 (proof-of-concept)
2. 算法框架验证
3. 数据结构测试
4. 性能基准测试 (benchmark)

完全不能用于：
❌ 学术论文的核心论证
❌ 理论验证
❌ 生物学结论
```

## 【第 3 部分】非真实数据仿真的坦诚评估

### 3.1 直言不讳的结论

| 仿真版本 | 真实数据比例 | 科学价值 | 学术可用性 | 投稿等级 |
|---------|-----------|---------|----------|---------|
| v25 | 0% | ❌ 无 | ❌ 不可用 | ❌ 拒稿 |
| v30 | ~30% | 🟡 有限 | 🟡 仅代码框架 | ❌ 拒稿 |
| v31 | ~70% | 🟠 中等 | 🟠 可讨论 | 🟡 大修 |
| 04仿真 | 100% | ✅ 高 | ✅ 直接可用 | ✅ 接受 |

### 3.2 为什么非真实数据仿真基本无意义？

**核心原因**：科学是关于"自然界实际发生的事"

```
非真实数据仿真 =
    选择参数分布 (random.beta, random.lognormal, ...)
    ↓
    让生成的数值恰好落在"目标范围"内
    ↓
    声称"这证明了理论"

这不是科学，这是"参数工程"

科学的仿真应该是：
    加载真实数据
    ↓
    运行可重复的算法
    ↓
    获得定量结果
    ↓
    与理论对标
    ↓
    生成新的可检验预测
```

### 3.3 具体的逻辑问题

**问题 1：循环论证**
```
我们说：C.elegans 应该有 σ ≥ 4.0（理论预测）
然后：生成 σ = 4.238 的参数
最后：声称"验证了理论"

实际上发生的：我们选择了一个恰好满足我们预期的参数集
这不是验证，这是"自洽的谎言"
```

**问题 2：无法排除 null 假设**
```
理论 A：iNEST 实现了 SOC
理论 B：任何在目标范围内的随机参数也"满足"指标

我们的仿真不能区分 A 和 B
因此无法证明 A
```

**问题 3：无法进行假设检验**
```
p-value 是什么？
→ 没有对照组，无法计算
→ 因此无法判断显著性
→ 因此无法说"证明"
```

## 【第 4 部分】非真实仿真的合法用途

### 4.1 可以用的场景（必须明确标注）

**合法用途 1：算法原型**
```
"为了演示算法框架，我们用合成参数进行了初步仿真。
这只是概念验证，不是生物学结论。"

用途：
✓ 代码可运行性验证
✓ 性能 benchmark
✓ 框架整合测试
```

**合法用途 2：理论可行性**
```
"理论上，如果参数满足 X 条件，应该产生 Y 结果。
下面用合成参数演示这个逻辑链。"

用途：
✓ 展示理论的内部一致性
✓ 推导数学帮助
✓ 算法复杂度分析
```

**合法用途 3：误差分析**
```
"为了测试算法在参数误差下的鲁棒性，
我们添加了 ±10% 的高斯噪声到连接矩阵。"

用途：
✓ 健壮性评估
✓ 误差传播分析
✓ 参数敏感性研究
```

### 4.2 不能用的场景（必须禁止）

```
❌ "证明了理论正确"
❌ "验证了生物假设"
❌ "支撑了论文的核心结论"
❌ "多物种验证"
❌ 任何带有绝对性的声称
```
