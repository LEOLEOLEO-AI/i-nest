# P0: CST理论验证工程

> **目标**: 建立N=64→16384的全尺度CST数值验证能力，精确测定涌现相变临界阈值θ_k和增益系数α。

## CST-01~05: 小中尺度验证 (N=64~512)

**时间**: 2026 Q3 - 2027 Q1

| 实验 | 关键发现 | 数据规模 |
|------|---------|---------|
| CST-01 Ring基线 | CST=1.0, 无涌现迹象 | 10^5 timesteps |
| CST-02 Butterfly | CST=1.8, 效率提升80% | 10^5 timesteps |
| CST-03 128+Butterfly+FEP | CST=2.5, 超线性初现 | 10^6 timesteps |
| CST-04 256 临界相变 | 首次观察到CST的急剧跳变 | 10^6 timesteps |
| CST-05 512 Fractal | θ_k精确测定: N_crit=384±32 | 10^7 timesteps |

## CST-06~10: 大尺度验证 (N=1024~16384)

**时间**: 2027 Q2 - 2029 Q1

| 实验 | 关键问题 | 计算资源 |
|------|---------|---------|
| CST-06 N=1024 | α拟合精度 | GPU×4, 72h |
| CST-07 N=2048 | I/CST R²验证 | GPU×8, 168h |
| CST-08 N=4096 | 涌现鲁棒性 | GPU×16, 336h |
| CST-09 N=8192 | 规模无关性 | GPU×32, 720h |
| CST-10 N=16384 | 全尺度相变曲线 | GPU×64, 1440h |
