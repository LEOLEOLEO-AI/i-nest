## 核心愿景

> 靠异步电路、脉冲驱动。把软件定义互连 (SDI) 的机制作为液态拓扑的化合件，从建立模型到进行仿真验证，再到定义能量和惊讶度驱动的自演化机制去不断扩大规模，进行演化，模拟出不同生物特征的神经网络，做出不同神经网络规模的产品，实现物理自演化的智能落地。

## 关键技术突破点

### 1. 异步电路 + 脉冲驱动
- 参考 NULL Convention Logic (NCL) 或 Click 元件
- 脉冲事件驱动 → 天然省电（无事件=无功耗）
- 自适应 tau 即异步电路的**可变延迟线**

### 2. SDI 化合键 = 液态拓扑
- SDI 将物理互连抽象为**可编程的化合键**
- 化学键（长程）↔ 电突触（短程）双通道映射到 FPGA 互连矩阵
- 运行时 FEP 驱动键的形成/断裂

### 3. 自演化机制
- **变分自由能**：$\mathcal{F} = D_{KL}[q(\theta)||p(\theta)] - \mathbb{E}_q[\ln p(D|\theta)]$
- **最小惊讶度**：$\text{surprise} = -\ln p(D|\theta)$
- **最小能量**：SDI 化合键的物理功耗作为正则项

## 与 LNN 生态的对接

| LNN 组件 | iNEST 对应 | 映射方式 |
|----------|-----------|----------|
| 漏电流 $-\frac{1}{\tau}x$ | 自适应 tau 衰减 | 直接数学映射 |
| 非线性耦合 $f$ | SDI 化合键函数 $f_\theta$ | 结构映射 |
| 有界激活 $A$ | FEP 自由能盆地 | 概念映射 |
| ODE Solver | 异步脉冲电路 | 物理映射（非直接数学映射） |
