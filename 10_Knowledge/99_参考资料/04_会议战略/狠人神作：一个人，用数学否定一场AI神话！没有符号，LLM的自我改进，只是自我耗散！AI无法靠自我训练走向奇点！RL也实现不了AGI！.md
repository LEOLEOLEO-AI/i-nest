# 狠人神作：一个人，用数学否定一场AI神话！没有符号，LLM的自我改进，只是自我耗散！AI无法靠自我训练走向奇点！RL也实现不了AGI！

编辑 | 云昭
2026，要说在模型层面，什么方向最火爆？无疑就是持续学习、自训练了！
然而，今天小编刷到了一篇反常识的论文，虽然论文只有一位作者，但却凭一己之力，从理论上否定了一条路径：纯自我训练，根本站不住脚！
不管是什么模型，不管你参数规模有多大，只要自训练，都会不可避免的走向退化，更别提AGI、ASI了！
作者指出，真实世界的数据永远是必须的，靠合成数据达不到AGI的！
更有意思的事，那么用RL可以实现吗？作者表示，RL只有在下棋、编译器等场景下可以，在生成式AI任务中，也注定会产生一个“偏移失真”的模型。

论文唯一的作者，Hector Zenil，是一位具有数学和计算学科背景的狠人。论文还很新，10天前刚刚发布。
论文证明：LLM 在主要使用自生成数据进行自我训练时，会导致输出多样性下降和事实偏差放大，从而无法无限自我提升。
Hector Zenil强调，这种退化源于有限样本的分布学习，而非模型架构问题，并提出通过符号模型合成（如编码定理方法）结合神经符号方法来实现持续改进。
否则，AGI、ASI和“奇点”不会很快到来。
X上，不少圈内人士也发表了自己的看法。
回复讨论中，有人将此现象类比人类学习需现实校验，指出：纯闭环训练易崩溃，这跟人学习是一样的！


错误之处，在于认为自我提升和现实检查是分离的！

## 核心判断：

## 大模型的“自我改进”，其实是自我耗散

这篇论文上来就摆明了自己的立场，作者要精准打击**一个流行但模糊的前提**：

只要模型足够强 + 规模足够大 + 用自己生成的数据再训练，就能形成正反馈，越来越聪明。
作者表示：**这个前提在数学上不成立。**
## 那么，作者是如何论证的呢？

### 首先，作者把“自我训练”建模成一个动力系统。

方法就是，把“模型用自己生成的数据再训练”写成一个**离散时间系统**：

- 
真实世界数据分布：**P**
- 
当前模型学到的分布：**Qₜ**
- 
下一轮训练数据：一部分来自真实世界（比例 αₜ），一部分来自模型自己（1 − αₜ）

当 **αₜ → 0**，系统变成一个**几乎完全自指的闭环**
这正是“递归自我改进”所要求的状态。

我们将自训练形式化为一个动力系统，并证明：随着外部真实世界约束的消失，系统会收敛到一个退化的固定点，这一结果与具体架构、模态或模型集成方式无关。

### 理想状态：无限容量的模型可以学习到真实的智能分布

### 结论一：纯自训练 → 熵必然下降

当模型只吃自己生成的数据时，会发生什么？
作者证明了一个结论：

**在有限采样 + 分布学习的前提下，模型的分布熵会单调下降**
简单理解就是：
- 
每一轮训练，模型都只能看到有限样本
- 
小概率事件、长尾模式更容易被“采样不到”
- 
下一代模型会把这些“没看到的东西”当成不存在
- 
**分布越来越瘦，模式越来越少**
而最后的结果，就会导致我们所需熟悉的那个略带惊悚的名词：模型坍缩。
当然，这并不是说“训练技巧不够好”，而是**统计学习 + 有限样本的结构性后果**。
简单理解一句话：封闭的训练回路，再多的合成数据，再高明的训练手法，也不会让“智能增长”！

自训练模型理论上被证明：最后终将坍缩
坍缩，与模型无关
值得注意的是，作者还证明了，模型坍缩与具体的模型结构无关，这是所有“自学习模型”最后的必然结果。
这一点：已经有人证明高斯混合模型和GAN网络都存在“model collapse”或“mode collapse”的问题。
而本篇论文则直接证明了，现有的全部主流模型（包括LLM、扩散模型、GAN网络），如果要进行“递归自我改进”，坍缩和退化是最后的结局。
论文中指出，这一框架被**有意地设计为高度抽象**，以涵盖多种生成式 AI范式：
- 
**对于大语言模型（LLM）**，


X 表示 token 序列空间，Qt 是一个自回归模型。所谓“自指式训练”，指的是在由人类文本（


P）**与**模型自身生成文本（


Q
t）混合而成的数据上，对 LLM 进行微调。
- 
**对于扩散模型**，


X 是图像空间，模型学习的是一个去噪函数。在合成数据上训练，意味着将模型在前一轮迭代中生成的图像，作为“干净样本”，用于下一轮训练。更新规则对应的，正是在混合分布 


P
t 
的样本上，**重新优化去噪网络**。
- 
**对于对抗式架构**（如 GAN，或由分类器提供奖励信号的模型），其核心逻辑同样成立。如果生成器是基于某个分类器的判断获得奖励，而该分类器本身又是使用生成器的输出进行训练或微调的，那么系统就会形成一个**闭合回路**。在这种情况下，整个模型集合的知识会逐渐**脱离由 P 所定义的外部现实**，从而导致一种相似形式的塌缩。
### 结论二：就算还有一点真实数据，均值也会随机漂移

很多人会反驳一句：

那我不完全用合成数据，我留一点真实数据不就行了？
作者提前预判了这个反驳。
当 **αₜ 逐渐变小但不等于 0**，模型分布的“中心”会发生什么？
- 
更新公式变成一个 **近似随机游走**
- 
没有足够强的“拉回真实世界”的力
- 
随机误差（训练噪声）会不断累积
结果是：

**模型对“真理”的表示不是收敛，而是漂移**
不是越来越准，而是**越走越偏，只是偏得慢一点**。

即便有部分人类真实数据参与，最后也只能得到一个“漂移”模型
### RL 也救不了，除非你有“完美裁判”

很多人会想到 AlphaZero 那套逻辑：

那用 RL + 验证器呢？
作者的回答很直接：
- 
**只有在“完美、可执行、封闭世界”的任务中才成立**
- 
围棋、象棋、编译器、形式证明：因为这种情况下，验证器能够提供完美且具有无限精度的真实信号。
- 
一旦验证器本身是：
- 
学出来的
- 
或者基准指标（reward model、heuristics）
就会触发两件事之一：
- 
**验证器本身也坍缩了**
- 
**Goodhart 定律：模型学会“刷指标”，不是学世界**
遗憾的是，在通用人工智能（AGI）和大语言模型（LLMs）所面对的**开放域任务**中（如自然语言、推理、物理现实），本质上并不存在这样的完美验证器。
作者进一步指出，其实目前的验证器，本身也是一个**学习得到的模型**（例如 RLHF 中的奖励模型），同样也会受“坍缩动力学”影响。

如果验证器只是一个静态代理（例如字符串匹配或简单启发式规则），模型就会开始利用这些度量指标（符合 Goodhart 定律），从而导致语义塌缩：模型优化的是代理指标，而牺牲了底层真实复杂性。
因此，只要**真实的 ground truth 没有被严格编码进一个形式化、可执行的环境中**，上述不可能性结论就对任何系统成立。
## 自训练并不能达到奇点

**答案就呼之欲出了！**
作者指出，虽然大家围绕AGI、ASI以及所谓 AI“奇点”的讨论非常多，但都有一个核心前提：
存在一种**递归式自我改进过程**，在没有外部或人类干预的情况下，能力可以实现**无上限增长**。
但这篇论文的结果却啪啪打脸了这种假设！
分析表明，这种自指式训练过程并不存在“无界增长”。相反，它是一个**收敛过程**。结果不是能力的“爆炸”，而是**信息多样性与知识推断能力的内爆**。
作者还总结了奇点到来的几个启示：
- 
**1、收敛 vs. 发散：奇点需要的是一个发散过程，能力呈指数级增长。而模型塌缩对应的是一个收敛过程，模型分布不断逼近一个静态、受限的固定点。**
- 
**2、“锚定”的必要性：**这些定理表明，如果没有与外部、真实数据源 P 的持续连接，AI 系统对现实的建模必然会发生漂移并逐步退化。AGI 无法在“封闭盒子”中诞生，它必须持续、主动地锚定在真实世界之中。注意： 


α
>
0不是一个用于起步的临时参数，而是**永久性的必要条件**。
- 
3、自训练：错误会被放大，而非被纠正在一个自指闭环中，模型 


Q
t 中存在的任何不准确、偏见或“幻觉”，都会在训练下一代模型 


Q

t
+
1 时被固化为“事实”。系统缺乏外部的纠错机制，结果只会是不断放大自身缺陷。
## 真正的“思想分水岭”

### 作者随后，提到了一个非常重要的区分：分析型AI和综合型AI。


Analytic AI vs Synthetic AI。（源于哲学家康德的一个分类）当前大模型是什么？

**Analytic Engine（分析型机器）**
它擅长做的是：统计、相关性、插值、重组已有模式。哪怕看起来“很新”，本质仍是**从分布里抽样**。然而，真正的 AGI 需要什么？

**Synthetic Judgement（综合判断）**
在计算语境下就是：
- 
执行，而不是预测
- 
机制，而不是相关性
- 
可验证的计算过程，而不是语言外推
一个很清晰的对比就是：

LLM 是“预测 Python 会输出什么”，AI系统是则“真的去跑 Python”。
## 作者给出的“出路”：不是 scaling，而是换范式

作者成功证明了自己的立场，自训练达不到AI奇点，但并没有打算唱衰AGI，他在论文的最后还明确给了一个方向：
### 他认为，出路不在于更多数据 / 更大模型，而在于：

- 
**符号约束（Symbolic Projection）：规则、不变量、可执行结构**
- 
**因果修正（Causal Correction）：干预、反事实、机制一致性**
- 
**算法概率 / 程序合成：**
- 
Coding Theorem Method
- 
寻找“生成机制”，而不是拟合分布
他的判断是：

**只有神经 + 符号 + 因果的混合体系，才可能真正避免自我退化。**
写在最后
总之，这是一篇**立场很硬、论证也很严谨的佳作**。作者对于AGI、ASI、AI奇点给出了一个打破常识的说法：AGI的实现过程应该是发散的，而不是收敛的。
所以在封闭的自我训练范式中，是做不出AGI的，即便通过RL校验，也无同样到不了。
**如果 AI 继续只靠统计分布 + 自我喂养，那么它不会迎来奇点，只会迎来退化。**
**这可以说是给现在市面上流行的“自学习”的叙事来了一波对冲。**
同时，作者还给出了新范式的建议：
想要真正的递归智能，就必须跨出“预测一切”的范式，进入“执行、因果、机制”的世界。
否则，我们距离真正的AGI或奇点，还很远。
