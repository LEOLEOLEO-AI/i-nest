### **📚 01 复杂网络 - 理论基础与拓扑特性**

#### **复杂网络基本结构测度**

| 测度类别 | 定义 | 公式/说明 |
| :--- | :--- | :--- |
| **度** | 节点连接边的数量 | \( k_i = \sum_j a_{ij} \) |
| **度分布 P(k)** | 度为 k 的节点占比 | 描述网络异质性 |
| **度相关性** | 高度节点是否倾向于连接高度或低度节点（同配/异配） | 通过度-度相关系数衡量 |
| **最短路径长度 \( d_{ij} \)** | 节点 i 到 j 的最少边数 | 反映信息传递效率 |
| **网络直径** | 所有节点对间最短路径的最大值 | \( D = \max_{i,j} d_{ij} \) |
| **介数中心性** | 经过某节点（或边）的最短路径比例 | 衡量节点/边的“枢纽”作用 |
| **聚类系数 \( C_i \)** | 节点 i 的邻居间实际连接数与最大可能连接数之比 | \( C_i = 2e_i / [k_i(k_i-1)] \)，\( e_i \) 为邻居间边数 |
| **全局聚类系数** | 网络中三角形数量与连通三元组数量之比 | 反映局部稠密性 |
| **模体（Motif）** | 网络中显著高频出现的小规模子图模式 | 功能模块的结构基础 |
| **社团结构** | 节点被划分为内部连接紧密、外部连接稀疏的群组 | 常用模块度（Modularity）量化 |
| **图谱（Spectrum）** | 邻接矩阵或拉普拉斯矩阵的特征值集合 | 揭示网络整体动力学性质 |

#### **现实世界网络的拓扑结构**
- **小世界特性**：通常具有高聚类系数与短平均路径长度；局部紧密连接，全局信息传递高效。
- **无标度分布**：度分布服从幂律 \( P(k) \sim k^{-\gamma} \)（\( 2 < \gamma < 3 \)），少数“枢纽”节点拥有极高连接度，而大多数节点度较低。

#### **经典网络生成模型对比**

| 模型类型 | 核心机制 | 拓扑特征 | 局限性 |
| :--- | :--- | :--- | :--- |
| **Erdős–Rényi 随机图** | 每对节点以概率 p 独立连边 | 泊松度分布，低聚类，长路径 | 无法解释现实网络异质性 |
| **广义随机图** | 指定度序列后随机匹配 | 可控度分布，无相关性 | 忽略结构相关性 |
| **小世界网络（WS）** | 规则环+随机重连 | 高聚类 + 短路径 | 度分布窄，非幂律 |
| **静态无标度网络** | 预设幂律度序列后配置模型 | 幂律度分布，无演化机制 | 无法反映增长过程 |
| **演化无标度网络（BA）** | 网络增长 + 优先连接 | 幂律度分布（\( \gamma=3 \)） | 忽略地理、成本等约束 |

#### **加权网络**

边带有权重，需扩展结构测度：

| 加权测度 | 定义 | 公式/说明 |
| :--- | :--- | :--- |
| **加权度（强度）** | 节点连接边的权重总和 | \( s_i = \sum_j w_{ij} \) |
| **加权聚类系数** | 考虑边权重的三角形闭合强度 | - |
| **加权最短路径** | 路径长度为权重倒数之和（如时间、成本） | - |

#### **静态和动态鲁棒性**
- **无标度网络**：对随机失效（随机移除节点）高度鲁棒，但对蓄意攻击（移除高度节点）极为脆弱。
- **静态耐受性**：
  - **关联性影响**：无关联网（度无相关）临界阈值可解析求解（如 Molloy-Reed 条件）；关联网络中同配性增强鲁棒性，异配性降低鲁棒性。
  - **攻击耐受性**：提升策略包括增加冗余连接、限制枢纽节点负载、引入模块化设计。
- **动态效应**：
  - **相继故障**：局部过载引发级联失效，常用负载-容量模型模拟（节点负载超阈值则失效，负载重分配至邻居）。
  - **通信拥塞**：数据包在网络中路由时，枢纽节点易成为瓶颈，优化策略包括自适应路由、流量控制。

#### **传播过程**

| 传播类型 | 关键机制 | 说明 |
| :--- | :--- | :--- |
| **传染病传播** | **均匀混合假设** | 传统 SIR/SIS 模型假设个体充分混合，忽略网络结构。 |
|  | **无关联网络** | 在配置模型下，流行阈值 \( \lambda_c = \langle k \rangle / \langle k^2 \rangle \)，无标度网络因 \( \langle k^2 \rangle \to \infty \)，故 \( \lambda_c \to 0 \)，即任意弱传染均可爆发。 |
|  | **关联网络** | 同配性降低传播阈值，异配性提高传播阈值。 |
|  | **疾病波动** | 有限系统中，即使平均再生数 \( R_0 < 1 \)，仍可能出现大型暴发（随机共振效应）。 |
| **谣言传播** | **传播规则** | 谣言传播者在接触已知情者后停止传播。 |
|  | **经典模型** | Daley-Kendall 模型（DK 模型），网络结构显著影响终态规模与速度。 |

#### **同步与集体动力学**
- **同步**：耦合振子相位或状态趋于一致的现象，广泛存在于电力网、萤火虫闪光、神经节律等系统。
- **主稳定函数方法**：将同步稳定性问题转化为单个振子受扰动的 Lyapunov 指数计算，依赖耦合矩阵的特征值谱。
- **网络同步倾向**：

| 耦合矩阵类型 | 特征谱性质 | 同步条件 |
| :--- | :--- | :--- |
| 实对称（无向加权） | 实特征值 | \( \alpha \lambda_2 > \alpha_{\text{crit}} \)（\( \lambda_2 \) 为代数连通度） |
| 非对称（有向） | 复特征值 | 需满足 MSF 在复平面上的稳定性区域覆盖所有 \( \alpha \lambda_i \) |
- **耦合振子与混沌同步**：
  - **Kuramoto 模型**：相位振子同步，临界耦合强度与网络结构相关。
  - **混沌系统**（如 Rössler）：需要更强耦合，同步窗口受网络异质性抑制。

### **🔄 02 图表征学习 - 低维嵌入技术**

**核心目标**：将图中节点嵌入低维表征并有效保留图的结构信息。

#### **传统图嵌入方法**

| 方法 | 核心思想 |
| :--- | :--- |
| **MDS 多维尺度变换** | 通过 Isomap 学习表征近似保留低维空间中节点间地理距离。 |
| **LLE 局部线性嵌入** | 假设每个节点及其邻居节点都位于或接近一个局部的线性流形，每个节点都可以通过其邻居节点重建。 |
| **LE 拉普拉斯特征映射** | 通过 ε-邻域或 K-近邻构建图，利用热核选择图中两个节点的权重，通过基于拉普拉斯矩阵的正则化得到节点表征。 |

#### **现代图嵌入方法**

| 类别 | 子方向 | 典型方法 |
| :--- | :--- | :--- |
| **保留图结构和属性的图表征学习** | **保留图结构** | - **Deep Walk**：随机行走+Skip-Gram<br>- **node2vec**：二阶随机行走，广度优先抽样+深度优先抽样<br>- **LINE**：保留一阶接近度和二阶接近度<br>- **SDNE**：用具有多个非线性层的深度自编码器架构来保留二阶接近度 |
|  | **保留图属性** | 重于保留所有类型图的传递性以及有符号图的结构平衡性；**HOPE**：总结了4种测量方法，利用广义SVD问题对高阶接近度进行了因子化。 |
| **带有侧面信息的图表征学习** | **带有节点内容** | **TADW**：在学习节点的低维表征时会考虑与节点相关的丰富信息（如文本）。 |
| **保留高级信息的图表征学习** | - | 保留网络结构、建立节点表征和目标任务之间的联系。 |

#### **可扩展性挑战与解决方案**
- **核心挑战**：
  - 大量的内存需求：计算整个邻接矩阵和图节点嵌入需要巨大的内存空间。
  - 低效的梯度更新：训练需要递归更新图中的每个节点，对大规模图不可行或低效。
- **解决方案**：**抽样**（选择所有样本的一个分区来代表整个样本分布），关键问题包括：
  - 如何设计高效的抽样算法？
  - 如何保证抽样质量？

#### **可解释性**
- **重要性**：可信性、公平性、对抗性攻击的鲁棒性、后门攻击的鲁棒性。
- **解释维度**：
  - 哪些输入边更重要？
  - 哪些输入节点更重要？
  - 哪些节点特征更加重要？
  - 什么样的图形模式可以最大限度地提高某一类的预测能力？
