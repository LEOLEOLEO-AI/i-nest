# 『麻省大学夏强飞 Nat. Electron.』基于忆阻器的细胞神经网络：实现高速像素内计算的新突破

在传统冯·诺依曼架构中，数据在存储器与处理器之间频繁搬运，导致能耗高、延迟大，难以满足实时视觉计算的需求。细胞神经网络作为一种受生物视网膜启发的架构，通过局部连接与模拟信号处理实现大规模并行计算，长期被视为图像处理与偏微分方程求解的理想硬件基础。然而，传统 CMOS 实现往往依赖复杂的混合信号电路，面积大、功耗高，难以扩展。 麻省大学夏强飞教授团队提出一种 基于忆阻器的细胞神经网络（CeNN）硬件系统，结合数字孪生模拟平台、65 nm CMOS 芯片与多级非易失忆阻器阵列，实现了高速、低功耗的像素内模拟计算。该系统可执行边缘检测、线检测、背景移除、连通性分析等任务，并可用于求解三维 Navier–Stokes 方程，展示了 CeNN 在图像处理与科学计算中的巨大潜力。⚡ 一句话
通过忆阻器与 CeNN 的深度融合，本文实现了真正意义上的高速、低功耗、可扩展的像素内模拟计算。
🔑 研究亮点
- 仿生 CeNN 架构实现大规模并行模拟计算：  每个像素对应一个细胞单元，局部连接与连续时间动力学使网络天然适合图像处理与 PDE 求解。
- 忆阻器作为可编程模拟突触：  采用非易失忆阻器实现模拟权重存储，具备高线性度、低功耗与零静态能耗，显著优于传统 CMOS 突触。
- 数字孪生平台支持全流程模拟与硬件控制：  基于 Python 的 HyE 平台可进行任务配置、网络仿真、硬件控制与结果可视化，实现软硬件一体化开发。
- 支持 PDE 求解的模拟计算能力：  CeNN 可求解二维热方程与三维 Navier–Stokes 方程，在空间与时间分辨率提升时仍保持高效率。
- 65 nm CMOS + 忆阻器混合硬件验证：  构建 5×5 CeNN 芯片与外接忆阻器阵列，实现垂直线检测与边缘检测，收敛时间低于 50 微秒。
- 忆阻器桥式突触具备低温稳定性：  实验与 SPICE 仿真表明忆阻器突触在 82 K 下仍保持良好线性度，而纯 CMOS 突触在低温下严重退化。📊 图示要点

- 扩展到更大规模的 CeNN 阵列，实现更复杂的视觉任务 
- 将忆阻器突触集成到片上，实现全模拟 CeNN 芯片 
- 引入学习算法，实现可训练的 CeNN 权重更新 
- 将 CeNN 用于实时视频处理、机器人视觉与科学计算该研究展示了忆阻器与 CeNN 结合的巨大潜力，为下一代高速、低功耗、可扩展的模拟视觉计算系统奠定了基础。
