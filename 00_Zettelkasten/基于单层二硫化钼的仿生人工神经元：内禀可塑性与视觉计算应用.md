# 基于单层二硫化钼的仿生人工神经元：内禀可塑性与视觉计算应用

## 摘要

📄 **研究背景与核心突破**   - **发表信息**：2025年8月11日在线发表于《Nature Electronics》，接收日期2024年1月23日，接收日期2025年7月10日   - **核心创新**：首次在器件层面实现基于晶圆级单层二硫化钼（MoS₂）的仿生人工神经元，通过调节膜电位

基于晶圆级单层 MoS₂，作者首次在器件层面实现了具备内禀可塑性的人工神经元，并展示了其在光适应和类脑视觉计算中的潜力。亮点如下：

1.引入“内禀可塑性”机制：                      
传统突触器件多强调突触可塑性（STDP、Pavlov 型学习），本文首次强调神经元层级的可塑性——通过调节膜电位实现神经元本身的适应性。

2.器件级神经元构建：                      
使用单层 MoS₂薄膜同时实现 DRAM 与 CMOS 反相器集成，形成可积分激发的人工神经元。

3.视觉系统仿真：                      
具备对光敏性的动态调节能力，模拟人眼的光适应（明视觉/暗视觉），实现了神经元层面的类脑视觉功能。

4.阵列级演示：                      
制备 3×3 光感神经元阵列，展示了图像光编码与视觉适应，为未来更大规模二维材料类脑阵列奠定基础。

摘要

能够精确模拟多种神经元行为的神经形态硬件可用于开发边缘智能。人们已经探索了融合突触可塑性（即增强或减弱突触连接的自适应变化）的硬件，但要模拟完整的学习和记忆过程，需要多种可塑性机制的相互作用，包括内在可塑性。本文展示了如何通过结合基于晶圆级单层二硫化钼薄膜的动态随机存取存储器和反相器来创建一个积分激发神经元。在该系统中，可以调节动态随机存取存储器电容器中的电压（即神经元膜电位），以模拟内在可塑性。该模块还可以通过动态调节人类视觉系统的光敏感度来模拟其明视和暗视适应。我们制作了一个
3 × 3 的光感受器神经元阵列，并演示了光编码和视觉适应。我们还使用该神经元模块模拟了一个用于图像识别的仿生神经网络模型。

 

研究背景和主要内容

随着测量场景复杂度和数据量的不断提升，边缘智能硬件的开发面临着速度和功耗方面的挑战。生物神经系统在感知和识别复杂环境方面比现有的人工视觉系统表现出更高的效率和更低的功耗。然而，视网膜传感器与仿生神经网络（BioNNs3）相结合，或许可以成为提高人工视觉系统效率并降低功耗的解决方案。

利用电脉冲在神经元之间传递信息是最接近生物神经活动的形式，而对电脉冲进行精确的时间处理是 BioNN
中信息编码的关键策略。然而，目前基于互补金属氧化物半导体电路的脉冲编码方案消耗了大量的硬件资源，而将时间稀疏性编码策略融入神经形态硬件仍然具有挑战性。人类视觉系统能够对接收到的视觉刺激进行预处理，然后再将其发送到颞叶进行学习和识别等高级处理。当应用于人工视觉系统时，人类视网膜细胞的视觉适应功能可以降低硬件复杂性和功耗。

高级视觉处理与神经活动相关，包括突触和神经元内在特征的可塑性调节，这些对于学习、记忆和识别至关重要。人们已经使用各种突触设备（如电阻随机存取存储器、相变存储器和离子晶体管）研究了突触可塑性的多种机制，包括长期增强、短期可塑性和脉冲时间依赖性可塑性
(STDP) 。然而，单靠突触可塑性并不能阐明学习和记忆过程的全部范围。与神经元膜电位变化相关的几种现象，包括兴奋性突触后电位 (EPSP)
的扩增、脉冲阈值的调整和静息电位的变化，在神经元信号的传输和处理中发挥作用。这些内在的可塑性现象存在于几乎所有类型的神经元中，并且是生物神经系统学习和适应性不可或缺的一部分。

目前已经对神经内在可塑性的硬件实现进行了一些初步探索。一种混合互补金属氧化物半导体/电阻随机存取存储器神经元电路已经开发出来，它可以通过模拟神经元的内在可塑性来自适应地调节输入刺激，从而精确控制脉冲发放率。一种神经晶体管可以模拟神经元的非线性积分过程，并通过调整脉冲发放阈值来实现内在可塑性。据报道，使用相变存储器器件可以实现基于内在可塑性的神经元振荡行为和脉冲阈值调制。这些进展凸显了内在可塑性在构建
BioNN
计算架构方面的潜力，特别是在增强神经网络的学习稳定性和环境适应性方面。然而，为了有效可靠地利用生物神经元的内在可塑性，需要开发具有更高稳定性、效率和晶圆级集成度的简化神经形态硬件。

在本文中，我们报告了一种神经元模块，它结合了基于晶圆级单层二硫化钼（MoS2）薄膜的动态随机存取存储器（DRAM）和反相器。二维半导体，如
MoS2，具有高效的光-物质相互作用、宽的光吸收光谱范围和高度可调的光电特性。近年来，二维半导体材料的晶圆级生长和小规模集成电路的制造取得了进展，推动了基于二维半导体的边缘智能向系统级发展。通过利用
MoS2 的可调光电特性，我们的神经元模块表现出与人类视觉系统类似的明视和暗视适应能力。我们还利用该方法创建了一个 BioNN
电路，该电路可以模拟神经元膜静息电位的固有可塑性，并执行首次脉冲时间 (TTFS) 编码。最后，我们使用该神经元模块模拟了一个用于图像识别的 BioNN
模型，并表明它可以简化或降低在硬件中实现仿生视觉系统的电路和处理算法要求。

图1c展示了神经元模块的光学显微图像和电路原理图。该模块由一个 DRAM 和一个基于 MoS2场效应晶体管的反相器组成。DRAM 单元用作积分器，其中
MoS2晶体管（T1）调节电容器上电荷的写入或擦除。反相器的输入连接到电容器，该电容器提供实现阈值触发功能所需的电压。确定 DRAM 的电容C约为 29
pF（补充图3 ）。MoS2晶体管和反相器的基本特性在补充说明1和补充图4 – 6中提供。此外，我们利用多功能神经元模块构建了一个 3×3 BioNN
电路（补充图1）。此外，补充图2展示了一个扩展的 BioNN，它包含具有不同功能的多功能神经元。

我们的多功能神经元模块应该像一个具有中频动态的神经元一样工作。电容器两端的电位 ( C ) 代表神经元细胞的膜电位
( V C )。为了复位神经元模块，将正选定电压 ( V S ) 施加到 T1 栅极以导通晶体管。同时，将写入电压 ( V W) 通过
T1的漏极写入电容器，将膜电位复位至静息电位。写入电容器的电荷Q reset可以使用V W  ×  C计算。

当没有输入刺激时，通过施加负V S可以将神经元的输出维持在低电压水平。此时，施加适当的光或电信号，驱动晶体管 T 1进入亚阈值区域，从而将
T 1上的漏电流调节到适当的水平。结果，随着存储的电荷通过 T 1放电，V C逐渐下降。

光信号的TTFS编码

在我们的设计中，第一个输出脉冲的时间包含了BioNN后续训练和学习所需的所有必要信息。这种编码策略被称为TTFS编码，比广泛使用的基于发放率的编码更高效。我们利用MoS2固有可塑性的调制和光敏特性，实现了具有时间稀疏性的光信号的TTFS编码。
