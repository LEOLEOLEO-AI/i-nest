# 四博士生冲刺会议 — 专利分工修正

> R1 原稿含10条权利要求，其中权1-7为方法，权8-10为系统。现拆分为两个独立专利，分工明确。

---

## 专利1：SDI调度方法专利 · 学生C

| 项目 | 内容 |
|------|------|
| **正式题目** | 一种支持拓扑中心计算的软件定义互连调度方法 |
| **专利类型** | 方法发明专利 |
| **当前状态** | 10条权利要求中的7条方法权利要求已由Genspark起草，待完善交底书 |
| **递交截止** | **2026年6月16日（明天）** |
| **优先级** | 🔴 最高 |

### 权利要求（权1-7，从原R1拆分）

| 权号 | 类型 | 核心内容 |
|------|------|----------|
| 1 | **方法独立** | S1接收原语序列 → S2原语优化(合并+融合) → S3双缓冲拓扑重构 → S4数据在拓扑中流动完成搬运与变换 |
| 2 | 从属 | 双缓冲：当前寄存器+影子寄存器，执行期间预加载下一拓扑 |
| 3 | 从属 | Route≡Transform映射：FFT蝶形/AllReduce/矩阵转置/前缀和等 |
| 4 | 从属 | 相邻拓扑合并优化 |
| 5 | 从属 | 原语融合消除中间数据 |
| 6 | 从属 | MPI/BLAS→原语兼容映射 |
| 7 | 从属 | 影子寄存器与当前寄存器间原子交换通路，纳秒级拓扑切换 |
