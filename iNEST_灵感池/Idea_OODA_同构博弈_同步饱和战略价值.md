## 一、问题

OODA（Observe-Orient-Decide-Act）是军事/竞争决策的核心框架。在多智能体同构博弈（所有 agent 结构相同、规则相同）中：

**当所有 agent 的 OODA 环同步饱和时，会发生什么？**

同步饱和 = 所有 agent 同时完成 OODA 环 → 同时行动 → 同时观察 → ...

## 二、假说

同步饱和有两种极限：

| 状态 | 特征 | 战略价值 |
|------|------|----------|
| **共振饱和** | 所有 agent 同步到同相位 | 群体协同最大化，但丧失多样性 |
| **混沌饱和** | OODA 环周期发散，无同步 | 个体独立性强，但无群体效应 |

**最优态在中间**：部分同步（集群内同步，集群间异步）→ 既有局部协同，又有全局多样性。

## 三、与 SDI 多脑框架的连接

SDI v31-v32 的多脑协作实验天然映射到这个问题：

- 每个 brain = 一个 OODA agent
- inter-brain bonds = agent 间的通信信道
- 同步程度 = inter-brain lock ratio

**预测**：中等 lock ratio（0.3–0.6）对应最优群体智能——既不过度同步（丧失多样性），也不完全独立（丧失协同）。

## 四、应用场景

- 无人机蜂群：部分同步 → 编队 + 独立避障
- 卫星星座：部分同步 → 覆盖优化 + 独立任务
- 投资组合：部分同步 → 跟投信号 + 独立判断
