# P4: 涌现度量与工具链

> **目标**: 开发iNEST专属的涌现度量工具箱和基准测试集, 建立可复现的涌现评估标准。

## CST-Meter v1.0

开源Python工具箱, 功能:

| 模块 | API | 输入 | 输出 |
|------|-----|------|------|
| cst_calc | 计算CST值 | network_graph | S_c, T_c, Γ_st, CST, I_est |
| phase_detect | 检测涌现相变 | cst_timeseries | θ_k, phase_label |
| avalanche | 神经雪崩分析 | spike_raster | 规模/时长分布, σ |
| fractal | 分形维度 | topology_matrix | D_f |
| emergence | 涌现评分 | network_dynamics | E_score ∈ [0,1] |

## Emergence-Bench

| Benchmark | 网络类型 | 涌现度量 |
|-----------|---------|---------|
| EB-SNN | 脉冲神经网络 | 雪崩统计+信息论 |
| EB-RC | 储备池计算 | 记忆容量+非线性度 |
| EB-Transformer | 注意力机制 | 涌现能力标度律 |
