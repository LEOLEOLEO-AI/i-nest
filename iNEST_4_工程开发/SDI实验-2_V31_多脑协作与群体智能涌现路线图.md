## 一、从 V30 到 V31：自然延伸的逻辑

### V30 已完成

V30 验证了**单脑内部的多功能区协同**：在一个 C.elegans 尺度的网络中，四个功能模块各自独立演化出小世界拓扑（σ > 4.0），并通过跨区连接实现功能融合：

| 脑区 | N | σ | 功能 | 表现 |
|------|---|-----|------|------|
| **视觉皮层** (vis) | 100 | 5.35 | 光信号处理 | — |
| **化学感知** (chem) | 100 | 5.82 | 化信号处理 | — |
| **联合皮层** (assoc) | 150 | 6.37 | 多模整合 | — |
| **运动输出** (motor) | 100 | 4.55 | 行为驱动 | — |
| **融合系统** | — | — | 光趋性+化趋性+模式补全 | photo:81.1%, chemo:78.6%, pattern:100% |

### V30 的核心洞察

> **不同功能脑区在相同的自组织原理（STDP+FEP+BCM）下，各自收敛到不同的拓扑特征，但均满足 σ>4.0 的小世界临界态。** 这意味着：多脑区协同不是通过统一调参实现的，而是每个脑区在与环境和彼此交互中自主找到各自的临界点。

### V31 的核心命题

> **如果单脑的多区协同已经涌现出基本智能行为（趋光、趋化、模式补全），那么多个独立演化的"脑"实例之间，能否通过竞争与协作，涌现出更高阶的群体智能？**

这是从 **"涌现智能"（Emergent Intelligence）→ "群体智能"（Collective Intelligence）** 的自然跨越。

### 2.2 三个子实验

#### 实验 2-1：资源竞争（Competition）

**场景**：两个独立的脑（Brain A & Brain B）共享有限资源（光/化学信号源），各自通过趋光/趋化行为争夺资源。

| 参数 | 设定 |
|------|------|
| 脑实例 | 2 × (vis+chem+assoc+motor) = 8 个脑区 |
| 连接组 | 各自独立初始化，独立演化 |
| 资源 | 两个光源/化学源，位置随时间变化 |
| 交互通道 | 共享环境状态（资源位置），无直接脑间连接 |
| 观测指标 | 资源捕获率、能耗效率、拓扑分化度 |

**核心假设**：竞争压力将驱动脑A和脑B各自演化出不同的拓扑策略（如一个侧重速度、一个侧重精度），形成**功能生态位分化**。

#### 实验 2-2：信号协作（Cooperation）

**场景**：脑A感知到信号后，通过共享通道将信息传递给脑B，脑B据此调整行为。模拟"观察学习"的最低限度形式。

| 参数 | 设定 |
|------|------|
| 脑实例 | 2 × 4 脑区 |
| 交互通道 | 脑A的 motor 输出 → 脑B的 assoc 输入（单向信号传递） |
| 任务 | 脑A训练趋光，脑B通过接收脑A的信号加速自己的趋光学习 |
| 观测指标 | 脑B的学习加速比、信号保真度、同步程度 |

**核心假设**：存在一个最优的信号传递强度——太弱则无加速效果，太强则脑B过度依赖脑A而丧失自主适应性。

#### 实验 2-3：竞争-协作混合（Mixed）

**场景**：N个脑实例（N=3,5,10）在共享环境中既有资源竞争又有局部信号协作，模拟群体智能的最低限度涌现条件。

| 参数 | 设定 |
|------|------|
| 脑实例 | 3/5/10 × 4 脑区 |
| 拓扑 | 每个脑独立初始化，脑间按空间距离动态建立/断开连接 |
| 交互 | 局部广播（邻居间）+ 全局资源竞争 |
| 观测指标 | 群体效率、拓扑分化、信息流图、涌现行为分类 |

**核心假设**：存在一个临界脑数量 N_c，超过后群体行为发生质变（如出现领导者-追随者分化、任务自发分工）。

## 三、关键机制设计

### 3.1 脑间化合键（Inter-Brain SDI Bond）

借鉴 V30 脑内四型化合键（E-S/E-L/I-S/I-L），引入**脑间键**作为多脑协作的物理基础：

| 键型 | 方向 | 物理含义 | 生物类比 |
|------|------|---------|---------|
| **B-S** (Broadcast-Soft) | A→B | 候选态广播键，低激活能 | 神经递质弥漫释放 |
| **B-L** (Broadcast-Locked) | A→B | 固化态广播键，高激活能 | 长程投射纤维 |
| **Q-S** (Query-Soft) | B→A | 候选态查询键 | 反馈连接 |
| **Q-L** (Query-Locked) | B→A | 固化态查询键 | 固化反馈通路 |

**演化规则**：脑间键同样遵循 STDP + FEP + BCM 三原理，键的创建/固化/断开由脑间交互的自由能变化驱动。

### 3.2 共享环境协议

```
environment = {
    resources: [{type, position, intensity, decay}],
    signals:   [{source_brain, target_brain, content, strength}],
    predators: [{position, speed, detection_radius}],  // v31.2
    physics:   {diffusion_rate, noise_level}
}
```

每步更新：
1. 各脑感知环境 → 内部 FEP 更新 → 输出 motor 行为
2. 行为改变环境状态（资源消耗、信号广播）
3. 环境状态变化反馈给所有脑
4. 脑间键根据交互历史进行 STDP 更新

### 3.3 群体涌现指标

| 指标 | 定义 | 物理第一性 | 目标 |
|------|------|-----------|------|
| **Φ_group** | 群体信息整合度 | Tononi IIT 的群体推广 | Φ_group > ΣΦ_i（超线性） |
| **D_topology** | 脑间拓扑分化度 | Jensen-Shannon 散度 | D > 0.3（显著分化） |
| **E_collective** | 集体效率 | 总资源捕获 / N / 单脑最优 | E > 1.0（协作增益） |
| **N_c** | 涌现临界规模 | σ_group 相变点 | 测定 N_c |
| **division_of_labor** | 自发分工指数 | 行为熵 H(role) | H > ln(2)（非平凡分工） |

## 四、与多智能体（Multi-Agent）路线的对接

### 4.1 V31 与传统多智能体的本质区别

| 维度 | 传统 Multi-Agent | V31 多脑协作 |
|------|-----------------|-------------|
| **智能体定义** | 手工设计的规则/策略 | 自组织演化的物理网络 |
| **通信机制** | 预定义消息协议 | STDP驱动的脑间键动态创建 |
| **学习方式** | RL/梯度下降 | FEP最小自由能自演化 |
| **涌现性质** | 设计者预期的协同行为 | 从简单物理规则自发涌现 |
| **理论基础** | 博弈论/控制论 | 非平衡统计物理/SOC |

V31 回答的是多智能体的**物理基础问题**：在不需要任何人工设计策略的前提下，纯粹的物理自组织能否产生群体智能？

### 4.2 两阶段推进

```
阶段1: V31 (物理验证)
  ├─ 实验2-1: 双脑竞争
  ├─ 实验2-2: 双脑协作  
  └─ 实验2-3: 多脑混合 (N=3,5,10)
        ↓
阶段2: V32 (Agent对接)
  ├─ 将V31验证的脑间键机制映射为Agent通信协议
  ├─ 用LLM驱动的Agent替换自组织脑，保留脑间键物理
  └─ 混合系统：物理脑 + 符号Agent协同
```

## 六、预期里程碑

| 里程碑 | 判定标准 |
|--------|---------|
| **M1: 竞争分化** | 双脑在相同环境下演化出显著不同的拓扑策略（D > 0.3） |
| **M2: 协作增益** | 脑B在接收脑A信号后学习加速 ≥ 1.5x |
| **M3: 群体涌现** | 确定 N_c，N > N_c 时出现自发分工或领导-追随结构 |
| **M4: 物理-Agent桥接** | 脑间键机制成功映射为Agent通信协议 |

---

## 七、风险与对策

| 风险 | 概率 | 对策 |
|------|------|------|
| 脑间键演化过于缓慢 | 中 | 提高脑间 STDP 学习率，允许初始手工连接作为种子 |
| 群体坍缩为单一策略 | 高 | 引入资源异质性 + 空间距离衰减，强制生态位空间 |
| N_c 超出仿真可行规模 | 中 | 先用简化脑模型（仅motor输出）测N_c，再上完整脑 |
| 与Agent对接的语义鸿沟 | 高 | V31出物理规律，V32做工程解释层，不强求一一对应 |
