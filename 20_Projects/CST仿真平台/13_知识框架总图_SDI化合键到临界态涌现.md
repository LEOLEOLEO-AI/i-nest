## 三、四条局部规则的完整状态机

```
键状态机：
                 n_LTP ≥ θ_LTP
    E-S ─────────────────────────► E-L
     ▲                               │
     │  T_decay后不活跃               │ T_decay后长期不活跃
     └───────────────────────────────┘
     
     │  n_LTD ≥ θ_LTD              
    I-S ─────────────────────────► 断开（消除）
     ▲                               
     │  FEP梯度 > Ea_S（低度节点）   
    断开 ────────────────────────► E-S（新键建立）

    E-L ── 全局E-L>60% + 高激活 ──► E-S（突触缩放降级）
    I-L ── 不衰减（层级结构稳定性）

规则编号：
  规则1: E-S→E-L（固化，STDP+LTP驱动）
  规则2: I-S→断开（修剪，LTD驱动）
  规则3: 断开→E-S（新建，FEP驱动）
  规则4: E-L→E-S（衰减，T_decay后）
  规则5: I-L保持（不衰减）
  规则6: E-L→E-S（突触缩放，Turrigiano机制）← v5新增
```

## 四、三原理 + 突触缩放的协同机制

| 机制 | 时间尺度 | 驱动量 | 作用 |
|------|---------|--------|------|
| **STDP** | 毫秒 | 脉冲时序差Δt | 每条键权重更新 |
| **FEP** | 秒 | 预测误差+复杂度 | 固化/修剪/建立决策 |
| **最小作用量** | 分钟-小时 | ∫Ea·|Δw|dt | 拓扑演化路径选择 |
| **突触缩放** | 小时-天 | 全局E-L饱和度 | 骨架解聚，维持可塑性 |

**四者缺一不可**：
- 无突触缩放 → 晶化死锁（v4教训：E-L=99%，α→∞）
- 无FEP → 随机游走，不收敛到临界态
- 无最小作用量约束 → 拓扑演化路径低效
- 无STDP → 无法执行具体权重更新
