## 正文

根据以上描述，以biorender风格生成符合《Nature》期刊排版风格的学术图形，需包含以下核心要素：

-   **配色体系**：采用《Nature》经典冷色调为主（如浅灰蓝、深靛蓝、橄榄绿、暖棕等），避免高饱和色彩，主色调明度控制在 50%-70%，辅助色用于突出关键数据点或模块，遵循期刊「高对比度、低视觉干扰」原则。
    
-   **布局结构**：
    
    -   采用模块化分区设计，各子图（A/B/C/D）以网格布局排列，间距占图宽 10%，留白比例不低于 20%。
        
    -   坐标轴、图例、标注框采用无衬线字体（如 Arial/Helvetica），字号与《Nature》标准一致（主标题 12pt，坐标轴标签 10pt，图例 8pt）。
        
-   **风格要求**：
    
    -   数据图需呈现「科研严谨性」与「视觉简洁性」结合，线条采用 0.75pt 实线，数据点用空心圆 / 正方形（直径 2pt），误差线用细虚线（0.5pt）。
        
    -   示意图需突出物理结构或算法流程的逻辑关系，模块用圆角矩形（边框 0.5pt 深灰），连接线带箭头（宽度 1pt），关键节点用阴影效果区分层级。
        

-   所有文本需为可编辑状态，便于后期修改标注内容，遵循《Nature》「图文自洽」原则（图形信息与论文正文描述完全对应）。
    

#### **分场景细化提示词**

根据不同图形类型，可在基础框架上补充以下描述：

##### **1\. 数据可视化图表（如信号频谱、芯片性能对比）**

" 生成《Nature》风格的 \[折线图 / 柱状图 / 热力图\]，展示 \[电子信息 / 通信\] 领域 \[信号强度 / 带宽利用率 / 芯片功耗\] 数据。要求：

  

-   X 轴为 \[时间 / 频率 / 工艺节点\]，Y 轴为 \[幅值 / 效率 / 温度\]，主曲线用 #1E88E5（蓝），对比曲线用 #43A047（绿），背景网格线为浅灰（#EEEEEE）。
    
-   数据点添加 95% 置信区间阴影（透明度 30%），图例位于右上角，采用白底黑边矩形框。
    
-   底部添加比例尺（如「10GHz / 格」），关键数据点用星号标注并附注释框（如「\* 峰值功耗点」）。"
    

##### **2\. 系统架构示意图（如 AI 芯片架构、通信协议栈）**

" 绘制《Nature》风格的 \[芯片架构 / 网络协议\] 示意图，体现 \[集成芯片 / 人工智能\] 领域 \[多核处理器 / 深度学习模型\] 结构。要求：

  

-   采用分层模块化设计，底层为硬件层（浅灰背景），中层为算法层（浅蓝背景），顶层为应用层（浅绿背景），各层用 0.5pt 深灰实线分隔。
    
-   核心模块（如 CPU/GPU/ 神经网络层）用圆角矩形（填充色 #E3F2FD），内部标注英文缩写（如「CNN」），模块间用带箭头虚线（#607D8B）表示数据流。
    
-   关键接口（如 PCIe/USB）用橙色高亮，添加简要文字说明（如「数据吞吐量：20GB/s」），底部添加比例尺（如「10mm」）。"
    

##### **3\. 流程图（如通信算法流程、芯片设计流程）**

" 生成《Nature》风格的 \[算法流程图 / 设计流程图\]，展示 \[网络通信 / 电子信息\] 领域 \[信道编码 / 版图设计\] 逻辑。要求：

  

-   采用自上而下瀑布式布局，起始节点用椭圆（填充 #BBDEFB），处理节点用矩形（填充 #CFD8DC），决策节点用菱形（填充 #ECEFF1）。
    
-   流程线用 1pt 实线（#263238），分支条件用斜体字标注（如「SNR>20dB?」），关键步骤添加黄色高亮边框（0.75pt）。
    
-   右侧添加辅助说明栏，用项目符号列出各步骤核心参数（如「① FFT 点数：1024」），字体颜色 #37474F。"
    

##### **4\. 显微 / 仿真图（如芯片显微照片、电磁场仿真）**

" 生成《Nature》风格的 \[芯片显微图 / 电磁场分布图\]，适用于 \[集成芯片 / 电子信息\] 领域 \[纳米器件 / 天线设计\]。要求：

  

-   图像主体采用灰度调色（0-255 阶），关键结构（如晶体管 / 谐振腔）用伪彩色标记（红色表示高场强区域，蓝色表示低场强区域），色标位于右侧（高度占图高 30%）。
    
-   添加比例尺（如「500nm」）和方位标识（如「Top View」），左上角标注成像技术（如「SEM」），底部添加像素分辨率（如「1024×1024 px」）。
    
-   背景采用纯黑（#000000），避免杂散光干扰，图像锐度设置为「High」，噪点控制在 5% 以内。"
    

#### **通用补充说明**

-   输出格式为矢量图（.EPS/.PDF），分辨率不低于 600dpi，确保印刷清晰。
    
-   所有文本需为可编辑状态，便于后期修改标注内容，遵循《Nature》「图文自洽」原则（图形信息与论文正文描述完全对应）。
