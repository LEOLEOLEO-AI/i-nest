## 一、三原理的角色分工

| 原理 | 作用尺度 | 驱动量 | 在SDI中的载体 |
|------|---------|--------|-------------|
| **STDP** | 单条键，毫秒级 | 突触前后脉冲时序差Δt | 每条E-S/I-S键的权重更新 |
| **最小自由能（FEP）** | 单个SDI节点，秒级 | 预测误差+复杂度代价 | 键的固化/断开决策（规则1-5） |
| **最小作用量** | 全局网络，分钟-小时级 | 拓扑演化路径的总代价∫L dt | 网络从随机态→小世界态的演化轨迹 |

三原理在**不同时间尺度**上协同——STDP负责微秒-毫秒，FEP负责秒-分钟，最小作用量描述整体演化轨迹。

## 二、最小自由能（FEP）——每条键的局部决策

### 2.1 自由能定义

对SDI节点i，其变分自由能F定义为：

```
F_i = Σⱼ [ (ŷᵢⱼ - yᵢⱼ)² + Ea_ij · wᵢⱼ² ]
     预测误差项          复杂度惩罚项
```

- `ŷᵢⱼ`：节点i对邻居j激活状态的预测（由当前权重w决定）
- `yᵢⱼ`：j的实际激活状态（STDP观测到的）
- `Ea_ij · wᵢⱼ²`：维持该键的"代价"（活化能×权重²）

### 2.2 STDP是FEP梯度下降的物理实现

```
STDP权重更新 = -η · ∂F_i/∂w_ij

∂F_i/∂w_ij = -2(ŷ-y)·激活 + 2·Ea·w

LTP（Δt>0）：减小预测误差 → -∂F/∂w > 0 → w↑
LTD（Δt<0）：减小冗余复杂度 → -∂F/∂w < 0 → w↓
```

**结论**：STDP不是人为规则，是FEP梯度下降在忆阻器物理层的自然实现。

### 2.3 固化/断开决策 = 自由能极小值判断

```
E-S → E-L（固化）：找到了F的局部极小值（稳定吸引子盆地）
I-S → 断开：w→0是F的全局极小值之一（低代价低效益连接消除）
新键建立：∂F/∂w_new < 0（建立新连接能减小惊讶度）
```

### 2.4 可测量量

```python
F_total = Σᵢⱼ[ (prediction_error_ij)² + Ea_ij * w_ij**2 ]
```

**仿真预测**：F随时间单调下降，在临界态达到极小值平台。

## 三、最小作用量——拓扑演化路径

### 3.1 作用量定义

网络拓扑从初始状态G₀演化到最终状态G_T，其作用量：

```
S[G(t)] = ∫₀ᵀ L(G, Ġ) dt

拉格朗日量 L = 信息传递效率 - 拓扑变化代价
            = σ(t) · v_prop  -  Σᵢⱼ Ea_ij · |Δw_ij/Δt|
```

- `σ(t)`：当前小世界系数（越大传播效率越高）
- `v_prop`：平均激活传播速度
- `Σ Ea_ij · |Δw_ij|`：每步拓扑变化的总活化能代价

### 3.2 最小作用量原理的含义

```
δS/δ[G(t)] = 0
⟺ 欧拉-拉格朗日方程：d/dt(∂L/∂Ġ) = ∂L/∂G
⟺ E-L固化速率 × 效率增益 = 拓扑变化代价
```

**直观含义**：网络不会走"弯路"。从随机态到小世界态的演化路径，是**效率提升和代价之间最优平衡**的路径——既不急于固化所有连接（代价太高），也不懒于固化（效率损失太大）。

### 3.3 与经典力学的完整类比

| 经典力学 | SDI网络拓扑演化 |
|---------|---------------|
| 粒子位置 x(t) | 网络拓扑 G(t) |
| 动能 T = ½mv² | 信息传播效率 σ(t)·v |
| 势能 V(x) | 自由能 F(G) |
| 拉格朗日量 L = T-V | L = 效率 - 代价 |
| 最小作用量 δ∫L dt=0 | 最优拓扑演化路径 |
| 牛顿定律（局部） | STDP+FEP规则（局部） |

**最深刻的结论**：牛顿定律和最小作用量原理是等价的。同理，**STDP+FEP局部规则 和 全局最优拓扑演化路径是等价的**。

## 四、自组织临界态——三原理的不动点

### 4.1 自由能景观的三个区域

```
F(网络状态)
   │          次临界区              超临界区
高 │    （E-L过多，过度有序）  （E-S过多，过度混乱）
   │         F↑ 预测误差积累    F↑ 复杂度代价激增
   │              \                /
低 │               \              /
   │                ▼            ▼
   │           ┌────────────────┐
   └───────────┤  临界态（鞍点）  ├──────────────
               │  F最低，α→1.5  │
               └────────────────┘
```

### 4.2 两个反馈回路驱动系统到临界态

**回路1（次临界→临界）**：
```
E-L太多 → 预测误差↑（只会重复旧模式）→ F↑
→ FEP驱动：T_decay衰减E-L为E-S
→ E-S增多 → 网络可塑性恢复 → 新模式可学习 → 预测误差↓ → F↓
```

**回路2（超临界→临界）**：
```
E-S太多 → 复杂度代价↑（随机噪声主导）→ F↑
→ FEP驱动：θ_LTP触发E-S固化为E-L
→ E-L增多 → 骨架稳定 → 信号传播有序 → 复杂度代价↓ → F↓
```

**两个回路的平衡点 = 临界态（F全局极小）= α→1.5**

### 4.3 相变序参量

```
序参量 Φ = 激活关联长度 / 系统尺寸

Φ << 1：次临界（关联局部化）
Φ ≈ 1：临界态（关联延伸至全系统）← 目标
Φ >> 1：超临界（全局同步爆发）
```

### 4.4 无中央控制的自组织

系统到达临界态**不需要**：
- ❌ 全局监控器
- ❌ 人为设定目标函数
- ❌ 反向传播梯度
- ❌ 外部参数调整

只需要：
- ✅ STDP（局部时序规则，每条键独立执行）
- ✅ FEP（局部惊讶度最小化，每个节点独立计算）
- ✅ 物理活化能Ea（由忆阻器材料特性决定）

**这三条规则 + 物理约束，数学上可以证明系统将以概率1收敛到临界态（Beggs&Plenz 2003实验验证，SOC理论预测）。**
