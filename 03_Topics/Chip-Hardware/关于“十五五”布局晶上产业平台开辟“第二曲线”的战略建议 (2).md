# 关于“十五五”布局晶上产业平台开辟“第二曲线”的战略建议

> 创建时间: 2025-02-02

### 关于“十五五”布局晶上产业平台开辟“第二曲线”的战略建议


（聚焦高速在线破译解密与数字射频战场动态适应能力的重大需求与建设路径）

**一、战略紧迫性：两大核心需求驱动晶上平台“第二曲线”突围**


- 

**高速在线破译解密需求**：

现代情报战对 **“实时性、精准性、智能化”** 提出更高要求，传统破译解密系统受限于分立器件与板级集成，难以满足 **“秒级响应”** 的战略情报时效性需求。

- 晶上超大规模逻辑资源（如FPGA、存算一体单元）可实现 **“高速在线破译解密”**，显著提升情报处理效率，保障国家安全。


- 

**数字射频战场动态适应需求**：

现代战争向 **“全域感知、智能决策、瞬时响应”** 方向发展，传统射频系统难以实现 **“天线后动态波束成形（DBF）、实时信号处理、智能态势感知”** 等功能。

- 晶上异质集成可构建 **“数字射频大脑”**，支撑中国版数字射频战场，实现 **“射频动态适应战场”** 的作战能力。


**二、战略价值：晶上平台支撑国家重大需求的核心优势**
**（一）高速在线破译解密的战略价值**


- 

**技术优势**：

**超大规模逻辑资源**：晶上集成数百万逻辑单元，支持并行计算与硬件加速，实现 **“秒级破译”**，满足战略情报时效性要求。

- **存算一体架构**：通过晶上存算融合，减少数据搬运开销，提升解密效率 **10倍以上**。


- 

**应用场景**：

**战略情报处理**：在国家级情报中心部署晶上破译解密系统，实现 **“实时情报分析-动态决策支持”** 的闭环能力。

- **网络安全防护**：在关键信息基础设施中嵌入晶上解密模块，防范高级持续性威胁（APT）。


**（二）数字射频战场动态适应的战略价值**


- 

**技术优势**：

**动态波束成形（DBF）**：晶上集成多通道射频前端与AI处理单元，支持 **“智能波束指向-干扰抑制-目标跟踪”** 的动态重构能力。

- **实时信号处理**：基于晶上异构计算架构，实现 **“多源信号融合-目标识别-态势生成”** 的全流程处理，延迟降低至微秒级。


- 

**应用场景**：

**全域感知**：在战区部署晶上数字射频系统，构建 **“全天候、全频段”** 侦察网络，提升战场态势感知能力。

- **智能电子战**：通过晶上AI实时生成干扰波形，实现对敌通信系统的精准压制与欺骗。


**三、建设内容与实施路径**
**（一）高速在线破译解密能力建设**


- 

**核心技术攻关**：

**晶上FPGA架构**：开发支持超大规模逻辑资源的晶上FPGA，2025年完成原型验证，2027年实现量产。

- **存算一体解密引擎**：研制基于晶上存算融合的密码破译芯片，支持 **“实时解密-动态分析”** 的全流程处理。


- 

**应用部署**：

**国家级情报中心**：建设晶上破译解密平台，支持战略情报的实时处理与分析。

- **关键信息基础设施**：在金融、能源、通信等领域部署晶上解密模块，提升网络安全防护能力。


**（二）数字射频战场动态适应能力建设**


- 

**核心技术攻关**：

**晶上DBF模块**：开发支持多通道动态波束成形的晶上射频前端，2026年完成军用原型验证。

- **智能信号处理单元**：基于晶上AI架构实现 **“信号融合-目标识别-态势生成”** 的全流程处理，延迟降低至微秒级。


- 

**应用部署**：

**战区试点**：在东部、南部战区部署晶上数字射频系统，构建 **“智能感知-精准干扰”** 能力。

- **装备升级**：推动现役雷达、通信装备的晶上化改造，提升作战效能。


**四、可操作举措与政策保障**


- 

**国家专项支持**：

设立 **“晶上高速破译解密专项”**（50亿元），支持关键技术攻关与场景应用。

- 将晶上数字射频战场纳入 **“十四五”国防科技工业重点工程**，给予50亿元财政投入。


- 

**产业生态培育**：

组建 **“晶上破译解密产业联盟”**，推动密码企业与晶圆厂协同创新。

- 在雄安、粤港澳建设 **“晶上数字射频产业创新中心”**，提供设计、流片、测试一站式服务。


- 

**人才与标准建设**：

在“卓越工程师学院”开设 **“晶上系统工程”** 专业，培养复合型人才。

- 制定 **《晶上破译解密技术标准》** 与 **《晶上数字射频系统规范》**，主导国际标准制定。


**五、总结与建议**


- **战略意义**：

晶上平台是支撑高速在线破译解密与数字射频战场动态适应的 **“战略基石”**，将重塑全球科技与产业竞争格局。


- **核心目标**：

2027年前实现 **“两个领先”**：破译解密效率领先、数字射频作战能力领先。


- **建议纳入国家规划**：

将晶上平台列为 **“十五五”国家重大科技专项**，制定《晶上支撑国家安全与新兴产业行动纲要》。


**方案特色**：


- 紧扣国家重大需求（高速破译解密、数字射频战场），凸显晶上平台的不可替代性；

- 细化技术路线与实施路径，确保战略可落地、可考核；

- 设计“政策-资本-人才”组合拳，构建闭环生态体系。
