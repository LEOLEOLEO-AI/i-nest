## 📊 **理论基础三大支柱**

### **支柱1：物理储备池计算（Physical Reservoir Computing）**

```auto
核心原理：
┌─────────────────────────────────────┐
│ 输入层 → 物理介质储备池 → 读出层  │
│         (自组织非线性动力学)        │
│         ↓                           │
│   无需训练内部连接权重！           │
│   仅训练线性读出层                  │
└─────────────────────────────────────┘
```

**理论优势**：

-   ✅ **计算效率**：训练成本降低99%（仅训练读出层）
    
-   ✅ **物理本质**：利用材料固有非线性动力学
    
-   ✅ **时间多尺度**：天然处理多时间常数信号
    

**数学模型**：

```auto
储备池状态更新：
x(t+1) = f(W_in·u(t) + W·x(t))
  其中 W 固定（物理拓扑决定）

输出计算：
y(t) = W_out·x(t)
  仅 W_out 需要训练
```
