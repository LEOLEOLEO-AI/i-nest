# LNN vs iNEST SDI 深度机制对比综述

- **类型**: 技术对比笔记
- **时间**: 2026-06-03
- **标签**: LNN, SDI, FEP, STDP, 液态神经网络, 自演化, 连续时间动力学
- **来源**: Codex 主对话框深度分析

---

## 一、核心公式对照

### LNN (Liquid Time-Constant Network, Hasani 2021/2023)

```
dx/dt = -(1/tau)*x(t) + f(x, I, t, theta)*(A - x(t))
         1.被动衰减             2.输入驱动的非线性门控
```

### iNEST SDI 节点动力学

```
dL_h(t)/dt = f_theta(h(t), x(t), t) + FEP 驱动的键固化/断开
             1.状态演化               2.拓扑自组织
```

### 共通核心

> 连续状态演化 + 局部自适应 + 最小能量路径收敛

---

## 二、LNN 逐项拆解

| 项 | 含义 | 工程实现 |
|----|------|---------|
| -(1/tau)*x | 被动漏电流（leak）：状态向零衰减。tau 可学习，由输入 I 通过小型 NN 动态决定 | PyTorch nn.Linear -> sigmoid -> tau |
| f(x,I,t,theta) | 非线性门控函数：小型 MLP，输出 in (0,1)，控制进入饱和区速率 | nn.Sequential / CfC 闭式解 |
| (A - x) | 有界饱和项：A 是上界，保证状态不发散 | torch.clamp 或动力学自然收敛 |

**核心巧思**：不是学习权重矩阵乘法，而是让每个神经元学习自己的时间尺度 tau。信息处理快的输入→小 tau（快速遗忘），慢信号→大 tau（慢速整合）。等效于无显式 attention 矩阵的注意力机制。

---

## 三、iNEST SDI 逐项拆解

| 项 | 含义 | 工程实现 |
|----|------|---------|
| f_theta(h,x,t) | 节点隐藏状态 L_h 的时变动力学 | v8 中以级联激活传播实现（NetworkX 有向图 + 三类神经元种子） |
| FEP 固化/断开 | 根据自由能 F = sum((y_hat-y)^2 + Ea*w^2) 决定键状态转换 | STDP 累积证据 → FEP 判断 → 键状态机转换 |

---

## 四、四维度根本差异

| 维度 | LNN | iNEST SDI |
|------|-----|-----------|
| 拓扑 | 固定（训练后边结构不变） | 自演化（键可固化/断开/新建） |
| 学习信号 | 监督（需标签 y_true，反向传播） | 无监督/自监督（FEP 惊讶度最小化） |
| 时间尺度 | 单一尺度（tau 由输入决定但网络一次性训练） | 多尺度：STDP(ms) -> FEP(s) -> 最小作用量(min) -> 突触缩放(h) |
| 物理基础 | 工程近似（仿生启发 + ODE 数值近似） | 第一性原理（FEP 来自统计物理，STDP 是 FEP 梯度下降物理实现） |
| 硬件映射 | 软件推理（ONNX->Jetson/TPU） | 物理原生（忆阻器 V=IR 天然加权求和，键状态机=异步电路状态） |

---

## 五、tau 映射关系

```
LNN tau               <->  iNEST 键的稳定性
tau 小（快速遗忘）      <->  E-S 键（短时程，Ea=0.15）
tau 大（慢速整合）      <->  E-L 键（长时程，Ea=0.85）
tau 由输入决定          <->  键类型由 FEP 决定（一阶梯度信息驱动）
```

**关键差异**：LNN 的 tau 训练后固定；iNEST 的键类型可持续演化，可在推理中根据数据分布偏移自动调整网络结构。

---

## 六、工程融合路径

1. **引入可学习的局部 tau**：在每个 SDI 节点的状态更新中增加 -(1/tau)*h 项
2. **CfC 闭式解用于 FEP 预测**：h(t) = sigma(-f(t)) * A 替代简单预测模型
3. **不混淆核心差异**：LNN tau 是参数（训练得到），iNEST 键类型是状态（自演化得到）
