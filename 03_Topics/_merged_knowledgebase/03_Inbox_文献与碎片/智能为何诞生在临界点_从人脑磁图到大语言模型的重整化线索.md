# 智能为何诞生在临界点:从人脑磁图到大语言模型的重整化线索

- **笔记本**: 我的剪藏
- **时间**: 2026-03-15 00:14

原文链接：[智能为何诞生在临界点:从人脑磁图到大语言模型的重整化线索](https://mp.weixin.qq.com/s/Th7UTlCTWRdNSfNz26rhMA)
在物理学中，临界点是一个介于完全有序与完全无序之间的特殊动力学状态。在这种状态下，系统会表现出尺度不变性，即系统的统计特征在不同空间或时间尺度上呈现出相似的结构。人脑可能就运行在这样的临界点附近，这是神经科学中的一个长期的有趣假设。最近有学者探索了这一课题【文献1】，这种在统计物理中非常重要的现象，是否也存在于人脑的大尺度活动之中。课题几十年来，神经科学家在各种神经信号中发现了尺度行为，例如，大脑信号中的 1/f 噪声、长程时间相关性以及所谓的神经元雪崩往往呈现幂律分布。在统计物理中，这类幂律结构通常意味着系统处于或接近临界状态，因为在这种状态下，扰动可以跨多个尺度传播。以往关于临界性的最有力证据大多来自微观层面的神经元记录，例如对动物大脑中单个神经元或小型神经网络的电生理测量。人脑在宏观尺度上是否也遵循相同的统计物理规律，一直是一个悬而未决的问题。方法针对这一问题，论文分析了人类静息态脑磁图（MEG）数据。MEG 可以记录由神经电流产生的微弱磁场，具有极高的时间分辨率，因此能够在非侵入条件下观察大脑的大尺度动态。论文借鉴了统计物理中的核心思想 - 重整化，研究不是只在一个固定空间尺度上分析脑信号，而是将高度相关的传感器信号逐步合并，形成更大的有效变量。如果大脑确实具有尺度不变性，那么随着这种逐级粗粒化，信号的统计结构应该保持某种稳定的规律。图1. MEG数据的现象学重整化分析现象通过将这种方法应用到 MEG 数据，论文观察到一个非常显著的现象：神经活动的一些整体统计性质，例如方差、相关结构以及时间波动，在粗粒化过程中呈现出稳定的尺度律（scaling laws）。当系统尺度逐渐变大时，信号统计结构并不随机变化，而是遵循可预测的幂律关系。更有意思的是，这些测得的尺度指数与此前在动物大脑神经元群体活动中测得的指数非常接近。这暗示着，从单个神经元到整个大脑，大规模神经活动可能遵循相似的集体动力学原则。论文进一步分析了神经元雪崩的统计特性。神经元雪崩指的是神经活动在网络中以级联方式传播的事件，其规模和持续时间常常呈现幂律分布，被认为是临界动力学的重要标志。研究发现，在进行多尺度粗粒化之后，这些雪崩的统计分布仍然保持尺度不变。显示这些幂律行为并不是测量分辨率的偶然结果，而是反映了大脑活动内部真实存在的多尺度结构。图2. 静息态人脑MEG中的尺度行为分析为理解这些现象，论文提出了一个基于统计物理的大脑动力学模型。该模型类似于一个非平衡的、具有自适应机制的Ising网络。在这个模型中，每个神经单元可以类比为一个自旋，它与其他单元相互作用，而整个网络的状态在相互作用和反馈调节下不断演化。当学者根据 MEG 数据推断模型参数并进行模拟时，模型能够再现许多真实脑活动的特征，例如振荡结构以及雪崩统计规律。更重要的是，这些尺度行为只有在系统参数被调节到临界区域附近时才会出现，而在明显有序或明显无序的状态下则不会出现。一个重要发现是，大脑活动中观察到的尺度律对“神经网络中兴奋与抑制（E/I）之间的平衡”非常敏感，这个E/I平衡可能是大脑维持临界动力学的重要生理机制。当兴奋与抑制之间保持适当比例时，系统会自然接近临界状态，从而产生尺度不变的动力学。如果这种平衡被打破，尺度结构就会迅速退化。通过分析 MEG 信号中的尺度律，学者还提出了一种非侵入式预估人脑E/I 平衡的方法。可能有助于类似癫痫、自闭症和精神分裂等涉及E/I 平衡失调类的疾病的诊疗。图3. 临界态附近自适应Ising模型中的非高斯活动与尺度行为结论人类静息态的大脑活动并不是简单的背景噪声。相反，它似乎受到类似物理临界系统的普适尺度规律所支配。大脑不像传统工程系统那样以固定结构运行，而更像一个自组织临界系统，通过持续调节兴奋与抑制之间的关系，使神经活动能够在多个尺度上自由传播并进行信息处理。论文佐证了大脑是一种运行在临界点附近的复杂系统。在这种状态下，系统既不会陷入完全静止的有序状态，也不会进入完全混乱的无序状态，而是保持一种稳定与灵活之间的动态平衡。这种临界状态被认为能够最大化系统的信息传播能力、动态范围以及适应性，这些特性正是高级认知功能所需要的。启发读者都知道，笔者过去三年，一直将重整化应用于解释大语言模型（LLM）智能原理。如果我们把这篇论文与LLM的重整化过程放在一起观察，会发现两者本质上都在处理同一个深刻的数学问题：一个复杂的动力系统如何在跨越巨大的空间或时间尺度时，仍然能够保留最核心的、具有功能性的信息结构？大脑表现出的“缩放不变性”，在重整化理论中意味着，大脑并不是在处理离散的、局部的孤立信号，而是运行在一个全局耦合的临界态上。这种临界态的存在，使得微观神经元的电活动能够以一种受控的方式，逐级“涌现”成宏观的认知信号。这与笔者描述的LLM的层次化表征有着极强的对称性。在 Transformer 架构中，底层的 Token 可以类比为 MEG 中的传感器原始信号，而每一层 Attention 操作本质上都在执行一种基于语义相关性的粗粒化。正如论文中通过相关性结构来决定哪些传感器应该被聚合成一个块，LLM 则是利用注意力权重来决定哪些 Token 的组合能够构成更高阶的语义概念。如果一个LLM训练得足够好，它在不同深度层级上的信息流也会表现出类似的幂律缩放规律。这意味着，无论是处理单词、短语还是长篇段落，模型底层的数学逻辑是一致的。论文中用来仿真的 Ising模型 ，也跟笔者一直推崇的神经网路架构不谋而合，或许自旋模型才是 AGI 终极的架构选择。探讨更有趣的点在于论文提到的“E/I 平衡”控制。在大脑中，兴奋与抑制的精密微调将系统推向了临界点，既避免了过度同步引发的癫痫，也避免了关联匮乏导致的思维解体。笔者判断或许这种平衡对应着LLM训练过程中的权重初始化和正则化策略。一个处于临界点的模型，其表征空间具有最大的动态范围和灵活性。这解释了为什么 Scaling Law（缩放定律）在语言模型中如此普遍，当模型规模增加时，它实际上是在拓宽其重整化路径，从而能够捕捉到更深层次、更隐秘的语言统计结构。所以，这篇论文不仅仅是在研究大脑，也能为我们理解人工智能提供神经科学参考。真正的智能或许并不取决于碳基还是硅基物理载体，而是取决于系统是否能通过某种自组织的反馈机制，把自己维持在重整化流的临界点上。进一步，是否可以通过监测 LLM 激活值的重整化流，来判断一个模型是否真的产生了高阶认知？即： 如果一个模型的表征在粗粒化过程中迅速坍缩成高斯噪声，那它只是在做简单的统计预测；如果它能像 MEG 数据那样保持缩放不变性，那它可能真的捕捉到了语言的物理规律。这是不是能够得到优于 Epiplexity的认知智能衡量方法？！文献1，Scaling and tuning to criticality in resting-state
