# 『DGIST Nat. Commun.』晶圆级被动忆阻交叉阵列：向类脑规模神经形态计算迈进

> 笔记本: 我的剪贴板  
> 创建时间: 2025-10-03

原文链接: [https://mp.weixin.qq.com/s/MOBFC88iXhd9e4-h0HVkJA](https://mp.weixin.qq.com/s/MOBFC88iXhd9e4-h0HVkJA)


随着类脑计算的发展，构建高密度、低功耗的神经形态硬件成为关键。被动交叉阵列（passive crossbar）以其高空间效率被视为潜在方案，但过去难以保证高良率和均匀性。该研究团队提出了一种协同设计策略，在标准 CMOS 兼容工艺下，成功在 4 英寸晶圆上制造出良率超 95% 的被动忆阻交叉阵列，并演示其在神经网络任务中的有效性。

**🔑 研究亮点 **
- 
**高良率晶圆级制造**：在 4 英寸晶圆上制备被动交叉阵列，平均器件良率达到 ~95%，是被动交叉阵列迈向大规模集成的重要突破。
- 
**协同器件 / 结构设计**：采用底电极 “回填” 工艺 + 六边形线形设计，抑制边缘 **rabbit-ear** 的形成，减少尖锐结构引起的高电场，降低硬击穿风险。
- 
**优异开关特性与稳定性**：典型 SET 电压 ~1.3 V，RESET ~ −2.0 V。开 / 关比（DR，I_ON / I_OFF）平均 ~9.8，耐久性超过 10⁵ 次循环，保持时间超过 10⁵ 秒。
- 
**多级 / 模拟状态存储**：通过可调递增脉冲策略，成功调控多个电导状态（可达 4–6 bit 量化），表现出良好的可控性与保留性。
- 
**神经网络演示**：基于该交叉阵列，用外训练 + 调整策略，完成 Fashion MNIST 图像识别任务，模拟向量–矩阵乘法 (VMM) 与推理流程。
- 
**可扩展设计潜力**：该设计兼容后端集成（BEOL）3D 交叉架构，并可在未来与 CMOS 混合集成，朝向脑尺度 (10¹⁵ 连接) 的神经形态硬件迈进。

**🧠 结论与展望**
**结论**
该研究展示了一条从工艺、器件到系统兼顾的路线：在标准 CMOS 兼容流程下，用协同设计策略成功制造出晶圆级高良率的被动忆阻交叉阵列。器件具备稳定的开关特性、多级可调电导、长期耐久性，并在神经网络任务上展示出应用可行性。
**展望**
- 
**3D 交叉阵列与后端集成**：未来可在 CMOS 芯片背面集成多层被动交叉阵列，提升连接密度。
- 
**向脑尺度扩展**：以 10¹⁵ 级别连接为目标，推动硬件规模与神经网络尺度同步增长。
- 
**算法与器件联合优化**：在器件不确定性、噪声、非理想特性下，设计适配的 SNN 算法与训练策略。
- 
**工艺可靠性与寿命优化**：进一步提升良率、降低变异、增强循环寿命，是推进产业化的关键。
