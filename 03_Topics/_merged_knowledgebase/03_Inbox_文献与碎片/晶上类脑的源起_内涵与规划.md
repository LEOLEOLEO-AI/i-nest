# 晶上类脑的源起、内涵与规划

- **类型**: plain_text
- **时间**: 2025-08-05 17:08:47

## 内容

做一个面向社会发布的颠覆性前沿科研方向规划PPT，核心内容”晶上类脑的源起、内涵与规划“，用现代科技风格，多引用最新的paper数据，并标明引用出处，叙述逻辑如下：

P1：智能时代的核心竞争是“智力规模、密度与经济性”之争

P2：当前以Transformer为代表的各种LLM，尽管在诸多领域取得突飞猛进的颠覆性进展，也在AI4Science中不断取得突破，但“巨高能耗”成为当前范式可持续发展的最大拦路虎，智能时代呼唤智力能够绿色普惠、泛在普适的可持续发展范式！

P3：如何发现新范式？如何颠覆现有范式的思维视角、方法论和实践规范？

P4：新范式的法则：最小自由能（能量消耗）。遵循物理世界中辛顿和霍普菲尔德提出的最小自由能能量景观，以最小自由能作为最高法则，导出必须以物理而非模拟，必须靠内在而非外在，通过与外在环境的迭代交互，通过自复制、自演化、自适应实现本自具足的智能。

P5：新范式的第一性：复杂性。人脑作为宇宙中已知的最复杂系统，复杂性是其第一性，如何以复杂性思维，而非当前的还原论、乃至系统论来指导类脑智能的创新是颠覆性成败的关键。冯诺依曼在自复制自动机中提出“系统的复杂度超过阈值就可以对抗熵增，超高阶复杂度演化下去”，但没有给出复杂度的衡量标准以及内涵。

P6：我们的创新：网络时空协同复杂度（CST，Spatial Temporal Complexity）。在冯诺依曼复杂度阈值定性思想的基础上，针对复杂系统的核心复杂网络，提出网络时空协同复杂度定量概念，用时空协同的新视角去定义、度量复杂度，并建立复杂度与智能涌现的关系。空间代表（结构、智能承载），时间代表（动力学、智能表达），二者有效协同才能实现大脑中最高效的临界态。

P7：类脑智能的新理论：网络时空协同复杂度涌现智能（iNEST，Intelligence Emerging from Network Temporal-spatial Synergy）。当网络的时空协同复杂度超过环境复杂度，且满足最小自由能法则，就能自发涌现觅食、寻迹、协同、创造等智能行为。当前的ANN，在不断扩大空间复杂度，以LLM达到万亿顶峰，但网络结构的时间复杂度很低；自然界的BNN，通常拥有高的时间复杂度，通过进化不断提升空间复杂度，以人脑的大脑皮层最为特殊和高阶；未来具身智能的发展方向是同时获得高时间复杂度和高空间复杂度，以期实现AGI乃至ASI。为智能发展指明了方向和定量分析定理。

P8：类脑智能的新路径：物理时空协同网络具身智能。借助忆阻器、光子等非线性器件实现物理基底网络，以电、化学控制的物理拓扑连接与断开等形成类脑动态激活子网；以大规模并行脉冲异步电路物理实现类神经网络并行稀疏激活机制；以环境互动中的物理网络演化实现类脑的记忆、创造等机制。在物理机制上支持空间上通过自组织、自分形可以形成模块化、分层、小世界的高阶拓扑结构；时间上通过同步、临界与可塑等实现跨尺度的时间动力学相变态，通过物理网络而非当前的逻辑网络，遵循最小能量消耗通过在与环境交互中自演化时空复杂度，实现基于网络时空协同复杂度的涌现智能。

P9：类脑智能的新载体：介观尺度“智涌”脑。以软件定义晶上系统（SDSoW，Software Defined System on Wafer）作为最佳的类脑技术物理载体，融合软件定义互连、晶圆级介观尺度集成、异构异质融合等优势，通过材料-器件-网络-复杂度-智能的垂直整合与协同，探索一条基于复杂度持续提升的智涌脑自主发展路径。
