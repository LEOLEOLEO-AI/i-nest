# Nat Neurosci：为什么我们的大脑能高效决策？真正的大脑分区，比你想的聪明得多！

我们常听说“大脑不同区域负责不同功能”比如前额叶是“思考和决策中心”。但这种说法其实隐含了一个假设：大脑像地图一样，被划成了一个个固定“街区”，每个街区有明确边界和专属任务。
然而，科学家最近发现，事情没那么简单。在小鼠的前额叶皮层（相当于人类负责计划、自控和决策的脑区），传统的解剖分区（比如根据细胞长得什么样来划分）和神经元实际怎么工作并不匹配。有些相邻区域的神经元活动模式天差地别，而相隔较远的区域反而“步调一致”。
基于此，2026年1月20日，卡罗林斯卡学院神经科学系Marie Carlén 研究团队在nature neuroscience杂志发表了“A prefrontal cortex map based on single-neuron activity”揭示了基于单神经元活动的前额叶皮层图谱。作者通过记录清醒小鼠超过24,000个前额叶皮层（PFC）神经元的活动及其空间位置，发现高分辨率的功能图谱并不匹配传统细胞构筑定义的亚区边界；相反，自发活动和行为任务中对“选择”的调谐均与PFC内部的连接性层级结构高度一致，表明连接性而非细胞构筑主导了功能组织。研究还发现，PFC及其高层级区域以低频、规律性自发放电为特征，而对“选择”敏感的神经元显著富集于高自发放电群体中，揭示了基于连接性的层级结构与特定功能特性之间的紧密关联。该数据驱动方法为跨脑区、跨物种解析大脑功能组织提供了可扩展的新范式，推动了对神经活动、结构与功能的整合理解。图一 小鼠大脑中自发放电单神经元活动模式的表征作者在清醒小鼠中记录了24,000多个神经元的自发活动，其中一半来自PFC，涵盖11个传统亚区。他们发现，单个神经元的放电模式虽稳定，但彼此差异显著。为此，用三个简单指标描述每个神经元：放电率（多活跃）、爆发性（是否成簇放电）和记忆性（放电节奏是否有规律）。后两者能更客观地刻画放电的时间结构。根据电脉冲波形宽度，神经元被分为宽波宽（多为兴奋性）和窄波宽（多为抑制性）两类。窄波宽神经元确实更活跃、放电更“有节奏”。后续分析主要聚焦于宽波宽神经元，以深入探究PFC内部的功能组织。图二 前额叶皮层的放电模式揭示其功能分区与抑制主导的工作状态为了全面理解大脑神经元在自然状态下的放电模式，作者基于24,000多个宽波宽神经元（多为兴奋性）的三个特征：放电率、爆发性（是否成簇放电）和记忆性（放电节奏是否规律）训练了一个自组织映射（SOM）。SOM将神经元按放电模式自动聚类，结果显示：放电率和爆发性彼此独立变化，而高记忆性通常出现在低爆发性区域，说明“规律放电”和“爆发放电”在实际中互斥。每个神经元被归入最匹配的SOM节点，这些节点在不同小鼠间高度一致，说明结果可靠。PFC的神经元广泛分布于SOM中，尤其偏好低放电率、规律放电的模式；而其他脑区（如丘脑、海马）则更多表现为高爆发性。进一步分析发现，PFC内部也存在差异：内侧区域（如内侧前额叶、前边缘皮层、前扣带皮层）显著富集规律放电神经元，而外侧区域（如外侧眶额皮层、次级运动皮层）则偏向爆发放电。为量化这种差异，作者引入“E分数”，发现皮层浅层多为爆发型神经元，深层则以规律型为主（因技术限制，95%记录来自深层）。使用另一独立数据集验证后，结果高度一致。对窄波宽神经元（多为抑制性）的单独分析也显示类似格局：内侧PFC富集高放电率但规律放电的抑制性神经元，而外侧区域则多为高放电率且爆发性强的类型。综合来看，内侧前额叶皮层整体呈现“低活跃但高度规律”的放电特征，这可能源于局部抑制性神经元的强调控。这种以抑制为主导的稳态，或许正是前额叶支持复杂认知（如决策、注意力）所需的精细信息处理基础。图三 自发放电活动图谱反映了前额叶皮层内部的层级结构鉴于小鼠前额叶皮层的细胞构筑与连接结构不一致，作者将该区域细分为42个更小的功能单元，基于深层宽波宽神经元的自发放电模式（如放电率、规律性、爆发性）计算“E分数”进行分析。结果发现：内侧前额叶和前边缘皮层高度富集低放电率、规律放电的神经元，形成一个均一的功能模块；而次级运动皮层和外侧眶额皮层腹侧则富集爆发性强的神经元。这些活动模块常跨越传统解剖边界，说明经典亚区内部并不功能同质。高分辨率数据还显示，外侧眶额皮层中仅最腹侧一小块具有爆发特征，过去因分辨率不足被高估。内侧区域的活动相似性支持基于连接的“腹内侧功能分区”假说，但未发现背内侧或腹外侧对应的特异模式，可能因本研究聚焦输出层（深层），而连接图谱包含输入信息。

进一步与之前基于单神经元追踪定义的60个连接层级区域比对，发现高度一致：层级越高的区域（如前边缘皮层）；层级越低的区域（如外侧眶额腹侧）。尽管内侧前额叶未提供层级得分，却是规律放电富集最强的区域，暗示其处于更高层级。综上，自发放电模式与连接层级紧密关联，却与细胞构筑分区脱节，提示应超越传统解剖划分，采用融合活动、连接与层级的新框架理解前额叶功能组织。

总结

作者的研究表明，自发放电模式、感觉反应和任务调谐等不同神经活动特征，为理解PFC的功能组织提供了互补视角。特别是，自发放电特性与基于连接的层级结构紧密关联并预测神经元参与感觉响应和任务调谐的倾向，凸显连接性而非细胞构筑才是PFC功能组织的核心原则。该发现挑战了传统解剖分区范式。作者提出的数据驱动框架具有可扩展性，为未来在更丰富行为和刺激条件下解析PFC乃至其他脑区的功能组织提供了有力工具。
