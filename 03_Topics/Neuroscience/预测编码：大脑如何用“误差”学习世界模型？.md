# 预测编码：大脑如何用“误差”学习世界模型？

- **时间**: 2026-02-04 23:16

原文链接: https://mp.weixin.qq.com/s/jmnS7__CwJKEdVMkBU7u8Q

如果问一个 AI 工程师：神经网络是如何学习的？他会告诉我们：反向传播（Backpropagation）。即：网络先输出结果，计算总误差，然后把这个误差通过复杂的微积分链式法则，一层层传回网络的每一个权重。但是，大脑可能不这么做。生物神经元没有全局时钟，无法保存复杂的导数，也无法让信号精准地逆向流过轴突。那么，大脑这台只有 20 瓦功率的湿件计算机，是如何实现比高能耗的大模型更高效的学习的？1.层级化的“预测-误差”回路预测编码的核心思想由Rao和Ballard在1999年的一篇经典论文中奠定，后来被Karl Friston发扬光大。它认为皮层是一个层级生成模型。1.1规则：向下预测，向上报错想象大脑是一个跨国公司：•CEO（前额叶/高阶皮层）：坐在办公室里，看不见市场，只能根据报表做“预测”：“下季度销量应该涨10%”。•中层经理（视觉联合皮层）：接收CEO的预测，将其细化：“那意味着A地区要卖5000台”。•一线员工（V1/视网膜）：真正看到现实：“A地区只卖了3000台”。关键：一线员工不需要把“卖了3000台”汇报上去。员工只需要汇报“差额”（-2000台）。这就是预测误差（Prediction Error）。1.2规范微回路（Canonical Microcircuit）预测编码之所以不仅是心理学假说，是因为解剖学家在大脑皮层的6层结构中找到了对应的硬件：•第5/6层（深层）：负责发送预测。它们的轴突向下投射到低级皮层。•第2/3层（浅层）：负责计算误差。它们接收来自高层的预测和来自底层的输入，两者相减，算出的误差向上投射。

这被称为“规范微回路假说”。如果这是真的，那意味着整个新皮层——不管你是看、听、还是动——都在运行同一套算法代码。2.为什么我们要相信这个理论？2.1奇怪的“遗漏反应”当你听“滴、滴、滴、滴……”时，你的听觉皮层反应会越来越弱（因为预测完美，误差为0）。突然，声音停了。此时，你的听觉皮层会爆发出一阵强烈的活动。请注意：没有任何声音输入（物理能量为0），但神经元却爆发了。

唯一的解释是：大脑预测会有“滴”，但现实是“空”，于是产生了一个巨大的“缺失误差”。这直接证明了神经元编码的是误差，而不是输入本身。2.2解释错觉：卡尼萨三角

看这张图，你是不是看到了一个白色的三角形？其实物理上并不存在三角形。预测编码解释说：大脑为了解释那三个“吃豆人”的形状，提出了一个最高效的假设（Prior）——“上面盖了一个白色三角形”。

这个“预测”非常强，强到甚至抑制了V1区对“缺口”的误差报告。你看到的不是现实，是你大脑为了消除误差而生成的“解释”。3.争议与批判：理论的边界预测编码目前仍面临巨大的争议，主要集中在以下三点：3.1争议一：它是“万能药”还是“万金油”？批评者（如一些认知心理学家）认为，预测编码变成了一个“Just-so story”（想当然的故事）。它太灵活了。如果实验发现神经元反应增强，支持者说是“预测误差大”；如果反应减弱，说是“预测成功被抑制”。一个能解释所有现象的理论，往往无法预测任何新现象。科学需要的是可证伪性。3.2争议二：它是因还是果？许多支持预测编码的现象（如适应效应），也可以用传统的前馈疲劳（Fatigue）机制解释：神经元看多了累了，所以反应弱，不需要引入复杂的“预测-误差”计算。虽然近年来的光遗传学实验开始区分这两者，但彻底的定论尚未形成。3.3争议三：注意力是“精度”还是“增益”？我们在[注意力是怎样产生的，什么是“心流”状态？](https://mp.weixin.qq.com/s?__biz=MzI3NTQ2NjQyNw==&mid=2247484352&idx=1&sn=4940ea9b5cb1cad191a51a6bf976afbd&scene=21#wechat_redirect)讲过，预测编码认为注意力是调节“误差的精度”。但在生理学上，有些注意力机制表现为直接放大信号（Gain），而不是调节方差。这两者在数学上不等价。这表明预测编码的数学模型可能过于简化了生物现实。4.理论分级为了理清现状，我们可以把“预测编码”的信徒分为四个层级：Level 1：描述性假说（Descriptive）主张大脑利用上下文来处理信息。Level 2：算法性假说（Algorithmic）主张大脑具体通过 Input - Prediction = Error 的公式来计算。Level 3：实现性假说（Implementational）主张皮层第 2/3 层是误差神经元，5/6 层是预测神经元。Level 4：大一统理论（Grand Unified Theory）主张预测误差最小化是大脑（乃至生命）唯一的指令。

5.结论如果我们将预测编码的逻辑推演到极致，会发生什么？如果大脑不仅想感知世界（最小化感知误差），还想改变世界（最小化行动误差）；如果大脑不仅想预测现在，还想预测未来的存活概率；那么，所有的生物学功能——从细胞代谢到意识思考，是不是都可以归结为一个物理学方程？有一个人认为是这样的。他构建了一个被称为“神经科学的大统一理论”的框架。有人说他是天才，有人说他是疯子，还有人说他的数学公式没人能看懂。下一篇：自由能原理（FEP）：把感知、行动、注意统一到一个方程里（以及为何它难证伪）。参考文献1.理论奠基(Foundational Theory)Rao, R.
