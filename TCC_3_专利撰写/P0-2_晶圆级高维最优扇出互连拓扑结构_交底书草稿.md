# P0-2 晶圆级高维最优扇出互连拓扑结构 交底书草稿

## 发明名称

面向晶圆级大模型推理的高维最优扇出互连拓扑结构

## 技术领域

本发明涉及晶圆级系统、片上网络、片间互连、拓扑优化与大模型推理加速技术领域，尤其涉及一种适用于晶圆级或大面积异构集成系统的高维最优扇出互连拓扑结构。

## 背景技术

随着大模型推理规模持续增长，现有晶圆级或大面积集成系统面临如下问题：

- 节点数量增长后，传统 `Mesh/Torus/Fat-Tree` 难以兼顾带宽、延迟和布线复杂度
- 高维互连设计多停留在概念层，缺少面向物理布局和最坏时延约束的结构方案
- 在无 HBM 或片上存储受限条件下，互连结构成为决定整体吞吐和时延的核心瓶颈

## 现有技术缺陷

1. 固定二维或低维拓扑难以有效支撑晶圆级规模的高频集合通信。
2. 现有拓扑方案对扇出、跨区跳线和拥塞路径缺乏统一设计。
3. 传统拓扑优化往往偏算法仿真，未与实际物理布局边界强绑定。
4. 无法面向推理负载特征构建低最坏时延的结构。

## 发明要解决的技术问题

本发明旨在提供一种高维最优扇出互连拓扑结构，使晶圆级系统能够在有限布线资源、受约束存储和高并发推理场景下，获得更优的：

- 平均通信时延
- 最坏情况下通信时延
- 布线可实现性
- 扇出利用率与扩展能力

## 发明内容

### 一、拓扑结构核心思想

本发明提出一种高维最优扇出互连拓扑结构，基于以下要点构建：

- 以元拓扑作为基础构件
- 通过 `SDI-Bond` 或等效拓扑复合规则生成高维复合网络
- 对远距离跳线、跨区连接、局部聚类和全局汇聚路径进行联合约束
- 面向目标工作负载对拓扑进行结构化扇出设计

### 二、结构组成

该拓扑结构可包括：

- 局部计算簇
- 中间汇聚节点或交换节点
- 跨区域扇出链路
- 全局高维联接骨架
- 面向负载的拓扑切换或多模式映射接口

### 三、关键方法

1. 根据晶圆面积、节点数量、互连密度和目标负载参数建立约束模型。
2. 将通信需求分解为局部聚合、跨区传输和全局归约三类路径需求。
3. 构建多维扇出候选拓扑，并对每个候选结构评估平均时延、最坏时延、跳数、链路占用和布线代价。
4. 选择满足物理约束且综合指标最优的高维扇出结构。
5. 将所述结构映射到具体晶圆级或大面积系统中。

### 四、关键创新点

- 将“扇出设计”与“物理布线约束”统一考虑
- 将“高维拓扑选择”与“目标推理负载特征”绑定
- 支持在晶圆级系统中构造更低最坏时延的互连结构
- 为后续 `SDIO-N` 和原语级执行提供物理骨架

## 有益效果

本发明可带来以下效果：

- 在相同资源下降低通信最坏时延
- 提高大模型推理中的全局信息汇聚效率
- 降低跨区通信热点和拥塞概率
- 为晶圆级、芯粒级和大面积基板级互连提供统一设计思路

## 具体实施方式

### 实施例 1：晶圆级 LLM 推理互连

- 在晶圆级计算阵列中划分多个局部计算簇
- 为每个局部簇配置汇聚节点
- 采用高维扇出骨架连接多个局部簇
- 对 `AllReduce / AllGather / Token` 分发等负载执行不同映射策略

### 实施例 2：板级-晶圆级协同扩展

- 板内采用局部高连通拓扑
- 板间采用受控高维扇出链路
- 通过统一约束模型保证扩展时延不发生断崖式恶化

## 拓扑参数与最坏时延说明

为支撑权利要求和实施例，本发明定义以下关键参数和性能对比，代理人可基于此绘制参数曲线和对比表。

### 核心拓扑参数

1. **节点规模 ($N$)**：晶圆级系统包含的计算/汇聚节点总数。
2. **端口基数 ($k$)**：每个节点可用的物理互连端口数（如 8、16、64）。
3. **拓扑维度 ($D$)**：网络展开的维度，本发明采用 $D \ge 3$ 的高维结构。
4. **最坏跳数 ($L_{max}$)**：网络直径，即最远两个节点间通信所需的最大跳步数。
5. **扇出因子 ($F$)**：局部簇到跨区链路的汇聚比和扇出比。

### 性能对比表（Mesh vs. 高维最优扇出）

在固定端口基数 $k=4$ 且节点规模 $N=1024$ （如 $32 \times 32$ 阵列）下：

| 对比项 | 传统二维 Mesh | 本发明高维最优扇出拓扑 |
| --- | --- | --- |
| 维度 ($D$) | $2$ | $\ge 3$（复合元拓扑） |
| 平均跳数 ($L_{avg}$) | $\approx 21.3$ | 显著降低（依赖具体 $D$ 和扇出） |
| **最坏跳数 ($L_{max}$)** | $62$ ($31+31$) | **显著收敛（如 $< 15$）** |
| 拥塞热点分布 | 中心易拥塞 | 跨区链路负载均衡 |
| 规模扩展性 | 扩展墙效应明显 | 保持对数级最坏时延增长 |

本发明通过在局部构建高连通簇（如星型或全互连），并在跨区构建结构化的高维跳线，打破了二维布局下 $O(\sqrt{N})$ 的最坏时延增长曲线，将其优化至更接近 $O(\log_k N)$ 的量级。

5. 根据权利要求 1 所述的拓扑结构，其中所述拓扑结构支持 `AllReduce`、`AllGather`、`Token` 分发或其它集合通信映射。

7. 根据权利要求 6 所述的方法，其中所述负载约束模型与大模型推理阶段的多级汇聚需求相关联。

## 现有技术对比

| 对比项 | 传统低维拓扑 | 本发明 |
| --- | --- | --- |
| 拓扑维度 | 二维或低维为主 | 高维复合拓扑 |
| 设计依据 | 偏经验或偏算法 | 物理约束与负载联合优化 |
| 关注指标 | 平均性能为主 | 平均性能 + 最坏时延 |
| 扇出策略 | 缺少结构化扇出设计 | 显式设计跨区扇出 |

## 实施参数边界建议

- 节点规模：可覆盖 `16`、`64`、`256` 或更大节点规模
- 扇出层级：优选 `2-4` 层
- 跳数约束：优选对关键归约路径设置最大跳数上限
- 存储条件：适用于无 HBM、有限 SRAM 或分层缓存场景
- 优化目标：至少包含平均时延、最坏时延、链路占用率中的两项
