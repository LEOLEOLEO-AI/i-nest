# 专利布局总览

生成时间：2026-05-03

## 目标

把当前已经分散在项目、论文和战略文档中的专利方向，压缩成可推进的专利池，作为 `30_Outputs` 的正式组成部分。

## 当前判断

- 目前专利方向已经很清楚，但尚未从项目和论文材料中独立抽取成专利交底文档
- 近期最重要的不是“列出更多专利名”，而是把最容易形成壁垒、最容易被抢位的专利先写出来
- 当前专利应服务两类目标：
  - 防守：保护 `TCC / SDI / SDIO-N / 网内原语 / 拓扑重构`
  - 进攻：围绕 `CST / 忆阻器 / 类脑 / SDSoW` 形成中长期高壁垒池

## 专利池

### P0：近期优先交底

1. 一种支持网络中心计算的软件定义互连系统架构及调度方法

- 价值：这是 `TCC + SDI` 近期最核心的总专利，覆盖系统架构、原语核和重构控制
- 对应项目：海河实验室重大专项
- 对应论文：`B1/B2/B3`
- 当前工作：收敛系统框图、接口层次、原语边界和权利要求主链

2. 面向晶圆级大模型推理的高维最优扇出互连拓扑结构

- 价值：针对晶圆级/高维互连拓扑的关键卡位专利
- 对应项目：海河主线、晶圆级中长期路线
- 对应论文：`B2`
- 当前工作：补拓扑仿真与最坏时延、布局规则、扇出设计约束

3. 一种面向万亿参数大模型的网内原生 AI 梯度归约通信加速系统

- 价值：针对 `AllReduce` 等集合通信的直接工程专利
- 对应项目：海河主线
- 对应论文：`B1/B3`
- 当前工作：明确在网累加、门控同步、数据面路径与软件栈拦截关系

### P1：中期跟进

4. 基于查找表的网内二值化神经网络硬件加速装置及方法

- 价值：已有明确成果基础，适合补强已有网内计算方向
- 对应项目：网内 AI 推理与 FPGA 原型
- 对应论文：已有相关成果展示

5. 基于时空协同复杂度的神经网络架构自适应演化方法

- 价值：这是 `CST` 从理论走向方法专利的关键入口
- 对应项目：NSFC 重大项目
- 对应论文：`A1/A11/A9`

6. 一种模拟生物突触可塑性与稳态自由能的神经形态芯片资源分配方法

- 价值：连接 `FEP/STDP`、类脑、忆阻器和神经形态硬件
- 对应项目：NSFC 中长期路线
- 对应论文：机制层和器件层论文
