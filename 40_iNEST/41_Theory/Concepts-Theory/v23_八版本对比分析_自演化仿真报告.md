## 二、核心发现

### 2.1 v11的高σ是假象

v11 σ=7.019 表面优秀，但 **L=0.000**——图已断开。σ=C/C_rand / L/L_rand，当 L→0 时 σ→∞。v11 的 FEP 吸引子引导过度，将网络推入"星形"退化结构：
- 所有节点汇聚到少数 hub
- 路径长度趋于零（但图不再连通）
- EL 坍缩到 0.5%（全电突触）
- 收敛率 99.6%（过度锁定）

**教训**：σ 必须配合 L>0（连通性）检验才有意义。

### 2.2 v22的自适应θ机制最关键

v22 用 **自适应θ周期**（T_theta(t) = T_base × L(t)/L_ref / v_c）控制 LTP/LTD 节奏：
- 网络小时 θ 短 → 快速固化 → 避免 LTP 过饱和
- EL 39.1% 在 [15-28%] 目标附近，健康
- σ=2.736 虽未达 4.0，但图连通且结构稳定

### 2.3 融合的关键障碍：FEP信号未穿透可塑性规则

v23 三次迭代中，FEP 吸引子机制（盆地追踪、梯度裁剪、JEPA 自适应）仅作为**旁路计算**添加，未整合进 `apply_rules()`/`stdp_update()` 核心：
- θ 调制太间接（仅影响 LTP/LTD 频率，不影响方向）
- 直接 h 状态 FEP 推（η=0.015）远弱于级联动力学
- 结果：参数级融合对 σ 无影响（始终 2.736）

### 2.4 v22+连接组版本：EL健康但σ静态

将最小作用量观测器接入真实连接组后：
- EL 固化路径健康：29.7% → 66.4%
- σ=8.500 恒定（基于静态图拓扑，边权重变化不改变图结构）
- 异常检测始终"临界"（σ 无法向目标 5.87 收敛）
- dS/dt 持续为负（系统向最小作用量演化）

## 三、关键洞察与建议

### 3.1 融合的正确路径

v11 吸引子（σ 结构）和 v22 自适应θ（EL 多样性）的融合需要**可塑性层级的整合**，而非参数叠加：

```python
# 伪代码：正确的融合架构
def stdp_update_with_fep(active_mask):
    for each edge:
        # v22: 自适应θ决定是否触发LTP
        if n_ltp > theta_ltp_current:
            # v11: FEP盆地引导决定固化方向
            fep_signal = F_converged[src]  # 源头节点是否已达吸引子
            if fep_signal:
                weight *= 1.1  # 稳定固化
            else:
                weight *= 1.05  # 探索性固化
```

### 3.2 动态σ计算

v22+ 的教训：σ 必须基于**动态图拓扑**（包括边权重），而非仅静态连接：
- 权重低于阈值的边应从图中排除
- σ 应同时追踪 σ_weighted（加权图）和 σ_binary（二值图）

### 3.3 下一步优先事项

1. **v24：深度FEP-STDP融合** — 将盆地收敛信号嵌入可塑性规则
2. **动态σ指标** — 基于权重感知的图拓扑计算
3. **连通性保持约束** — 在FEP梯度下降中加入 L>0 硬约束
4. **长程仿真** — N_STEPS≥500，观察EL和σ的长期稳态
