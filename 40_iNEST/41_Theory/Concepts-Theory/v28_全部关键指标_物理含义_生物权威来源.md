## 三、BCM 滑动阈值（v25 新增，v28 增强）

### 物理第一性

BCM (Bienenstock-Cooper-Munro, 1982) 是视觉皮层可塑性的经典理论：

```
θ_bcm(t+1) = θ_bcm(t) + η × h² × (h - θ_bcm)
```

- **h² 项**: 突触后活动平方 → 高频活动比低频活动对阈值的推动力大得多（非线性）
- **(h - θ_bcm) 项**: 反馈项——h > θ 时阈值上升（防饱和），h < θ 时阈值下降（防沉默）
- **收敛点**: θ* = E[h³]/E[h²]，即稳态阈值 = 活动立方期望/平方期望

### 生物对应

| 现象 | BCM 解释 | 实验证据 |
|------|---------|---------|
| 视觉剥夺 → LTD 阈值下降 | h 低 → θ 下降 | Hubel & Wiesel 单眼剥夺, 1960s |
| 高频刺激 → LTP 饱和后 LTD | h 高 → θ 上升过猛 | Kirkwood et al. 1996, *Nature* |
| 经验依赖的关键期 | θ 随发育滑动 | Bear & Singer 1986, *Nature* |
| 稳态可塑性 | θ 动态平衡 LTP/LTD | Turrigiano & Nelson 2004, *Nat Rev Neurosci* |

### v28 增强：惊讶度耦合

```
η_eff = BCM_ETA × (1 + 0.8 × tanh(surprise_i))
```

- **物理**: 高惊讶度 → 模型不可靠 → 加速适应速率
- **生物**: 去甲肾上腺素在意外事件时释放 → 全局可塑性窗口打开 (Yu & Dayan 2005, *Neuron*)

### 参数演进

| 版本 | BCM_ETA | θ 范围 | 惊讶度耦合 |
|------|---------|--------|-----------|
| v25 | 0.005→0.05 | 8.0→7.9 | 无 |
| v26 | 0.08 | [7.6, 8.0] | 无 |
| v27 | 0.15 | [7.6, 8.0] | 无 |
| **v28** | **0.25** | **[7.0, 8.0]** | ✅ tanh 耦合 |

## 四、小世界系数 σ（核心结构指标）

### 物理第一性

```
σ = (C/C_rand) / (L/L_rand)

C    = 实际聚类系数（邻居之间也互相连接的概率）
L    = 实际平均最短路径长度
C_rand = p = M/(N(N-1))  (随机网络聚类)
L_rand = ln N / ln(N×p)  (随机网络路径)
```

Watts & Strogatz (1998) 定义：**σ > 1 就是小世界网络**。σ 越大 → 同时具备高聚类（局部处理）和短路径（全局通信）。

### 生物基准值

| 网络 | N | σ | 数据来源 |
|------|---|-----|---------|
| C.elegans | 279 | **≈5.0** | Watts & Strogatz 1998; v28 仿真 |
| 猫视觉皮层 | ~10⁵ | **≈2-4** | Sporns & Zwi 2004, *Neuroinformatics* |
| 猕猴皮层 | ~10⁷ | **≈2-3** | Modha & Singh 2010, *PNAS* |
| 人脑 (fMRI) | ~10⁴ 脑区 | **≈2-5** | Bullmore & Sporns 2009, *Nat Rev Neurosci* |
| v28 N=1953 仿真 | 1953 | **19.45** | 偏高 — 真实连接组 motif 在大尺度放大 |

**注意**: 仿真 σ 偏高是因为真实连接组的生物 motif（前馈环、双向连接等）在 scale_connectome 中被精确复制，导致聚类异常高。实际大脑的 σ 因长程投射稀释聚类而下降。v28 的跨模块连接密度（5%）可能偏低。

### σ 目标值的含义

- σ < 1: 随机网络（无结构）
- 1 < σ < 3: 弱小世界（接近随机）
- 3 < σ < 8: 典型小世界（类脑）
- σ > 10: 超小世界（可能过度规则化）

## 五、E-L 固化比例（核心功能指标）

### 物理含义

```
EL = N(E-L bonds) / N(所有化学键)

E-S (btype=0): 候选态 — 低激活能 Ea=0.15, 高可塑性
E-L (btype=2): 固化态 — 高激活能 Ea=0.85, 低可塑性, 对应记忆
```

### 生物对应：突触的"记忆 vs 学习"平衡

| EL 比例 | 生物类比 | 功能含义 |
|---------|---------|---------|
| < 10% | 幼年关键期 | 高可塑性，快速学习 |
| **15-35%** | **成年正常脑** | 记忆-学习平衡 |
| 35-50% | 过度固化 | 记忆僵化，学习困难 |
| > 70% | 病理状态 | 可能是 PTSD/成瘾/强迫症 |

### 生物权威数据

**成年小鼠皮层**: 约 20-30% 的突触为稳定态（稳定超过 30 天）
- Holtmaat & Svoboda 2009, *Nat Rev Neurosci*: 树突棘 turnover ~3-5%/day, 稳定态 ~73%
- Yang et al. 2009, *Nature*: 运动学习后 ~20% 新突触稳定化

**v28 仿真 EL=28-31%**: 在生物正常范围内。

## 六、平均度 k_avg（网络稀疏性指标）

### 跨物种标度律

```
k(N) ≈ k₀ × N^ε    k₀ ≈ 10 at N=279, ε ≈ 0.14
```

| N | k_avg | 物理含义 |
|---|-------|---------|
| 279 | ~10 | C.elegans: 每个神经元 ~10 连接 |
| 10⁴ | ~22 | 果蝇蘑菇体 |
| 10⁷ | ~100 | 小鼠皮层 |
| 10¹⁰ | ~1000 | 人脑皮层 (突触层面 ~8000, 神经元层面 ~1000) |

**物理第一性**: 空间填充约束 — 轴突/树突的物理体积限制，电线 minimization 原理 (Chklovskii et al. 2002, *Neuron*)

## 八、FEP 收敛度（分级，非二值）

```
F_convergence[i] = sigmoid((basin_count[i] - 10) / 5)
                 ∈ [0, 1]: 连续谱, 非 yes/no
```

| F_convergence | 解释 |
|---------------|------|
| 0.0-0.3 | 探索态: 自由能仍在下降, 模型未收敛 |
| 0.3-0.7 | 过渡态: 盆地形成中 |
| 0.7-1.0 | 收敛态: 预测可靠, 输出可固化 |

**v28 典型值**: cv=0.99 (高收敛) — 300 步后几乎所有节点收敛。

## 九、最小作用量 S(t)

```
S(t) = ∫₀ᵗ [σ(t)·v_prop - Σ Ea·|dw/dt|] dt
       └─ 效率收益 ─┘   └─ 拓扑变更代价 ─┘
```

**物理第一性**: Hamilton 最小作用量原理 — 自然系统沿作用量最小的路径演化

**生物对应**: 代谢效率 — 大脑在能量约束下最大化信息传输效率 (Laughlin & Sejnowski 2003, *Science*)

**v28 dS/dt 典型值**: 负（效率提升跑赢代价，系统在优化）

## 十、BCM v28 惊讶度耦合机制

```
surprise_i = |F_local[i] - mean(F_history)| / std(F_history)
η_eff = BCM_ETA × (1 + 0.8 × tanh(surprise_i))
```

| surprise | η_eff 倍数 | 生物类比 |
|----------|-----------|---------|
| 0 | 1.0× | 无意外 → 常规 BCM 速率 |
