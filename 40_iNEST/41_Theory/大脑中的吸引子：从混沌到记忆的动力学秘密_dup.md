## 一、引言

❝
One of biology's grand challenges is to explain how order and complex function spring from inanimate physical systems composed of much simpler parts.
从微观的神经元放电到宏观的记忆、导航能力，大脑这个由数十亿神经细胞组成的“湿件”，始终在演绎着“简单部分涌现复杂功能”的奇迹。我们如何记住回家的路？如何在混乱的感官输入中提取稳定的信息？答案或许藏在一个优雅的数学概念中——**吸引子（Attractor）**。它就像隐藏在神经活动背后的“无形之手”，让无序的信号汇聚成稳定的模式，让大脑在变化中保持核心功能的有序。
本文主要是对于2022年的综述文章：Attractor and integrator networks in the brain[1]其主干内容的介绍。
## 二、从“动态系统”到“吸引子”

要理解吸引子，我们得先从“动态系统”说起。
### 什么是动态系统？

引用原论文中的定义：
❝
A dynamical system is a set of **variables** together with all the rules that determine their changes in value with the passage of **time**. The value of these variables at any given instant is called the state of the system at that moment. The state is a point (vector) in the state space of the dynamical system.
简单说，动态系统就是“一组变量+变化规则”的组合。比如：太阳系中行星的位置和速度是变量，万有引力定律是规则，它们共同构成了描述行星运动的动态系统；而大脑中，神经元的放电频率、神经递质浓度是变量，突触连接的兴奋/抑制规则是变化引擎，共同构成了神经动态系统。
这里的“状态空间”可以想象成一个多维坐标系：每个维度对应一个变量（比如一个神经元的放电率），系统在某一时刻的状态就是这个空间中的一个点。随着时间推移，这个点会按照规则“移动”，形成一条轨迹——这就是系统的动态演化。
### 什么是吸引子？

❝
An attractor is the **minimal** set of states in a state space, to which all nearby states eventually flow with time.
如果动态系统的状态空间是一片地形，吸引子就像“山谷”：无论你从山谷周围的哪个位置（附近状态）出发，最终都会滑向山谷底部（吸引子状态）。它是系统演化的“最终归宿”，是无序中涌现的稳定秩序。
举个生活中的例子：钟摆的摆动就是一个动态系统，最终会停在最低点（静止状态）——这个最低点就是一个吸引子；而大脑中，当你听到一段熟悉的旋律时，经过冥思苦想之后终于回忆起是哪首曲子的时候，神经活动所稳定在的那个模式，就是记忆的吸引子（事实上，这也正是[Hopfield网络](https://mp.weixin.qq.com/s?__biz=MzkyMjYxODEzOA==&mid=2247484594&idx=1&sn=1ce035866adcaa431b303180cab9e8ae&scene=21#wechat_redirect)的建模思路）。

稳定记忆就像是记忆山谷中的谷底
## 三、吸引子的假设与简化

大脑不是抽象的数学模型，要让“吸引子”理论落地，需要满足一些关键假设和要求。
### 1. “有效自主”的动力学：排除外部干扰的“纯态”

神经系统的吸引子研究依赖一个核心假设：我们需要观察**“effectively autonomous” dynamics**（有效自主动力学）。这意味着：输入信号要么不随时间变化，要么“未被调谐”——即不会针对性地驱动吸引子状态的特定子集。
为什么要这样？因为吸引子的核心特性是**内在稳定性**，如果外部输入持续“推”着系统走，我们就分不清观察到的稳定模式是系统自己的“选择”，还是被输入“强迫”的结果。就像观察钟摆的自然摆动时，得先关掉外力推动，才能看到它最终停在最低点的吸引子特性。
❝
简单说，调谐就是神经元（或输入通路）对某些特征（刺激、位置、方向、速度、声音频率……）有偏好：当那种特征出现时，它们活跃得更多；当不出现时，活跃得更少。用更技术的说法，神经元有一条 tuning curve（调谐曲线），描述其放电率如何随某个外部变量变化。 举几个直观例子帮大家理清概念：
- 
视觉皮层里某些细胞对边缘方向有偏好，看一个特定方向的线条它们就高放电——这是“方向调谐”。
- 
海马区的place cell在动物到达某个位置时强烈放电——这是“位置调谐”。
### 2. 用神经元的“输出”定义状态：聚焦功能相关的变量

在神经系统中，我们通常**用神经环路中神经元的放电输出（spiking outputs）作为系统状态的变量**。这是因为：神经元的放电是信息传递的核心载体，而吸引子的意义正是解释“这些放电如何形成稳定功能”。
比如在记忆研究中，我们关注的是“哪些神经元同时放电”“放电频率如何变化”，这些放电模式构成的状态，才是与记忆功能直接相关的吸引子载体。如果选择无关变量（比如神经元的大小），就无法捕捉到功能层面的稳定模式。
这些假设和要求，为我们在混乱的神经信号中“捕捉”吸引子提供了标尺。接下来，我们将通过具体的神经环路例子（比如网格细胞、记忆网络），看看吸引子如何在大脑中“工作”——而Fig 2中的图像，正是这些抽象概念的直观呈现。

### 4，2 吸引子的特性

吸引子在神经计算中重要，不只是因为它能把高维活动“收束”起来，更在于收束后的几何形态和动力学特性共同赋予网络鲁棒的表征与运算能力。把这些特点连成一句话：吸引子把高维的噪声世界压缩到低维的有效流形上，并在该流形周围建立起**“拉回力”**，只让与表征相关的分量长期保存，而把无关扰动迅速耗散。
首先是**隐藏的低维流形**。大脑的状态空间维度非常高——每个被记录的神经元都是一个坐标维度。——但在许多任务和表征下，真实的活动并不填满整个空间，而是限制在一个更低维的子空间或流形上（例如一条线、一片面或一个闭合环）。把这种现象想象成三维房间里的地毯：尽管房间能容纳任意三维运动，实际活动却都限制在地毯上。低维流形的好处显而易见：解读、解码和存储都变得更可行——系统只需要管理那条“便道”上的坐标，而不是全部无用的自由度。
其次是**去噪的机制与“拉回力”到底是什么**。直觉上，吸引子在状态空间上像一道沟槽或碗：噪声把状态推上坡，但坡度会把它再次拉回沟底。把系统在线性化后，可以把小扰动  的演化写成

最后是**维度的 Goldilocks 原则**：吸引子的维度既不能太低也不能太高。若吸引子维度接近或等于整个状态空间的维数，那么“约束”就不存在，系统无法把噪声有选择性地耗散——状态会像在平原上随机游走，记忆与连续编码的优势丧失。相反，若吸引子维度太低（过度投影），系统可能丢失必要的表达自由度，无法区分不同的行为或环境变量。理想情况介于两者之间：吸引子的维度低于原始空间（带来压缩与鲁棒），但又足够高以承载所需的信息量。这一“恰到好处”的维度选择，是大脑在“容量（可表示能力）”与“鲁棒性（抗噪声能力）”之间的折中。

吸引子与状态空间维度一致=灾难
接着我们在4.1中的双神经元例子，上图展现了不同维度的吸引子的状态。我们发现，如果吸引子的维度和状态空间维度一致，那么处处都是吸引子——吸引子也毫无意义了。
总的来说，吸引子不是静态的标签，而是由网络拓扑、突触动力学与时间尺度共同雕刻出的“几何沟槽”：它把无关扰动沿横向耗散，把关键信号沿切向保留，从而在嘈杂的生物环境中提供既精确又稳健的表征与计算基础。
## 五、吸引子及其存储的记忆类型

在神经网络中，**表征（representation）**指的是将外部输入映射到内部的表征状态（representational states）上，并在需要时能够**稳定、可重复地检索**这些状态（“标签”）。吸引子网络（attractor network）通过形成一组稳定的内部状态，来表示和记忆外部的离散或连续变量。这种映射既可以通过**前馈学习过程**将外部状态关联到内部吸引子状态（长时记忆），也可以在初始化到某个吸引子后，通过**持续的神经活动**维持状态（短时记忆）。
吸引子按照稳定状态的分布形式，可以分为两大类——**离散吸引子**和**连续吸引子**，它们在大脑中承担不同的计算与记忆功能。
### 1. 离散吸引子：像钢琴键一样的孤立稳定态

离散吸引子（Discrete attractors）是一组**孤立的稳定状态**，彼此之间没有连续过渡。每个稳定态可视作一个“记忆单元”或“功能模式”，系统状态在干扰下会回到最近的吸引子点，而不会停留在中间的“混合态”。
就像钢琴的琴键：每个键是一个稳定的音高，按下不同的键会切换到不同的状态，但不会有稳定的“介于 Do 和 Re 之间”的音高。
**功能与例子**
- 
常用于工作记忆：例如记住“苹果”“香蕉”“橙子”三个词，每个词对应一个离散吸引子状态。
- 
即使有噪声或干扰，神经活动也会稳定在当前词对应的状态，而不会变成混合记忆。
- 
Hopfield 网络是典型的离散吸引子模型，其稳定状态是通过长期学习（长时突触可塑性）形成的。
### 2. 连续吸引子：像滑动条一样的连续稳定流形

连续吸引子（Continuous attractors）由**连续分布的稳定状态集合**组成，形成一个低维的流形（manifold），如一条环、一个平面等。系统可以沿着流形平滑地移动，每个位置都对应一个稳定表征。
就像音量滑动条：从静音到最大音量之间，每一个位置都是稳定的，且可以连续调节。
**功能与例子**
- 
常见于需要表示连续变量的神经系统，如空间位置、方向、角度等。
- 
例如网格细胞（Grid cells）编码空间坐标，动物移动时，神经状态沿连续吸引子流形平滑移动，实现**路径积分**（path integration）。
- 
头向细胞（Head-direction cells）形成环状吸引子，代表动物当前朝向。
### 3. 吸引子的两种记忆形式

吸引子网络可同时支持两种记忆形式：
- 
**长时记忆（Long-term memory）**：存储在网络权重结构中，决定了吸引子集合的形状与位置。
- 
**短时记忆（Short-term memory）**：当网络被初始化到某个吸引子状态时，通过持续的神经活动维持该状态。
如果这些短时记忆状态能够仅凭内容（或部分内容）而被重新激活，而不需要显式“地址”，那么它们就是**内容可寻址（content-addressable）**的。例如，在 Hopfield 网络中，一个被噪声污染的输入模式会自动收敛回最近的存储模式。
需要注意的是，这种短时记忆功能依赖于**先前通过长期可塑性形成的稳定状态**。即使是基于突触前易化（presynaptic facilitation）的短时记忆模型，也需要依托于先前形成的稳定性网络结构来保证可重复激活。
**总结**
- 
**离散吸引子**适合存储离散类别信息，如特定物体、词汇或决策状态。
- 
**连续吸引子**适合表示连续变量，如空间位置、运动方向或角度。
- 
吸引子动力学使得系统能够在高维神经状态空间中，沿着低维稳定流形进行鲁棒的表征与记忆。
## 六、Integration

在大脑的“计算工具箱”里，**Integration（积分）** 是实现“长时间尺度信息累积”的核心功能。它让神经环路突破单个神经元的时间局限，把“瞬间信号”转化为“持续的、有意义的累积结果”，就像给大脑装上了“时间计算器”。
### 1.积分的意义

单个神经元的“记忆”很短：受膜电容、离子通道特性限制，神经元只能在 **10-100毫秒** 内整合输入（比如判断“最近有没有接收到强刺激”）。但大脑需要处理的任务，往往涉及**更长时间的信息累积**：
- 
走路时，要把每一步的“速度”积分成“走了多远”（几秒到几分钟的累积）；
- 
转头时，要把“角速度”积分成“转了多少度”（动态、持续的累积）；
- 
甚至思考时，要把“零散的想法”积分成“完整的逻辑链”（分钟到小时的累积）。
如果没有“积分”功能，大脑就会变成“健忘的瞬间记录仪”——每一步、每一个动作的信息都会瞬间消散，无法形成持续的行为、记忆或认知。
### 2. 积分的本质

从数学上看，积分是 **“对变化率的时间求和”** ：
- 
位移 = 速度 × 时间（ ）；
- 
角度 = 角速度 × 时间（ ）。
在神经科学中，积分的逻辑完全一致：神经环路通过**持续整合“变化率信号”（如速度、角速度、注意力变化），生成“累积的状态表征”** 。比如，网格细胞整合“速度信号”生成“位置信息”，运动皮层整合“肌肉收缩信号”生成“肢体位置”，本质都是同一套“神经积分”逻辑。
### 3. 积分如何在大脑里“落地”

单个神经元确实能对输入做短时累积，但它们的膜时间常数通常只有几十毫秒，这不足以解释动物在秒到分钟尺度上对位置、朝向或注视保持的需求。连续吸引子网络提供的思路是把“要积累的量”换成网络状态在一条允许路径（低维流形）上的位移：外部的速度信号被映射为沿着该流形的定向驱动力，状态沿流形移动的距离就是时间积分的结果。
在进一步阐述实现细节之前，先给两个术语简短定义（用精确而不晦涩的语言）：
- 
**bump（活动峰）**：在某些连续拓扑的神经表面上（例如按位置或偏好排列的神经元格子），一小片神经元群体的活动高于背景，形成局部峰值，这个峰的位置可以编码一个连续变量（如空间位置或朝向）。
- 
**limit cycle（极限环）**：系统在状态空间里沿着一条闭合轨道持续流动，形成周期或准周期的活动模式；与静止的 bump 相比，极限环描述的是持续有方向的流动。
实现“沿流形位移以完成积分”的常用方案之一是所谓的 **copy-and-offset（复制与偏移）** 架构。它的关键要点如下（以神经机制而非比喻说明）：将主网络构造成若干功能相近的子网络或子群体，每个副本在局部连接上带有微小的系统性偏移（即连接权重的微小非对称）。单个副本的这些偏移会在其内部产生一个方向性的动力学倾向（例如使活动峰在该副本上倾向于向某一方向滑动）；当互为相反偏移的副本耦合在一起时，会形成一个在无输入状态下总体静止的“推–拉平衡”。速度输入通过差异性投射到不同副本（即某些副本比其他副本得到更强的驱动），这种差异破坏了平衡，使活动峰朝被更强驱动的副本对应的方向移动。总体效果是：活动沿着吸引子流形移动，其瞬时速度由投射到副本集合上的速度分量决定；因此，流形位置的时间导数近似等于速度输入的线性放大加噪声项，

**等等！** 读者们可能会困惑——“速度输入”从何而来？这种信号并非凭空出现，而通常源自动物自身的感知与运动系统——例如前庭系统/内耳提供的角速度信息、肌肉本体感觉或运动命令的副本（efference copy/proprioceptive signals），以及视觉产生的视场运动信息（optic flow）。这些信号可以被送到不同的子网络或不同神经子群，从而以差异化方式驱动 copy-and-offset 结构。
生物学实现上有若干可行机制：微小的不对称连接或方向性突触调制可以产生“偏移”；局部递归兴奋加上较远程抑制能把活动维持成稳定的 bump 并在横向偏离时迅速纠正；慢时程突触成分（例如 NMDA 受体介导的电流）与网络级反馈提供必要的长时常数。实验证据也支持网络级机制而非单细胞内在机制产生积分：例如，眼动积分器在降低网络反馈或加药阻断时变为“leaky integrator”，时间常数显著下降，说明长期保持依赖于网络结构；果蝇神经罗盘的连接构型也明确显示了类似复制与偏移的结构。
### 一句话总结

积分是神经环路突破“瞬间记忆”局限的关键功能，它通过 **连续吸引子的“流形轨道”+“偏移连接动力”+“抗干扰稳定器”** ，实现了“变化率→累积量”的转换。无论是导航、运动控制还是认知决策，积分功能都是大脑“时间微积分”的底层逻辑——没有它，我们连“走直线”“转个弯”都做不到，更别提复杂的行为和思维了。
## 七、大脑中的吸引子

要判断一个神经环路是否真的在使用吸引子动力学，科学家们提出了四个核心标准。这些标准就像一套“检测工具”，帮助我们在复杂的神经信号中锁定吸引子的踪迹。
- 

标注一：稳定性（Stability）吸引子最核心的特质是“抗干扰能力”。就像山谷里的石头，即使被轻微推动，最终还是会停留在山谷底部——神经吸引子也一样，即使受到随机噪声或短暂扰动，系统状态也能保持在吸引子附近，或快速恢复到稳定模式。比如记忆不会因为个别神经元的偶然放电而消失，正是这种稳定性的体现。
- 

标注二：可访问性（Accessibility）一个真正的吸引子网络，其所有可能的稳定状态都必须能被“激活”。对于离散吸引子来说，这意味着通过合适的输入信号，系统能在不同的稳定状态间切换（比如从“记住苹果”切换到“记住香蕉”）；对于连续吸引子，这意味着状态能沿流形平滑移动到任意位置（比如网格细胞能编码环境中每一个坐标点）。如果某些状态永远无法通过输入达到，那就不符合吸引子的定义。
- 

标注三：低维性（Low dimensionality）吸引子必须“嵌入”在低维空间中，其维度远低于整个神经状态空间的维度。大脑的状态空间由成百上千个神经元构成（每个神经元的放电率都是一个维度），但吸引子的活动模式只会被约束在少数几个维度里（通常1-2维）。这就像3D房间里的蚂蚁，虽然身处高维空间，却只在2D的地面上活动——这种低维约束正是吸引子实现高效计算的关键。
- 

标注四：内在动力学（Intrinsic dynamics）吸引子的核心特性必须由神经环路**内部的连接规则驱动**，而不是完全依赖外部输入。在“有效自主”的条件下（即输入不变或没有针对性的驱动信号时），系统仍能展现稳定的吸引子行为。比如网格细胞在黑暗中（缺乏视觉输入）仍能维持六边形放电模式，证明其模式源于内部连接，而非外部信号的直接映射。
### 7.1 离散吸引子：皮层中的Up与Down状态

离散吸引子最简单的非平凡形式是**双稳态（bistability）**，而大脑皮层中的“Up and Down states”（兴奋态与抑制态），就是这种双稳态的经典展现。a.状态切换的“可视化证据”

- 
**MUA（Multi - Unit Activity，多单位活动）与膜电位（）轨迹**：Fig3a上方，MUA代表一群神经元的整体放电活动，是单个神经元的膜电位。可以看到，神经元并非持续放电或沉默，而是在两种状态间切换：
- 
**Down状态**：膜电位超极化（更负），MUA低，神经元近乎“休眠”；
- 
**Up状态**：膜电位去极化（更接近放电阈值），MUA骤升，神经元群体“集体兴奋”。两种状态持续时间可达数百毫秒到数秒，像神经环路的“开关”，稳定又规律。b.双稳态的“量化特征”

- 
**膜电位直方图**：中间的直方图清晰呈现两个尖锐峰值，对应Down（左峰）和Up（右峰）状态。峰值尖锐且分离，说明状态稳定——即使系统存在内在噪声（如神经元随机离子通道波动），神经活动仍会快速“纠正”回这两个状态，不会停在中间模糊地带。
- 
**状态空间散点图**：下方散点图中，横轴是滤波后的膜电位（），纵轴是滤波后的MUA（）。点的分布呈现“双簇状”，对应两种状态：左下角簇是Down状态（低膜电位、低MUA），右上角簇是Up状态（高膜电位、高MUA）。这种聚类，直接证明皮层活动被约束在两个离散的吸引子状态。c.双稳态的“网络起源”
这种双稳态**并非单个细胞的“自主选择”**，而是皮层网络集体作用的结果：
- 
**Up状态维持**：神经元间的“自兴奋连接”（如锥体细胞间的兴奋性突触）持续激活，让群体保持高放电；
- 
**Down状态限制**：全局抑制性连接（如中间神经元的抑制）防止兴奋过度扩散，维持状态边界；
- 
**状态转换动力**：从Up→Down，源于神经元“适应机制”（持续放电后离子通道调整，降低兴奋性）；从Down→Up，由随机事件（如少量神经元自发放电触发群体兴奋）或外部信号驱动，像“重启开关”。d.吸引子标准的“完美契合”
从吸引子四大标准看：
- 
**稳定性**：噪声干扰下，状态会快速回归两个峰值（直方图尖锐、散点聚类），抗干扰能力强；
- 
**可访问性**：通过合适输入（如感觉刺激、内部指令），系统能在Up与Down间切换，两种状态均可被“激活”；
- 
**低维性**：尽管皮层有海量神经元，活动却被约束在“Up/Down”两个离散状态（近似1维吸引子），实现高维到低维的“信息压缩”；
- 
**内在动力学**：双稳态由网络连接（自兴奋+全局抑制）驱动，而非单个细胞特性——即使切断丘脑、纹状体输入，皮层仍能自主产生状态切换（虽与这些脑区同步性强，但核心动力源于皮层内部）。
### 7.2 连续吸引子的实例：网格细胞

内嗅皮层的网格细胞（Grid cells）是连续吸引子动力学的“教科书级证据”。这些细胞通过**周期性三角晶格放电模式**编码空间位置，它们的放电区域在二维环境中形成规则的六边形网格，像瓷砖一样铺满整个探索空间：

一个grid cell的放电情况
假如我们走进了一个大小的房间（上图的横轴坐标就是空间距离），并且我们只关注大脑中**一个**grid cell的放电情况，我们会发现相当神奇的一点——网格细胞的放电位置自发地排列成了无数六边形的组合。而不同的网格细胞响应空间中的不同位置，比如我们假设上图所展示的grid cell为cell1，那么同一“模块”的cell2可能就是上图的图案产生了一个平移。所谓“同一模块”，实际上就是边长相同的六边形。
grid cell的这一特点非常神奇，然而，当一堆grid cell共同作用时，更神奇的事情发生了：

grid cells的“甜甜圈”环面流形：左边环面流形图的横纵坐标本身没有直接的物理意义，它们是非线性降维算法（如 t - SNE、UMAP ）的 “抽象输出” ，核心作用是在二维平面上呈现高维数据的 “结构关系”。相邻点代表 “时间或空间上相似的群体状态”（如动物在相邻位置时，网格细胞的放电模式相似，对应图中相邻的点 ）

环面结构：点云自然聚成近似圆形的流形，且颜色呈现 “螺旋式渐变”—— 这说明群体状态随时间的变化，是沿环面的连续移动：从冷色（早）到暖色（晚），状态绕环面转了一圈又一圈，完美符合 “连续吸引子沿低维流形移动” 的预测。
右图的灰色轨迹表示动物在 2D 环境中实际的运动路径（在开放空间里自由探索的轨迹），而左图中的彩色轨迹则表示群体状态随时间的移动路径（提取颜色对应的点，连起来就是状态轨迹）。
动物在环境中 “绕圈、折返” 时，群体状态在环面上的移动高度同步—— 环境轨迹的 “空间结构”，被编码成了环面流形上的 “状态移动”。 这种同步性证明：网格细胞的群体状态不是随机变化，而是严格对应空间位置的连续吸引子动力学 。
