### 1.2 飞书NICE知识库（31篇）核心内容

#### NICE理论体系核心

> **核心判断**：智能的本质在于连接复杂度，而非节点复杂度。

NICE（Network-Induced Complexity Emergence）理论体系定义了智能涌现的充要条件：

```
I ∝ CST = (S_c · T_c) · e^(α · Γ_st)
```

其中：
- **S_c**：空间复杂度（网络拓扑的结构丰富度）
- **T_c**：时间复杂度（网络动力学的时序丰富度）
- **Γ_st**：时空耦合度（拓扑-动力学的协同程度）
- **α**：涌现增益系数（系统从临界态获得的超线性增益倍数）
- **θ_k**：临界阈值（CST超过此值涌现发生）

**关键推论**：
1. 智能行为是多个复杂度维度**乘积**作用的结果（非加法），维度间存在**协同放大**效应
2. 存在明确的**相变临界点**，系统在临界态时信息处理能力达到最优
3. **涌现增益系数α > 1**时，系统产生"1+1>N"的超线性增益
4. 能量约束 P < P_max 是涌现发生的基础物理约束

#### SDI晶上系统工程路径

> 软件定义互连（SDI）化合键 = LINK原语的物理实现
> SDSoW（软件定义晶上系统）= TCC的物理载体
> 晶圆级SDI交换Mesh = NPC原语的硬件架构

#### 综合射频处理平台产品定义
