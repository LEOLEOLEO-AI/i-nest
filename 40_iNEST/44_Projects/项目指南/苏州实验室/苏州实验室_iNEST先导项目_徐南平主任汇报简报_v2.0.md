# 第一部分：正式项目指南

## 一、项目定位：为什么苏州实验室必须现在启动？

### 1.1 全球AI算力面临范式级拐点

当前AI产业的核心矛盾：

| 瓶颈 | 现状 | 数据来源 |
|------|------|---------|
| **散热极限** | 单芯片功耗从H100的700W飙升至B200的1000W+，液冷散热逼近物理极限 | NVIDIA官方TDP数据，2024-2025 |
| **互联瓶颈** | IBM带宽增速滞后于模型参数增速 | IBM HFI报告，2025 |
| **规模回报递减** | GPT-5训练成本超10亿美元，性能提升曲线趋缓 | Epoch AI估算，2025 |
| **冯·诺依曼瓶颈** | 存算分离导致90%+能耗消耗在数据搬运 | Horowitz ISSCC 2014 |

**结论**：单纯依靠scale-up和scale-out已逼近工程极限。

### 1.2 TCC范式：从算力中心到拓扑中心

邬江兴院士团队提出的TCC（拓扑中心计算）范式，核心三个颠覆：

1. **Route-Transform同构**：通信≡计算，网络连接本身就是计算资源
2. **SDI化合键**：软件定义互连四原语(FUSE+MAPS+GEMM+LINK)，网络拓扑可编程
3. **CST理论度量**：协同时空复杂度(CST)是智能涌现的条件阈值检测器，非连续智力刻度尺

> **关键声明**：CST公式不包含「连接数」变量；键规模与CST的函数关系是待验证的核心工程假设。

### 1.3 苏州实验室的不可替代战略位置

| 层级 | 内容 | 承担单位 | 苏州角色 |
|------|------|---------|---------|
| L5 | 六级涌现验证 | 七单位协同 | 参与 |
| L4 | SDSoW晶圆级系统集成 | **苏州(童祎)+天大** | **共同牵头** |
| L3 | 涌现架构/TCC-11工具链 | 复旦+浙大+NDSC | 适配 |
| L2 | 非线性器件/忆阻器阵列 | 北大+天大 | 测试协同 |
| **L1** | **SDI化合键/TCC-Link材料** | **苏州实验室** | **唯一承担** |

**一句话定位**：没有材料层的突破，上面所有层都是空中楼阁。

## 三、四年级技术路线

### 核心创新：器件哲学纠正

| 维度 | 传统精确计算 | iNEST涌现计算 |
|------|------------|-------------|
| 器件目标 | 阻值一致性(6-8bit) | ON/OFF开关(1bit) |
| 主要矛盾 | 器件一致性 | 非线性表征与网络动力学的映射 |
| 工艺要求 | 7nm/5nm先进工艺 | 28nm/12nm成熟工艺 |
| 工作主频 | GHz级高频 | MHz级(匹配生物时间尺度) |
| 器件变异性 | Bug | Feature(涌现驱动力) |

### 四年级里程碑

**Y1 (2026Q4-2027Q3) 材料→器件**：SDI化合键材料配方(5轮MPW) | 忆阻器I-V表征(α=2.8-4.5) → M1: 单节点ON/OFF验证

**Y2 (2027Q4-2028Q3) 器件→阵列**：128×128交叉阵列 | SDI交换矩阵FPGA IP | 北大/天大工艺固化 → M2: 阵列级映射验证

**Y3 (2028Q4-2029Q3) 阵列→系统**：SDSoW原型(>10K节点) | II-III级涌现验证 | CST-材料反向映射模型 → M3: 物理涌现验证

**Y4 (2029Q4-2030Q3) 系统→涌现**：万节点SDSoW | IV-V级涌现验证 | iNEST开源社区+TCC-Link标准 → M4: 四级涌现验证

# 第二部分：战略说帖

## 六、iNEST 的战略意义——物理 AGI 的中国路线

### 6.1 两条 AGI 路线的历史性分野

| 路线 | 代表 | 瓶颈 | 对中国的约束 |
|------|------|------|------------|
| **数字 AGI**（大模型 Scaling Law） | OpenAI、Google | 功耗墙、GPU 禁运、先进工艺受限 | 追赶难度大，受制于人 |
| **物理 AGI**（类脑涌现计算） | Intel Loihi、iNEST(中国) | 器件-算法-架构协同门槛极高 | **无禁运约束，可自主定义** |

iNEST 是物理 AGI 的「中国路线」——从第一性原理出发，不走模仿硅谷之路，开创基于**网络拓扑涌现智能**的新范式。

### 6.2 四个「唯一」——iNEST 的战略独特性

| 唯一性 | 内容 |
|--------|------|
| **唯一具备完整理论底座的物理智能路线** | CST 理论 + 六级自然常数阈值 + 40 系统跨物种检验(ρ=0.92)，国际上无对标 |
| **唯一覆盖材料→器件→架构→系统→涌现的全链条团队** | 七单位五层分工，国内无第二支团队具备此完整能力 |
| **唯一能开创「材料定义智能」新科研范式** | 材料的非线性动力学直接参与智能涌现，材料科学 × 网络科学 × 智能科学深度交叉 |
| **唯一不受先进工艺封锁的 AGI 路径** | 最高 12nm 工艺足矣，28nm 国产产线即可覆盖前两代，设备和材料均可国产替代 |

### 6.3 新材料科研范式：从「材料被选择」到「材料定义智能」

传统范式：材料被选择来满足计算需求（选更好的硅做更快的晶体管）。

iNEST 范式：**材料的内在非线性动力学直接参与智能涌现**——器件的氧空位迁移多时间尺度（ms→s→min→h）恰好对应神经动力学 STDP→FEP→BCM→SOC 的四级时序。材料不再是「被选择」的对象，而是「共同定义」智能属性的主动参与者。

这是材料科学、网络科学、智能科学的深度交叉，是苏州实验室可以定义的新科研范式。

### 7.4 新型举国体制的试验田

本项目天然具备「新型举国体制」的所有要素：

- **核心科学问题明确**：网络拓扑涌现智能的条件与机制
- **全链条覆盖**：材料(苏州)→器件(北大/天大)→架构(复旦/浙大/NDSC)→系统(苏州)→验证(七单位)
- **基础研究与应用落地并行**：4 年内从材料到系统全打通，从论文到芯片全产出
- **开放生态**：iNEST 开源工具链、TCC-Link 开放标准、共享验证数据集
- **多头经费整合**：实验室先导经费 + 科技委专项 + NSFC 重大计划 + 省市配套

这正是国家科技体制改革所期望的科研组织模式创新样板。
