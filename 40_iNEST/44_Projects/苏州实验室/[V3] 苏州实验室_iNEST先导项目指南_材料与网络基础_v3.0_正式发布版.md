# 苏州实验室 iNEST先导项目指南

# 从关键材料到智能涌现网络的全链条创新

> **申报单位**：iNEST联合团队 / **经费**：2亿元 / **周期**：4年 / **六单位联合**：苏州实验室(牵头, 项目执行期间招聘30-50人团队)+北大+天大+复旦+浙大+NDSC
>
> **核心定位**：为TCC拓扑中心计算范式和iNEST智能涌现理论提供**从材料到网络的全链条基础支撑**。

---

## 摘要

本项目以苏州实验室在功能氧化物、忆阻器材料、晶圆级工艺方面的核心能力为基础，为TCC-11原语（FUSE/MAPS/GEMM/LINK）、TCC-Link标准和iNEST自演化网络提供从材料到网络的系统工程实现。

**核心逻辑链**： 苏州实验室材料 → TCC-11原语物理实现 → SDI化合键（TCC硬件载体） → iNEST自演化网络（CST涌现） → 智涌脑产品（五代矩阵）

**关键创新——器件哲学纠正**：本项目的忆阻器需求与传统精确计算范式根本不同。不追求器件一致性——利用器件的**非线性动力学**作为网络涌现的物理驱动力。器件变异从Bug变为Feature。

**已获支撑**：CST V25理论40系统跨物种/跨架构自洽性检验（ρ=0.92, τ=0.83）。SDI六级智能跃迁全链路仿真（V9→V32→L4→L5→L6）。三项物理突破。**北京大学杨玉超团队双铁电栅极可调忆阻器（InfoMat 2026）提供TCC-11原语非线性物理加速；浙江大学潘纲团队Darwin3类脑芯片（NSR 2025）提供iNEST工具链与片上学习参考架构。**
**团队规模**：苏州实验室项目执行期间招聘30-50人（材料15-20+器件10-15+集成10-15），六单位合计博后6+博12+硕20+工程师30-50人。
**同步建设**：iNEST研发工具生态（仿真器+编译器+调试器+基准测试集），面向六单位联合研发的统一开发环境。

### 1.2 器件哲学：非线性动态映射——与精确计算的根本分野

本项目对忆阻器的需求与传统路线**方向性不同**：

| 维度 | 传统忆阻器精确计算 | iNEST非线性动态映射 |
|------|-----------------|------------------|
| 忆阻器角色 | 精确模拟存储器（阻值=权重） | **非线性动力学单元** |
| 追求目标 | 更高bit精度（6→8→16 bit） | **更丰富的动力学特征** |
| 对待变异 | 误差是敌人（需校准/纠错） | **非线性是资源**（多样性=探索能力） |
| 器件一致性 | CV<2% | **CV<20%即可** |
| 写入耐久 | >10¹² | **>10⁶-10⁷**（STDP频率） |
| 工作主频 | >1GHz | **<100MHz**（事件驱动，脉冲稀疏） |
| 最优工艺 | 先进制程 | **28nm互连最优 / 12nm天花板** |

**核心纠正**：传统路线试图用忆阻器做"比GPU更精确的MAC"——方向错误。iNEST将忆阻器非线性从"障碍"转化为"涌现驱动力"——器件的I-V滞回、氧空位随机迁移、导电细丝可塑性，正是网络涌现所需的物理动力学基底。

### 1.3 12nm工艺天花板：从第一性原理的证明

互连违反Dennard缩放定律——先进制程的金属互连线是受害者：

| 工艺节点 | M6线宽 | RC延迟 | 速率/线 | 带宽密度 |
|---------|-------|--------|--------|---------|
| **28nm** | 100nm | **3.8 ps** | **83 Gbps** | **52 TB/s/mm** |
| 7nm | 28nm | 97.8 ps | 3.3 Gbps | 7.3 TB/s/mm |
| 5nm | 20nm | 223.7 ps | 1.4 Gbps | 4.5 TB/s/mm |
| 3nm | 12nm | 798.9 ps | 0.4 Gbps | 2.1 TB/s/mm |

（数据来源：SDI互连工艺技术方案，经独立Elmore模型验算通过）

**28nm M6层互连性能是5nm的59倍。12nm是天成板——足以覆盖从感知到通用智能的全部键规模需求。iNEST是事件驱动架构（主频<100MHz），算力瓶颈在互连不在计算。**

### 1.4 三个材料层面的根源性瓶颈

| 瓶颈 | 传统方案的错误 | 苏州实验室攻关方向 |
|------|-------------|------------------|
| TCC原语缺乏物理载体 | 试图做精确MAC，方向错误 | **SDI化合键功能材料**：利用阈值开关做ON/OFF |
| TCC-Link物理层材料受限 | 追逐先进制程，互连反而退化 | **28nm M6层互连介质**：低损耗+高密度RDL |
| iNEST缺乏多时间尺度动力学 | 用CMOS单一时钟模拟 | **忆阻器非线性动力学映射**：氧空位迁移的多时间尺度 |

## 三、技术路线：五层全链条架构

```
材料(L1) → 器件(L2) → 架构(L3) → 系统(L4) → 智能(L5)

L5 智能涌现: CST测量 → 相变检测 → 六级等级判定（Ⅰ-Ⅲ级物理验证）
L4 系统集成: SDSoW晶圆级 + 3D堆叠 + 供电散热（苏州实验室+天大）
L3 架构设计与工具链: TCC-11原语 + iNEST四规则引擎 + SNN工具链 + TCC-Link标准（复旦+浙大潘纲+NDSC）
L2 器件制备: SDI化合键单元 + 交叉阵列(128→512) + 突触-神经元一体器件
L1 材料创新: 功能氧化物薄膜 + 电极/界面 + 低损耗互连介质 + 键合材料
```
