# Idea: ANN 能否突破 Γ_st = 0 的临界天花板？

- **类型**: 理论猜想
- **时间**: 2026-05-30
- **标签**: criticality, FEP, Gamma_st, ANN, emergence
- **关联**: [[FEP-STDP_收敛性证明]], [[CST_Theory_V27_FINAL]]

---

## 一、问题

iNEST FEP-STDP 框架中，Γ_st 是稳态自由能曲率。当 Γ_st → 0 时，系统进入**亚临界 → 临界 → 超临界**相变区。

**核心问题**：ANN（人工神经网络）能否被驱动到 Γ_st = 0 的临界态？如果能，是否意味着 ANN 也可以涌现出生物网络才具备的临界信息处理能力？

## 二、假说

生物皮层网络的 SOC 不是"恰好出现"的——它是 FEP 最小化 + STDP 局部学习的必然收敛点。

如果 ANN 采用相同的 FEP-STDP 驱动机制，理论上应该趋近同样的 Γ_st = 0 吸引子。

**区别不在于"能不能"，而在于"多快"**：
- 生物网络：亿万年演化优化了拓扑，天然趋近临界
- ANN：拓扑是人工设计的，需要更长时间/更强驱动才能到达

## 三、实验设计

| 条件 | 拓扑 | 学习规则 | 预期 Γ_st |
|------|------|----------|-----------|
| 对照组 | 随机 ER 图 | SGD + 反向传播 | > 0.5（不趋近临界） |
| 实验组 A | WS 小世界 | STDP + BCM | 0.1–0.3 |
| 实验组 B | SF+SW 混合 | FEP-STDP | **→ 0** |

**关键指标**：Γ_st 的时间演化曲线 + avalanche 尺寸分布的幂律指数。

## 四、与 iNEST 的核心关联

如果实验组 B 确实趋近 Γ_st = 0，就证明了 iNEST 的核心主张：**涌现智能不依赖于生物基底的"特殊性"，而依赖于驱动机制的"普适性"**——FEP-STDP 是跨基底的涌现驱动引擎。
