> **总纲设计原则**：
>
> - **由浅到深**：从高中/大学本科可理解的基础概念出发，逐步推进到研究前沿
> - **由简到繁**：单一学科概念 → 跨学科交叉 → 系统集成 → 原创理论
> - **螺旋上升**：每个专辑内部递进，专辑之间也递进，后续专辑回溯引用前置概念
> - **双轨并行**：每个选题同时覆盖"理论本体"和"CST/iNEST视角解读"
> - **五年节奏**：Year 1 打地基，Year 2 建框架，Year 3 跨学科桥接，Year 4 系统集成，Year 5 原创理论与工程实践

## 专辑A：数学基石

### ——“通往涌现智能的数学语言”

> **定位**：Year 1 · 面向有理工科基础的受众 · 每集30-45分钟  
> **目标**：建立后续所有专辑所需的数学直觉和形式化工具


| 编号  | 选题                         | 核心内容                                | 与CST/iNEST的联系                               |
| --- | -------------------------- | ----------------------------------- | ------------------------------------------- |
| A01 | **概率论基础：从频率到测度**           | 样本空间、σ-代数、概率测度、条件概率、贝叶斯定理           | 智能系统必须在不确定环境中推理，概率是第一语言                     |
| A02 | **贝叶斯推断：从先验到后验**           | 贝叶斯公式、共轭先验、MAP估计、贝叶斯网络              | 大脑作为贝叶斯推理机（Helmholtz/Friston），物理网络如何实现贝叶斯更新 |
| A03 | **随机过程与马尔可夫链**             | 马尔可夫性、转移矩阵、平稳分布、遍历性、随机游走            | 网络上的信息扩散、PageRank与节点重要性、马尔可夫链蒙特卡洛           |
| A04 | **线性代数：矩阵、特征值与谱**          | 特征分解、奇异值分解、正定矩阵、矩阵指数                | 邻接矩阵谱与网络性质的深刻联系（谱图论）                        |
| A05 | **图论基础：从欧拉到现代**            | 图的基本概念、树、连通性、路径、着色、匹配、平面图           | 所有网络分析的语言基础                                 |
| A06 | **谱图论：从拉普拉斯矩阵到Cheeger不等式** | 图拉普拉斯、Fiedler向量、谱聚类、等周不等式、Cheeger常数 | 网络分区质量的数学度量，对分带宽的理论上界                       |
| A07 | **代数拓扑入门：从欧拉示性数到同调群**      | 单纯复形、Betti数、同调群、欧拉示性数、持久同调          | 网络拓扑的"形状"不变量，高阶连接模式的数学刻画                    |
| A08 | **微分方程与动力系统基础**            | 常微分方程、相空间、不动点、稳定性分析、分岔理论            | 网络动力学的数学框架，涌现行为的动力系统描述                      |
| A09 | **变分法与泛函分析基础**             | 泛函、变分原理、欧拉-拉格朗日方程、泛函极值              | 最小自由能、最小作用量的数学基础                            |
| A10 | **优化理论：从梯度下降到凸优化**         | 凸优化、对偶理论、KKT条件、拉格朗日乘子法、Pareto最优     | 多目标权衡（性能-功耗-面积）、网络架构优化                      |
| A11 | **帕累托最优与多目标优化**            | Pareto前沿、支配关系、NSGA-II、多目标进化算法       | 晶上系统设计中的时延-带宽-功耗-面积四维Pareto权衡               |
| A12 | **测度论与积分**                 | Lebesgue测度、可测函数、Lp空间、Fubini定理       | 信息论与概率论的严格数学基础                              |
| A13 | **微分几何入门：流形与度量**           | 流形、切空间、黎曼度量、测地线、曲率                  | 信息几何的数学基础，概率分布空间的几何结构                       |
| A14 | **信息几何：Fisher信息与自然梯度**     | Fisher信息矩阵、统计流形、α-联络、自然梯度下降         | KL散度的几何意义，PPO/TRPO策略优化的几何本质                 |
| A15 | **范畴论初步：结构的数学**            | 范畴、函子、自然变换、万有性质                     | 不同层次涌现之间的形式化映射，跨尺度类比的数学基础                   |
| A16 | **递归、自指与哥德尔不完备定理**         | 递归函数、对角化论证、哥德尔编码、不完备定理、停机问题         | 自指系统的本质限制，智能系统的自我建模能力边界                     |
| A17 | **分形几何：自相似性的数学**           | 分形维数、Hausdorff维数、迭代函数系统、Mandelbrot集 | 大脑皮层的分形结构、网络的自相似性、多尺度复杂度                    |
| A18 | **张量与多线性代数**               | 张量积、张量分解（CP/Tucker）、张量网络            | 高阶网络交互的数学表示，量子计算与张量网络的联系                    |

## 专辑B：物理原理

### ——“从最小作用量到涌现的物理根基”

> **定位**：Year 1-2 · 面向有大学物理基础的受众  
> **目标**：建立"物理网络涌现智能"中"物理"二字的理论内涵


| 编号  | 选题                    | 核心内容                                     | 与CST/iNEST的联系                           |
| --- | --------------------- | ---------------------------------------- | --------------------------------------- |
| B01 | **最小作用量原理：物理学的第一性原理** | 拉格朗日量、作用量泛函、哈密顿原理、诺特定理                   | 智能系统的行为能否用变分原理统一描述？最小自由能是最小作用量在认知科学中的对应 |
| B02 | **哈密顿力学与相空间**         | 哈密顿量、正则方程、相空间流、刘维尔定理、辛几何                 | 网络状态演化的相空间描述，信息守恒与可逆计算                  |
| B03 | **统计力学基础：从微观到宏观**     | 微观态与宏观态、配分函数、玻尔兹曼分布、自由能                  | 物理网络的统计力学描述，节点状态的配分函数                   |
| B04 | **熵：从热力学到信息论的统一**     | 热力学熵、玻尔兹曼熵、Gibbs熵、Shannon熵、最大熵原理         | 熵作为"复杂度"和"不确定性"的统一度量                    |
| B05 | **自由能与最小自由能原理**       | Helmholtz自由能、Gibbs自由能、变分自由能、Friston自由能原理 | 物理网络通过最小化自由能实现自组织与适应                    |
| B06 | **相变与临界现象**           | 一阶/二阶相变、序参量、临界指数、标度律、普适性                 | 智能涌现作为一种相变——临界耦合阈值的物理意义                 |
| B07 | **重整化群：跨尺度的物理学**      | Wilson重整化群、块自旋变换、不动点、相关长度发散              | 深度学习与重整化群的类比，多尺度网络结构的物理根源               |
| B08 | **伊辛模型与自旋玻璃**         | 伊辛模型、Curie-Weiss平均场、自旋玻璃、挫折与能量景观         | Hopfield网络的物理基础（2024诺贝尔物理学奖）            |
| B09 | **模拟退火：从冶金到优化**       | Metropolis算法、退火调度、温度参数、收敛性定理             | 网络参数搜索的物理启发优化，避免局部最优的热力学策略              |
| B10 | **耗散结构与非平衡统计力学**      | Prigogine耗散结构、熵产生、最小熵产生原理、非平衡态           | 智能系统作为远离平衡态的耗散结构，能量流驱动自组织               |
| B11 | **临界态与自组织临界（SOC）**    | Bak沙堆模型、幂律分布、1/f噪声、雪崩动力学                 | 大脑的自组织临界态，物理网络接近临界点时涌现最大化               |
| B12 | **随机热力学与信息热力学**       | Landauer原理、Jarzynski等式、涨落定理、信息引擎         | 计算的物理极限，物理网络的能量-信息权衡                    |
| B13 | **场论入门：从经典场到量子场**     | 经典场、拉格朗日密度、对称性与守恒律、路径积分                  | 认知场论（Chae 2026论文的理论框架），网络动力学的场论描述       |
| B14 | **拓扑物理与拓扑保护**         | 拓扑绝缘体、Berry相、陈数、拓扑保护边缘态                  | 网络拓扑保护机制——某些通信模式对局部扰动的鲁棒性               |

| D11 | **源编码与数据压缩**            | 霍夫曼编码、算术编码、率失真理论、向量量化                      | 感知系统的信息压缩机制，物理网络的表征效率               |
