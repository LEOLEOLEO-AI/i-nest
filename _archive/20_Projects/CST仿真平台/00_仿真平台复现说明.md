## 常见问题

### 1. 依赖安装失败

- 优先确认 Python 版本与包安装权限
- 如果系统限制 `pip.exe`，使用 `python -m pip install ...`

### 2. 图片未生成

- 检查是否在正确目录执行
- 检查脚本是否有写权限
- 检查 `matplotlib` 是否正常安装

### 3. 数据读取失败

- 确认 `NeuronConnect.xls` 与 `connectome_v8_data.json` 在当前目录
- 优先使用已整理好的 `connectome_v8_data.json`
