## 核心论点

后摩尔时代，单芯片晶体管密度提升趋缓，算力突破的核心矛盾从器件密度转向**互连拓扑与器件多态性**。当前AI芯片片内互连（2D Mesh/Ring）拓扑单一，多态非易失器件（忆阻器）的工程化集成尚不成熟——这两个方向的突破将成为下一代AI芯片的核心差异化竞争点。

## 项目一（核心）

### 立项名称
**《面向AI加速器的小世界模块层级片上网络拓扑设计与关键技术》**

*英文：Small-World Hierarchical-Modular On-Chip Network Topology Design and Key Technologies for AI Accelerators*

### 技术背景
全球AI芯片片内互连均采用规整低复杂拓扑：
- NVIDIA NVLink：Ring拓扑，All-Reduce效率受限
- Tenstorrent Wormhole：2D Mesh，平均跳数高
- Intel Gaudi：Torus，扩展效率随规模下降

以小世界和模块层级特性为核心的片上网络，可在相同面积功耗下显著降低平均通信路径长度，提升集群协同效率——**全球尚无AI芯片公司以此为核心设计哲学**。

### 研究内容
1. 建立以平均路径长度、聚类系数、模块度、层级深度为综合目标的NoC拓扑优化理论，推导满足物理布线约束的最优拓扑族
2. 研究兼顾面积、功耗、时序约束的复杂拓扑工程实现方法，设计可综合的通用可重构互连单元
3. 研究面向Chiplet异构集成的跨die复杂拓扑互连，支持Chiplet间的层级模块化路由
4. 完成测试芯片流片，与2D Mesh/Ring/Torus在LLM训练/推理负载下全面定量对比

### 关键指标
- 平均通信跳数：较2D Mesh降低≥40%
- AllReduce通信效率：提升≥50%
- 相同面积/功耗下等效算力：提升≥35%
- 支持规模：≥10⁶节点单片
- 流片验证：≥1颗测试芯片

**申报渠道：** 工信部"集成电路"重大专项 / 科技部前沿微纳电子重点专项  
**经费建议：** 2000万—3500万 / 3年  
**合作落地：** 海光信息（DCU）、飞腾处理器、国产AI互连IP公司

## 项目二

### 立项名称
**《多态非易失器件阵列的晶圆级CMOS兼容异质集成工艺与器件可靠性》**

*英文：Wafer-Scale CMOS-Compatible Heterogeneous Integration Process for Multi-State Nonvolatile Device Arrays and Device Reliability*

### 技术背景
HfOx/TaOx/PCM等多态非易失器件是突破内存墙的核心技术路径，但：
- 现有器件可区分状态数普遍≤8态（3 bits），距32态（5 bits）仍有差距
- CMOS后道集成良率：现有最好水平约70%（1024×1024阵列），距工程化需要的85%仍有差距
- 器件间均匀性（σ/μ）：现有约15%~20%，需降至10%以下

### 研究内容
1. 研究HfOx/TaOx/PCM器件的材料调控方法，实现可区分状态数从8态提升至≥32态
2. 研究CMOS后道兼容的晶圆级集成工艺，重点解决界面缺陷控制、热预算兼容、高密度互连问题
3. 研究器件参数分布（器件间差异、周期间涨落）的统计特性表征与在线校准方法
4. 建立器件可靠性模型：保持特性（10年@85°C）、耐久性（≥10⁹次循环）、抗辐射性能

### 关键指标
- 可区分电阻状态数：≥32态
- 器件间阈值电压均匀性（3σ）：≤10% Vset
- 阵列集成良率：≥85%（1024×1024）
- 耐久性：≥10⁹次读写循环
- 保持特性：10年@85°C，误码率≤10⁻⁶

**申报渠道：** 科技部前沿微纳电子重点专项 / 国家自然科学基金重点项目  
**经费建议：** 2000万—4000万 / 4年

## 项目三

### 立项名称
**《存算一体近存计算阵列的片内复杂拓扑互连设计与能效优化》**

*英文：In-Memory Computing Array with Complex In-Chip Interconnect Design and Energy Efficiency Optimization*

### 技术背景
AI推理过程中内存带宽墙导致GPU 70%以上时间等待数据搬运，多态非易失器件（忆阻器）近存计算（PIM）是突破路径，但现有PIM阵列内部互连仍采用简单总线/矩阵结构，无法充分利用器件并行性。将复杂网络拓扑引入PIM阵列内部互连，可进一步降低数据搬运能耗。

### 研究内容
1. 研究多态非易失器件PIM阵列内部互连的拓扑优化，最小化数据搬运路径长度
2. 研究基于多时间尺度原理的分层存储与计算协同调度（片上SRAM→PIM阵列→片外HBM的多级协同）
3. 研究PIM阵列与传统数字逻辑混合精度计算的接口设计
4. 流片验证，实测能效指标

### 关键指标
- 推理能效：较纯SRAM架构提升≥3倍
- 数据移动能耗占比：从70%降至≤40%
- 阵列内部计算吞吐量：≥1 TOPS/W
- 混合精度支持：INT4/INT8/FP16

**申报渠道：** 科技部前沿微纳电子专项 / 科技部先进计算重点专项（跨专项申报）  
**经费建议：** 2000万—3500万 / 3年

## 项目四（晶圆级旗舰）

### 立项名称
**《晶圆级复杂拓扑智能计算系统：超越均匀网格互连架构的下一代算力平台》**

*英文：Wafer-Scale Complex-Topology Intelligent Computing System: Next-Generation Computing Platform Beyond Uniform Grid Interconnect*

### 技术背景
Cerebras WSE-3证明晶圆级单片集成可行（900K核心、44GB SRAM、900TB/s带宽），但其2D Mesh拓扑规整，拓扑复杂性处于集体行为相变阈值以下。**将晶圆级规模与复杂网络拓扑结合，是超越Cerebras的下一代技术路线，全球空白。**

### 研究内容（三代路线）
**第一代（近期，3年）：**
- 实现10⁶连接规模的单片复杂拓扑互连，支持≥5种可重构拓扑
- 拓扑复杂性（聚类系数×平均路径长度倒数）较2D Mesh提升≥10倍
- 流片验证，LLM训练负载下算力效率对比测试

**第二代（中期，5年）：**
- 10⁸连接规模，Chiplet集成方案
- 跨Chiplet层级模块化拓扑，集群扩展效率≥0.90

**第三代（长期，7年）：**
- 10¹⁰连接规模，3D堆叠，集成物理自适应突触（忆阻器）
- 跨越初级集体行为相变临界点，神经雪崩幂律验证

### 关键指标（第一代）
- 互连规模：≥10⁶物理连接，单片晶圆
- 支持拓扑类型：≥5种（含小世界、层级模块化、稀疏随机等）
- 拓扑重构时间：≤1ms
- 单位能耗有效算力：较Cerebras WSE-3提升≥3倍
- 流片：≥1颗功能验证芯片

**申报渠道：** 科技部前沿微纳电子重点专项 / 国家重点研发计划先进计算专项  
**经费建议：** 5000万—8000万 / 4年（第一代）  
**合作落地：** 紫光晶圆级算力平台（首发验证）、中芯国际/华虹（工艺）
