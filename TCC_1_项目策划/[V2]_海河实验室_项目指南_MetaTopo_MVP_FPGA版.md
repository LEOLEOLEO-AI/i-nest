# 海河实验室项目指南（2026年度重大专项）
## 基于元拓扑-SDI化合键理论的介观尺度网络智能涌现架构与端侧FPGA原型验证

### 一、 项目定位与建设背景
当前人工智能与实时信号处理领域面临严重的“通信墙”与“功耗墙”瓶颈。传统的冯·诺依曼“节点中心”架构已无法满足端侧大模型及海量数据并发的高效处理需求。本项目旨在通过“拓扑中心计算（Topology-Centric Computing, TCC）”新范式，突破现有架构限制。

为在三年内实现核心理论的快速工程闭环，本项目**将研究重心明确收敛至“FPGA原型验证”阶段**。抛弃高风险、长周期的全定制芯片流片路线，以“元拓扑-SDI化合键”理论为指导，聚焦“5+4”通信与算子原语体系，研制**SDIO-N接口标准**与**端侧“3+1” MVP极简架构（NPC-AR + CPC-M + CPC-S + SDI-Ctrl）的FPGA物理原型**。通过FPGA板卡真实展现一套硬件在“AI推理、视频分析、波束成形”多场景下的微秒级拓扑热切换与极致能效，为下一代晶圆级集成与类脑计算奠定坚实的工程与生态基础。

### 二、 核心研究内容（三大课题）

本项目设立三个强关联课题，形成“理论标准 -> 硬件原型 -> 场景验证与生态”的完整技术闭环：

#### 课题一：元拓扑-SDI化合键理论体系与SDIO-N接口标准研制
- **核心理论深化**：建立元拓扑完备性定理证明，推导网络拓扑变分演化方程，证明“5+4”通信与算子原语对任意分布式计算的完备表达能力；揭示“FFT-AllReduce同构映射”规律。
- **自主标准制定**：定义涵盖物理层（L1）、链路层（L2）、轻量协议层（L3）与原语语义层（L4）的SDIO-N接口标准规范，打破国际互连壁垒。
- **仿真与AI辅助设计平台**：购置大模型（LLM）服务与云算力，搭建结合大模型辅助RTL代码生成与拓扑演化仿真的云端开发测试环境（SDI-Bond Simulator）。

#### 课题二：“3+1”极简架构IP设计与端侧FPGA原型研制
- **核心IP核开发**：基于Chisel/SystemVerilog完成核心模块RTL设计：NPC-AR（支持多节点硬件AllReduce融合）、CPC-M（INT8/FP16脉动阵列矩阵乘）、CPC-S（复用AllReduce连线的FFT蝴蝶计算）以及支持微秒级重构的SDI-Ctrl控制器。
- **端侧FPGA原型板卡研制**：以多颗中等规模FPGA（如Kria K26或VCU128）为基础，通过高速SerDes实现板间互连，搭建TCC-Edge端侧验证物理板卡。
- **设备与服务支撑**：租赁高端示波器、逻辑分析仪及高速测试探头等仪器设备，保障物理层信号完整性测试与IP核功能验证。

#### 课题三：端侧MVP多场景系统验证与开源生态建设
- **多场景MVP验证**：在同一套FPGA原型硬件上，通过SDI拓扑热重构，分别运行：
  1. **场景A（端侧AI）**：4路以上视频流目标检测推理及原生硬件AllReduce融合。
  2. **场景B（信号处理）**：16通道数字波束成形（DBF），验证FFT-AllReduce物理路径复用。
- **全栈开源生态布置**：开发并开源PyiNEST-Lite轻量级SDK（含Python API与底层C驱动）。租赁云服务器搭建自动化CI/CD持续集成流水线与开源代码托管社区，形成开发者生态。

### 三、 考核指标
1. **理论与标准指标**
   - 形成具有独立知识产权的SDIO-N接口标准（L1-L4）草案并发布。
   - 在顶级期刊/会议发表高水平论文≥5篇；申请核心发明专利≥10项。
2. **FPGA原型系统性能**
   - 研制出TCC-Edge端侧FPGA原型板卡（至少包含双FPGA节点）。
   - **能效与延迟**：端侧多路AI推理融合延迟<5μs（较纯软件方案提升10倍以上）；1024点FFT处理延迟<5μs。
   - **场景热切换**：硬件层面从AI推理模式切换至DBF模式的拓扑重构时间≤1μs。
   - **同构复用**：验证FFT与AllReduce复用物理路径，节省≥20%的FPGA逻辑资源。
3. **开源与平台建设**
   - 建立完备的开源工具平台与文档知识库，开源RTL代码行数≥15,000行。
   - 上线大模型辅助的AI硬件研发流（AI-assisted EDA）工作流。

### 四、 经费预算建议与资源配置
本项目总经费申请额度：**1000万 - 1500万元人民币**。资金实行“重研发、轻重资产”的倾斜策略：
1. **人员招聘与劳务费（重点保障，约35-40%）**：大规模招聘顶尖体系结构架构师、RTL设计工程师、cocotb验证工程师及底层软件开发人员，打造一支高精尖的攻坚团队。
2. **云服务与大模型租赁（约15-20%）**：用于购买顶级大语言模型API（支撑70%测试用例与部分RTL生成）、租赁高性能云服务器（部署CI/CD、代码托管社区与大并发仿真）。
3. **软硬件开发与原型制造（约20-25%）**：购买/定制FPGA评估板（如AMD/Xilinx高端系列），PCB制板贴片打样，外协加工服务等。
4. **仪器设备租赁服务（约10-15%）**：采取**“以租代买”**模式，租赁昂贵的高端混合信号示波器、频谱仪、高速误码仪等，避免一次性重资产投入，提高资金使用效率。
5. **合作交流与专利发表（约10%）**：开源社区运营维护、标准推介会议、论文版面费与专利申请代理费。

### 五、 申报要求
申报单位需在网络拓扑理论、FPGA数字系统设计、端侧AI部署及雷达信号处理有深厚交叉背景。具有大模型辅助芯片设计（AI4EDA）经验及成熟开源社区运作能力的团队优先支持。
