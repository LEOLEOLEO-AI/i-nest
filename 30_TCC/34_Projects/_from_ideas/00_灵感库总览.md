# 💡 灵感库 — Map of Content

> 所有未经筛选的想法、假设、创意、发现，先捕捉再评估。
> 每周五在周总结中统一回顾并决定哪些值得推进。

## 使用方式

1. **快速捕捉**：发消息给AI "记录灵感：[内容]"，自动生成到本目录
2. **定期回顾**：每周五看一遍本目录，标记值得推进的
3. **升级转化**：有价值的灵感 → 移入对应研究方向的01_论文或03_项目策划

## 灵感状态标签

- `#待评估` — 刚捕捉，未思考
- `#有潜力` — 初步评估值得推进
- `#进行中` — 已纳入正式工作计划
- `#搁置` — 暂时放下，保留备查
- `#否决` — 评估后放弃，记录原因

## 本库灵感列表

```dataview
TABLE date, tags
FROM "灵感库"
WHERE type = "idea"
SORT date DESC
```
