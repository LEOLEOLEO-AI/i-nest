### **🔍 项目简介**

**tiny-NPU**是一个**极简、可综合、面向教学**的神经网络处理器（NPU）RTL项目，采用SystemVerilog实现，旨在帮助用户从零理解LLM/Transformer硬件推理和ONNX模型加速的真实硬件实现。
- **核心目标**：超越框架使用，掌握底层硬件实现原理
- **作者**：harishsg99（2026年2月发布，更新活跃）
- **开源协议**：MIT
- **代码仓库**：https://github.com/harishsg993010/tiny-NPU

### **🚀 核心能力**

#### **(一) LLM模式（主推功能）**

直接运行真实HuggingFace权重，专注于Transformer架构推理：
- **支持模型**：GPT-2、LLaMA、Mistral、Qwen2（小参数版本）
- **关键特性**：
  - INT8量化 + 多头注意力机制 + KV Cache存储
  - 自研128位微码指令集控制Transformer块执行流程

#### **(二) Graph模式（ONNX支持）**

将ONNX模型编译为NPU指令，实现端到端推理：
- **支持算子**：GEMM、Conv、Softmax、LayerNorm、GELU、Pooling、Gather、Slice、Concat、Resize、Cast等
- **技术特点**：
  - INT8 + FP16混合精度计算
  - DMA与GEMM并行处理（异步调度），提升硬件利用率

### **🏗️ 硬件架构**

#### **(一) 脉动阵列计算单元**
- **核心配置**：16×16权重固定阵列（可参数化扩展至24×24）
- **计算能力**：256个MAC单元，支持INT8和FP16双精度
- **数据流向**：激活值向下流动，权重向右流动
- **矩阵处理**：自动分块处理大于16×16的矩阵运算

#### **(二) 内存层次结构**

| 内存层级 | 容量 | 主要功能 |
| :------- | :--- | :------- |
| **SRAM0** | 64KB | 主工作空间，存储权重和激活值 |
| **SRAM1** | 8KB | 辅助存储，存放LayerNorm参数、残差数据 |
| **外部DDR** | ~226KB | 存储全层权重数据 |

#### **(三) 关键引擎**
- **LLM模式（6个核心引擎）**：
  - GEMM：脉动阵列矩阵乘法
  - Softmax：3-pass LUT实现
  - LayerNorm/RMSNorm：层归一化处理
  - GELU/SiLU：激活函数计算
  - Vec引擎：向量操作
  - DMA：AXI4主控接口，负责DDR读写

- **Graph模式（额外7个引擎）**：
  - Reduce、Math、Gather、Slice、Concat、Pool、Cast等算子专用引擎

### **💻 ISA指令集架构**

#### **(一) LLM模式：128位微码ISA**
```
127        112 111         96 95          80 79          64
+-----------+--------------+--------------+--------------+
|    imm    |      K       |      N       |      M       |
+-----------+--------------+--------------+--------------+
63          48 47          32 31          16 15    8 7    0
+-----------+--------------+--------------+-------+------+
|   src1    |    src0      |     dst      | flags |opcode|
+-----------+--------------+--------------+-------+------+
```- **操作码（opcode）**：NOP、DMA_LOAD/STORE、GEMM、VEC、SOFTMAX、LAYERNORM、GELU、KV_APPEND/READ、BARRIER、END

#### **(二) Graph模式：128位图ISA**
```
127             96 95              64 63     48 47   40 39   32
+----------------+------------------+---------+------+------+
|     imm2       |       imm1       |  imm0   | src2 | src1 |
+----------------+------------------+---------+------+------+
31   24 23   16 15    8 7       0
+------+------+-------+---------+
| src0 |  dst | flags | opcode  |
+------+------+-------+---------+
```- **支持操作**：DMA、GEMM、元素操作、Softmax、Reduce、Math、Gather、Slice、Concat、Pool、Cast等

### **🛠️ 软件工具链**

#### **(一) ONNX编译器**

路径：python/onnx_compiler/compile.py
- **核心流程**：形状推断 → INT8量化 → DDR/SRAM分配 → 张量描述表生成 → 操作降级 → 黄金计算验证
- **输出文件**：program.bin、tdesc.bin、ddr_image.bin、golden.bin、manifest.json

#### **(二) Python辅助工具**
- 黄金模型测试（tests/test_end2end）
- 微码汇编器（ucode_asm.py）
- LUT生成器（make_lut.py）
- 多种ONNX模型生成脚本

### **✅ 验证与仿真**

#### **(一) 仿真环境**
- **工具**：Verilator周期精确仿真
- **测试目标**：21个仿真目标（控制测试、引擎测试、集成测试、推理演示等）
- **调试支持**：波形查看（VCD文件，gtkwave）

#### **(二) 验证策略**
- **follow-actual黄金比较**：
  1. NPU计算所有Transformer块
  2. 读取实际NPU输出
  3. 与C++黄金模型比较logits（要求位精确）

#### **(三) KV缓存优化**
- **Prefill阶段**：完整计算并缓存K/V
- **Decode阶段**：仅计算新token，读取缓存
- **效果**：速度提升1.8x，位精确无精度损失

### **📊 关键指标**
- **面积**：~61.5KB SRAM使用（64KB可用）
- **性能**：GPT-2推理约500万周期（5 tokens输入 + 10 tokens生成）
- **精度**：位精确（max_err=0）对比黄金模型
- **可综合性**：目标Xilinx FPGA，支持ASIC综合

### **📋 运行流程**

#### **(一) 环境依赖安装**
```bash
# Linux/WSL2 Ubuntu 22.04
sudo apt update && sudo apt install -y build-essential cmake python3 python3-pip verilator

# Python依赖（LLM推理）
pip3 install numpy transformers huggingface_hub safetensors

# Graph模式额外依赖
pip3 install onnx
```
#### **(二) 构建仿真环境**
```bash
cd /home/robot/digital-chip-design-agents/examples/tiny-NPU-main  

# 方式1：Makefile直接构建（推荐）
make sim  # 构建并运行基础仿真

# 方式2：CMake构建（生成21个仿真目标）
make cmake_sim
```
#### **(三) 运行测试/演示**
1. **LLM模式单元测试**（无需外部数据）
   ```bash
   cd build/cmake  # CMake构建输出目录
   ./npu_sim             # 控制平面冒烟测试
   ./engine_sim          # 引擎计算测试（GEMM/Softmax等6项）
   ./integration_sim     # 注意力头集成测试
   ./gpt2_block_sim      # 完整GPT-2 Transformer块测试
   ```

2. **GPT-2端到端推理**
   ```bash
   cd /home/robot/digital-chip-design-agents/examples/tiny-NPU-main
   # 下载并量化GPT-2权重（首次运行）
   python3 python/tools/export_gpt2_weights.py  # 导出HuggingFace GPT-2权重
   python3 python/tools/quantize_pack.py        # INT8量化并打包为weights.bin
   ./demo_infer  # 运行推理，输出生成文本
   ```

3. **Graph模式（ONNX模型）**
   ```bash
   # 生成测试ONNX模型（如MLP）
   python3 python/onnx_compiler/gen_mlp_onnx.py  
   # 编译ONNX到NPU可执行文件
   python3 python/onnx_compiler/compile.py --onnx mlp.onnx  
   # 运行ONNX仿真
   ./onnx_smoke_sim  # MLP模型测试
   ./onnx_cnn_smoke_sim  # CNN模型测试
   ```

4. **可选操作**
   ```bash
   make test   # Python黄金模型对齐测试
   make wave   # 生成VCD波形，用gtkwave查看
   make lint   # Verilator代码检查
   ```
