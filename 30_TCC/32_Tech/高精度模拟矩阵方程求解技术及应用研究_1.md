**【背景介绍】**

矩阵方程Ax=b
的求解是线性代数及其在信号处理、科学计算和神经网络训练等领域中的核心问题。传统数字计算在高精度矩阵运算中面临复杂度高、能耗大及冯·诺依曼架构“存储墙”瓶颈等限制。模拟矩阵计算（AMC）利用阻变存储阵列将矩阵元素映射为电导值，可在物理层面实现并行矩阵–向量乘法，并通过闭环反馈电路一步完成矩阵求逆，显著提升计算效率和能效。然而，该方法受限于低精度与电路扩展性不足。尽管混合模拟–数字设计通过结合低精度模拟运算与数字迭代精化可部分提升精度，但仍依赖频繁的模数转换，难以充分发挥模拟计算优势。未来的发展方向在于实现可规模化、高精度的一步模拟求逆电路，以突破现有阻变阵列精度与制造工艺的限制。

基于此，北京大学黄如院士、孙仲研究员、蔡一茂教授和王宗巍教授提出了一种基于全模拟矩阵运算的高精度矩阵方程求解方案。该方案在保持模拟矩阵计算（AMC）低复杂度特性的同时，通过低精度模拟求逆（LP-INV）与高精度矩阵–向量乘法（HP-MVM）的迭代耦合，实现了高精度与高效率的兼顾。LP-INV 可在每次迭代中快速生成近似解，从而有效减少迭代次数；HP-MVM 则利用位切分（bit-slicing）技术提升计算分辨率，确保结果精度。采用 BlockAMC 算法，通过块矩阵分解实现了中大规模（16×16）矩阵方程的高精度求解。整个系统基于高性能 1T1R 结构的 RRAM 芯片构建，每个存储单元具备八级可编程电导状态，并采用 40 nm CMOS 工艺制备完成。该硬件平台实现了高精度模拟矩阵运算的可靠映射，结果充分验证了该模拟计算体系在计算精度、能效表现及体系可扩展性方面的卓越性能与工程可行性。该文章以“Precise and scalable analogue matrix equation solving using resistive random-access memory chips”为题发表在国际顶级期刊Nature Electronics上。

系统的瞬态响应与性能基准对比：图5通过时间响应与算力基准测试，对比了模拟矩阵计算（AMC）与主流数字处理器的性能。首先，瞬态测试结果显示：LP-INV电路在约120

**【文献总结】**

本文提出了一种高精度且可扩展的模拟矩阵方程求解器。该方案利用适用于阻变存储计算（RRAM）的低精度矩阵运算，在基于40
nm工艺制备的1T1R结构RRAM阵列上实现 3 bit
分辨率，并通过位切分技术（bit-slicing）确保整体计算精度。为提升规模可扩展性，研究采用并实验验证了BlockAMC算法，实现了16×16矩阵求逆并达到24位定点精度。该模拟求解器还成功应用于大规模MIMO信号检测，在128×8、256-QAM调制系统中，仅经三次迭代即可获得与32位浮点数字处理器相同的误码率性能，展现出模拟计算在高精度与高能效信号处理中的巨大潜力。
