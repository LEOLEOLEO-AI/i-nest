## 一、术语体系

| 术语 | 全称 | 角色定位 |
|------|------|---------|
| **SDI** | Software-Defined Interconnect | 软件定义互连节点，网络拓扑控制层，化合键载体 |
| **SYN** | Synaptic Neuron | 突触神经元节点，计算层，忆阻器存算一体物理实现 |
| **MT** | Meta-Topology | 元拓扑，由多个SDI节点按化合键规则组成的功能模块 |

**设计哲学**：SDI是"碳骨架"，SYN是"功能基团"。网络的复杂性和智能性来自SDI之间化合键形成的拓扑，而非SYN节点本身的计算强度。

## 二、SDI节点接口规范：(4类型 × N条) + 2路 设计

### 2.1 设计层次说明

SDI节点的接口分为**三个层次**，必须严格区分：

| 层次 | 描述 | 性质 | 上限 |
|------|------|------|------|
| **类型层** | 4种化合键类型（E-L/I-L/E-S/I-S） | 离散，固定4种 | 4种 |
| **数量层** | 每种类型的物理连线数 Nᵢ | 动态变量，STDP驱动 | N_max（工程设计量） |
| **物理层** | 实际芯片互连总扇出 K = ΣNᵢ | 工程量，可扩展 | Gen1≈100，Gen5≈10000 |

> **关键修正**：P1-P4不是"4条物理线"，而是"4个类型通道"。
> 碳的4价类比仅适用于**键的类型分类**，不适用于键的数量上限。
> 人脑神经元平均扇出7000个突触，SDI节点扇出同样是工程变量，不受"4价"限制。

### 2.2 P1-P4：SDI-SDI化合键类型通道（4类）

```
              SYN-A
               ↕ Q1（主路，双向）
               ↕ Q2（冗余路，双向）
  ┌──[E-L × N₁条]──┐
  ├──[I-L × N₂条]──┤  SDI节点  ├──[E-S × N₃条]──
  └──[I-S × N₄条]──┘
  
  总扇出 K = N₁ + N₂ + N₃ + N₄
```

| 通道 | 键型 | 极性 | 稳定性 | 活化能 | 物理连线数 | 功能 |
|------|------|------|--------|--------|-----------|------|
| **P1** | E-L | 兴奋（+） | 长时程 | 高Ea_L | N₁（动态） | 固化骨架，对应长期记忆 |
| **P2** | I-L | 抑制（−） | 长时程 | 高Ea_L | N₂（动态） | 侧抑制骨架，维持稀疏 |
| **P3** | E-S | 兴奋（+） | 短时程 | 低Ea_S | N₃（动态） | STDP学习通道，候选固化 |
| **P4** | I-S | 抑制（−） | 短时程 | 低Ea_S | N₄（动态） | 修剪通道，候选断裂 |

### 2.3 Nᵢ动态调节规则（轴突侧支发芽机制）

每个类型通道的物理连线数由以下规则动态调整，对应生物中的**轴突侧支发芽（axonal sprouting）**：

```
N₁(t) = round( Σw_EL / w_threshold )      # E-L：总权重超阈值时分裂出更多物理线
N₂(t) = round( N₁(t) / EI_ratio )         # I-L：始终保持 N₁:N₂ = EI_ratio = 4:1
N₃(t) = round( 活跃STDP事件数 × k_branch ) # E-S：高频激活时自动增加分支
N₄(t) = round( N₃(t) / EI_ratio )         # I-S：始终保持 N₃:N₄ = 4:1
```

**E:I比约束**从"接口数比"转为"物理连线数比"：
```
N₁ : N₂ = 4 : 1    （E-L : I-L）
N₃ : N₄ = 4 : 1    （E-S : I-S）
```

### 2.4 扇出路线图（对应六代工程）

| 代次 | 年份 | 目标K（总扇出） | 类比生物 |
|------|------|--------------|---------|
| Gen1 | 2027 | K ≈ 100 | 线虫（C. elegans，avg k=16） |
| Gen2 | 2029 | K ≈ 1,000 | 章鱼神经节 |
| Gen3 | 2031 | K ≈ 10,000 | 皮层锥体细胞（avg 7000突触） |
| Gen4 | 2033 | K ≈ 100,000 | 小脑浦肯野细胞（~200,000突触） |
| Gen5 | 2035 | K ≈ 1,000,000 | 人脑整体连接度 |

### 2.5 Q1-Q2：SDI-SYN异质接口（2路）

| 接口 | 类型 | 方向 | 功能 |
|------|------|------|------|
| **Q1** | 主路 | 双向 | 接收/发送SYN激活信号（常规工作路） |
| **Q2** | 冗余备路 | 双向 | 容错（Q1故障时接管）或接入第二类SYN单元 |

**Q接口支持的SYN类型**（软件定义，协议统一）：
- 忆阻器SYN：模拟突触+神经元，欧姆定律天然实现存算一体（主力）
- 数字逻辑SYN：RISC-V核/DSP，处理规则逻辑任务
- 光子SYN：Gen4路线图引入，高速长程信号处理
- 未来异质SYN：接口协议不变，类型软件定义扩展

## 三、化合键四种类型完整定义

### 3.1 E-L键（兴奋性长时程键）

**物理定义**：  
忆阻器电导值已收敛到低阻稳态（G_max附近），LTP事件累积次数≥θ_LTP后固化。活化能Ea_L高，单次STDP事件无法改变。

**数学表达**：
```
Ea_EL = Ea_max × (1 - exp(-n_LTP / θ_LTP))
当 n_LTP ≥ θ_LTP 时：键型 E-S → E-L（固化）
```

**在网络中的作用**：构成功能模块的骨架，维持学习结果，对应长时记忆。

**层级分布**：在L1（Hub层）和L2（区域协调层）中比例较高。

---

### 3.2 I-L键（抑制性长时程键）

**物理定义**：  
抑制性突触固化版本，对应皮层侧抑制环路的稳定结构。忆阻器配置为抑制性模式（权重为负），同样经历LTP固化过程。

**关键功能**：  
- 维持每个功能模块的稀疏激活（"赢者通吃"机制的物理基础）
- 保证层级汇聚时上层节点不被过度激活（防止癫痫样爆发）
- 在σ计算中贡献模块性（modularity Q值）

**工程约束**：每个SDI节点中I-L键（P2）数量与E-L键（P1）数量比≈1:4（皮层兴奋/抑制比4:1的类比）。

---

### 3.3 E-S键（兴奋性短时程键）

**物理定义**：  
忆阻器处于中间电导态，正在被STDP动态调制。活化能Ea_S低，对STDP事件敏感。

**状态机**：
```
E-S + 累积n_LTP次LTP事件 → E-L（固化，变为骨架）
E-S + 累积n_LTD次LTD事件 → I-S（极性翻转，变为抑制）
E-S + 长期不活跃(>T_silent) → 断开（连接消失）
断开 + 局部FEP梯度>Ea_S → E-S（建立新连接）
```

**工程意义**：这是STDP学习发生的主要场所，也是新连接建立的入口。

---

### 3.4 I-S键（抑制性短时程键）

**物理定义**：  
正在减弱的抑制连接，或从兴奋性连接极性翻转过来的过渡态。

**状态机**：
```
I-S + 累积n_LTD次LTD事件 → 断开（连接消失）
I-S + 局部抑制不足信号 → I-L（固化为抑制骨架）
I-S + STDP翻转 → E-S（极性再翻转）
```

**工程意义**：这是网络"修剪"冗余连接的主要机制，类比突触消除（synaptic pruning）。

## 四、元拓扑（MT）三种基础类型

类比碳的sp³/sp²/sp¹杂化，三种SDI化合键组合方式对应三种元拓扑：

### MT-α：全骨架型（类比碳sp³）

```
结构：4个E-L键全部使用，形成三维稳定模块
         SYN
          ↕Q1
SDI-A ─E-L─ SDI-B
  |              |
 E-L           E-L
  |              |
SDI-D ─E-L─ SDI-C
```

**特性**：高聚类系数C，稳定，不易被STDP改变  
**对应层级**：L0（忆阻器阵列内）、稳定记忆模块  
**σ贡献**：高局部聚类，C↑

---

### MT-β：混合可塑型（类比碳sp²）

```
结构：2个E-L + 1个I-L + 1个E-S，兼顾稳定与可塑
```

**特性**：半稳定，可响应新输入调整结构  
**对应层级**：L1（Hub层），动态学习区  
**σ贡献**：中等聚类+部分长程连接

---

### MT-γ：全可塑链型（类比碳sp¹）

```
结构：E-S + I-S为主，形成线性快响应链路
SDI-A ─E-S─ SDI-B ─E-S─ SDI-C ...
```

**特性**：快速响应，低稳定性，高带宽信号传播  
**对应层级**：L2（全局协调层），稀疏长程连接  
**σ贡献**：短路径长度L↓（小世界网络关键贡献）

## 五、层级汇聚规则与临界态约束

### 5.1 四层层级结构

| 层级 | 符号 | 节点规模 | 主导键型 | SYN密度 | σ目标 |
|------|------|---------|---------|--------|-------|
| L0 | 微观层 | ~10 SDI | E-L为主（MT-α） | 高密 | σ₀（基准×1） |
| L1 | 局部Hub层 | ~100 SDI | E-L/E-S混合（MT-β） | 中密 | σ₁ = σ₀×1.5 |
| L2 | 区域协调层 | ~1000 SDI | E-S/I-L（MT-β/γ混合） | 低密 | σ₂ = σ₀×1.5² |
| L3 | 全局协调层 | ~10000 SDI | I-L/E-S（MT-γ为主） | 极稀 | σ₃ = σ₀×1.5³ |

**层级汇聚比**：每上升一级，节点数×10，SDI-SYN连接密度÷3，σ值×1.5。

### 5.2 临界态幂律约束（α→1.5）

临界态的必要条件：连接强度分布服从幂律：

```
P(w > x) ∝ x^(-α)，目标 α = 1.5（Beggs&Plenz 2003验证值）
```

**工程实现**：
- E-S键的权重w初始化为均匀分布
- STDP更新规则在临界态下天然产生幂律分布（无需额外设计）
- **关键约束**：不能人为截断E-S键的权重分布，否则破坏幂律

**监测指标**：
```python
# 实时监测是否接近临界态
alpha = fit_powerlaw(weight_distribution)
# 目标：1.4 ≤ alpha ≤ 1.6
```

### 5.3 自组织演化规则（局部规则→全局临界态）

每个SDI节点仅基于**局部信息**执行以下5条规则：

```
规则1（固化）：E-S + n_LTP ≥ θ_LTP → E-L
规则2（消除）：I-S + n_LTD ≥ θ_LTD → 断开
规则3（新建）：断开 + ∂F/∂w_local > Ea_S → E-S（建立新E-S键）
规则4（衰减）：E-L + 不活跃时间 > T_decay → E-S（退回可塑态）
规则5（稳定侧抑制）：I-L + 层级汇聚比满足1:10 → 保持（不衰减）
```

**关键参数**（基于神经科学文献推导，已确认）：

| 参数 | 确定值 | 来源文献 | 推导理由 |
|------|--------|---------|---------|
| θ_LTP | **50次** | Frey&Morris 1997, J Neuroscience | L-LTP触发需75-100个脉冲，忆阻器响应比蛋白合成快，取×0.6 |
| θ_LTD | **20次** | Dudek&Bear 1992 | LTD比LTP更易触发，保证修剪优先于固化，维持网络稀疏性 |
| T_decay | **10⁵步（≈1天）** | Ebbinghaus遗忘曲线 | 无激活约1天（86,400ms）后长期记忆开始衰减，时间步=1ms |
| Ea_S | **0.15** | Watts&Strogatz 1998，p_rewire=0.12 | 最优小世界p=0.12，约15%节点维持动态可塑性 |
| Ea_L | **0.85** | 能量守恒+LTP固化壁垒比值 | Ea_L = 1-Ea_S，骨架键稳定性是新键的5.7倍 |
| E:I权重比 | **4:1** | DeFelipe 2002；096文件80/20法则 | 灵长类皮层80%兴奋+20%抑制，普适生物约束 |
| α_target | **1.5** | Beggs&Plenz 2003, J Neuroscience | 神经雪崩幂律指数P(S)∝S^(-3/2)，临界态物理标志 |

## 六、与仿真的对接

### 6.1 节点数据结构

```python
class SDI_Node:
    # 4路SDI-SDI接口
    P1: Bond_EL = None   # 兴奋长时程
    P2: Bond_IL = None   # 抑制长时程
    P3: Bond_ES = None   # 兴奋短时程
    P4: Bond_IS = None   # 抑制短时程
    
    # 2路SDI-SYN接口
    Q1: SYN_Interface = None  # 主路
    Q2: SYN_Interface = None  # 冗余路
    
    # 局部状态
    n_LTP: int = 0       # LTP事件累积计数
    n_LTD: int = 0       # LTD事件累积计数
    last_active: int = 0 # 最后活跃时间步

class Bond:
    type: str            # 'E-L'/'I-L'/'E-S'/'I-S'
    weight: float        # 当前权重 w ∈ [0,1]
    Ea: float           # 活化能
    source: SDI_Node
    target: SDI_Node
```
