iNEST 六层智能涌现架构

从感知到超智能的六层递进，每层对应一个自由能最小化尺度，层层耦合形成完整涌现链路。

L1 感知层 · Perception 传感器融合 + 事件相机 + 时序编码
L2 反应层 · Reaction 脉冲神经网络 + STDP 局部学习
L3 适应层 · Adaptation FEP-STDP 全局优化 + 元拓扑重构
L4 创造层 · Creation 涌现编码器 + 多区域协同 + 符号接地
L5 通用层 · General 跨域迁移 + 元学习 + 自主目标设定
L6 超智能层 · Super 多智能体涌现 + 集体意识 + 超越人类
